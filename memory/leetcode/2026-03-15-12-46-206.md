# 🎯 LeetCode 精讲 #206 - 反转链表

---

### 📋 题目描述

**来源**: LeetCode 第206题  
**难度**: Easy  
**链接**: https://leetcode.cn/problems/reverse-linked-list/

#### 题目原文
> 给你单链表的头节点 `head` ，请你反转链表，并返回反转后的链表。

#### 示例

**示例 1:**
```
输入: head = [1,2,3,4,5]
输出: [5,4,3,2,1]
```

**示例 2:**
```
输入: head = [1,2]
输出: [2,1]
```

**示例 3:**
```
输入: head = []
输出: []
```

#### 约束条件
- 链表中节点的数目范围是 `[0, 5000]`
- `-5000 <= Node.val <= 5000`

### 💡 解题思路

这道题是**链表操作**的基石，几乎所有链表相关题目都会涉及到反转的思想。掌握三种解法对面试非常重要。

#### 方法一：迭代法（三指针法）⭐推荐

**核心思想**
遍历链表，逐个将节点的 `next` 指针指向前一个节点。

**三指针定义**
- `prev`: 指向已经反转好的链表的头部（初始为 `null`）
- `curr`: 当前正在处理的节点（初始为 `head`）
- `next`: 临时保存 `curr.next`，防止断链后丢失

**图解过程**

```
初始状态:  1 -> 2 -> 3 -> 4 -> 5 -> null
          ↑
         head

步骤1: 保存next，断开curr.next指向prev
       null <- 1    2 -> 3 -> 4 -> 5 -> null
                    ↑
                   curr/next

步骤2: prev移动到curr，curr移动到next
       null <- 1    2 -> 3 -> 4 -> 5 -> null
              ↑    ↑
             prev curr

重复直到curr为null...

最终结果: null <- 1 <- 2 <- 3 <- 4 <- 5
                              ↑
                             prev (新的head)
```

**代码实现**

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        prev = None      # 已反转部分的头部
        curr = head      # 当前处理的节点
        
        while curr:
            next_temp = curr.next   # 步骤1: 暂存下一个节点
            curr.next = prev        # 步骤2: 反转指针方向
            prev = curr             # 步骤3: prev前移
            curr = next_temp        # 步骤4: curr前移
        
        return prev      # 最终prev指向新链表的头节点
```

**代码执行流程追踪**

以 `[1, 2, 3]` 为例：

| 轮次 | prev | curr | next_temp | 操作后 curr.next |
|------|------|------|-----------|------------------|
| 初始 | null | 1    | -         | -                |
| 1    | 1    | 2    | 2         | null             |
| 2    | 2    | 3    | 3         | 1                |
| 3    | 3    | null | null      | 2                |
| 结束 | 3    | -    | -         | -                |

返回 `prev = 3`，链表为 `3 -> 2 -> 1 -> null` ✓

#### 方法二：递归法

**核心思想**
先递归反转后面的链表，再将当前节点接到反转后链表的末尾。

**递归定义**
- `reverseList(head)` 返回以 `head` 为头的链表反转后的新头节点
- 假设 `head.next` 之后的链表已经反转完成
- 只需将 `head` 接到反转后链表的尾部

**图解过程**

```
原始: 1 -> 2 -> 3 -> 4 -> 5 -> null

递归到最深处（基准条件）:
      1 -> 2 -> 3 -> 4    5 -> null
                        ↑
                       返回5

回溯过程:
      1 -> 2 -> 3    4 <- 5
                   ↑    ↑
                  curr new_head
      
      操作: curr.next.next = curr  (4的next是5，让5.next指向4)
            curr.next = null        (断开4原来的next，避免环)
      
      结果: 1 -> 2 -> 3    4 <- 5
                        null
                        
继续回溯直到完成...
```

**代码实现**

```python
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        # 基准条件: 空链表或只有一个节点
        if not head or not head.next:
            return head
        
        # 递归反转后续链表
        new_head = self.reverseList(head.next)
        
        # 将当前节点接到反转后链表的末尾
        # head.next 是现在新链表的尾节点
        head.next.next = head   # 让尾节点的next指向当前节点
        head.next = None        # 当前节点变成新的尾节点
        
        return new_head
```

**递归执行流程**

```
reverseList(1)
  -> reverseList(2)
       -> reverseList(3)
            -> reverseList(4)
                 -> reverseList(5)
                      -> 返回 5 (基准条件)
                 -> 4.next.next = 4, 4.next = null, 返回 5
            -> 3.next.next = 3, 3.next = null, 返回 5
       -> 2.next.next = 2, 2.next = null, 返回 5
  -> 1.next.next = 1, 1.next = null, 返回 5

最终结果: 5 -> 4 -> 3 -> 2 -> 1 -> null
```

#### 方法三：头插法（使用虚拟头节点）

**核心思想**
创建虚拟头节点，遍历原链表，将每个节点采用"头插法"插入到虚拟头节点之后。

**图解过程**

```
初始: dummy -> null
      1 -> 2 -> 3 -> 4 -> 5

第1步: 将1头插到dummy后
       dummy -> 1 -> null
                2 -> 3 -> 4 -> 5

第2步: 将2头插到dummy后
       dummy -> 2 -> 1 -> null
                3 -> 4 -> 5

第3步: 将3头插到dummy后
       dummy -> 3 -> 2 -> 1 -> null
                4 -> 5

...继续直到原链表为空

结果: dummy -> 5 -> 4 -> 3 -> 2 -> 1 -> null
```

**代码实现**

```python
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(0)   # 虚拟头节点
        
        while head:
            # 保存下一个节点
            next_node = head.next
            
            # 头插法: 将head插入到dummy之后
            head.next = dummy.next
            dummy.next = head
            
            # 移动到下一个节点
            head = next_node
        
        return dummy.next
```

### 📊 复杂度分析

| 方法 | 时间复杂度 | 空间复杂度 | 说明 |
|------|-----------|-----------|------|
| 迭代法 | O(n) | O(1) | ⭐ 最优解，面试首选 |
| 递归法 | O(n) | O(n) | 递归栈空间，代码简洁但空间较差 |
| 头插法 | O(n) | O(1) | 需要虚拟节点，思路直观 |

**n = 链表长度**

### 🎯 关键点总结

#### 1. 迭代法的三个关键步骤
```
1. 保存next = curr.next（防止断链）
2. 反转 curr.next = prev
3. 移动指针 prev = curr; curr = next
```

#### 2. 递归法的核心理解
```
假设后面的已经反转好了，我只管接上去
- new_head = reverseList(head.next)  // 反转后续
- head.next.next = head              // 接上去
- head.next = None                   // 断开避免环
```

#### 3. 常见错误
- ❌ 忘记保存 `next` 节点，导致断链后无法继续
- ❌ 递归法中忘记将 `head.next` 置为 `null`，导致环形链表
- ❌ 返回 `curr` 而不是 `prev`（循环结束时 `curr` 已经是 `null`）
