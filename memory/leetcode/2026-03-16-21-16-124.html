LeetCode #124 - 二叉树中的最大路径和

Hard 二叉树 递归 DFS

📋 题目描述

二叉树中的 路径 被定义为一条节点序列，序列中每对相邻节点之间都存在一条边。同一个节点在一条路径序列中 至多出现一次。该路径 至少包含一个 节点，且不一定经过根节点。

路径和 是路径中各节点值的总和。

给你一个二叉树的根节点 root，返回其 最大路径和。

示例 1
      1      / \     2   3

输入：root = [1,2,3]

输出：6

解释：最优路径是 2 → 1 → 3，路径和为 2 + 1 + 3 = 6

示例 2
     -10       / \      9  20         / \        15  7

输入：root = [-10,9,20,null,null,15,7]

输出：42

解释：最优路径是 15 → 20 → 7，路径和为 15 + 20 + 7 = 42

💡 核心思路

对于每个节点，考虑经过它的最大路径和 = 左子树最大贡献值 + 当前节点值 + 右子树最大贡献值。但返回给父节点时，只能选择左或右中较大的一边（因为路径不能分叉）。

🎬 算法执行过程（点击步骤或代码行查看对应关系）

Python 实现


1# Definition for a binary tree node.
2# class TreeNode:
3#     def __init__(self, val=0, left=None, right=None):
4#         self.val = val
5#         self.left = left
6#         self.right = right
7
8class Solution:
9    def maxPathSum(self, root: TreeNode) -> int:
10        self.max_sum = float('-inf')
11        
12        def dfs(node):
13            if not node:
14                return 0
15            
16            # 递归计算左右子树的最大贡献值
17            left_gain = dfs(node.left)
18            right_gain = dfs(node.right)
19            
20            # 与0比较：负贡献不要
21            left_gain = max(left_gain, 0)
22            right_gain = max(right_gain, 0)
23            
24            # 经过当前节点的最大路径和
25            price_newpath = node.val + left_gain + right_gain
26            
27            # 更新全局最大路径和
28            self.max_sum = max(self.max_sum, price_newpath)
29            
30            # 返回给父节点的最大贡献值（只能选一边）
31            return node.val + max(left_gain, right_gain)
32        
33        dfs(root)
34        return self.max_sum

📊 复杂度分析

维度 复杂度 说明
时间复杂度 O(N) 每个节点只访问一次，N为节点总数
空间复杂度 O(H) H为树的高度，递归栈深度；最坏O(N)，平衡树O(logN)

🎯 关键点总结

1. 贡献值 vs 路径和

每个节点返回给父节点的是最大贡献值（只能走左边或右边之一），但更新全局答案时考虑的是完整路径和（可以走两边）。这是本题最核心的区分点。

2. 负贡献的处理

子树贡献值为负时，选择不走那条路径（贡献值取0），因为加上负数只会让路径和变小。

3. 后序遍历思想

必须先知道左右子树的结果，才能计算当前节点的结果，这是典型的后序遍历（左右根）。
