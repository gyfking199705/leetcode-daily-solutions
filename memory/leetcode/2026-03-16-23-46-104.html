LeetCode #104 - 二叉树的最大深度

Easy 树 / 递归 / DFS

📋 题目描述

给定一个二叉树，找出其最大深度。

二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。


说明：叶子节点是指没有子节点的节点。


示例：

3
/ \\
9 20
/ \\
15 7

给定二叉树 [3,9,20,null,null,15,7]，返回最大深度为 3

💡 核心洞察：一棵树的最大深度 = max(左子树深度, 右子树深度) + 1。空节点的深度为0。这是典型的「后序遍历」思想。

Python 实现



1class TreeNode:


2    def __init__(self, val=0, left=None, right=None):


3        self.val = val


4        self.left = left


5        self.right = right


6


7class Solution:


8    def maxDepth(self, root: Optional[TreeNode]) -> int:


9        # 递归终止条件：空节点深度为0


10        if not root:


11            return 0


12        


13        # 递归计算左子树深度


14        left_depth = self.maxDepth(root.left)


15        


16        # 递归计算右子树深度


17        right_depth = self.maxDepth(root.right)


18        


19        # 当前节点的深度 = 左右子树最大深度 + 1


20        return max(left_depth, right_depth) + 1

📊 复杂度分析

指标 复杂度 说明
时间复杂度 O(n) 每个节点只被访问一次，n为节点总数
空间复杂度 O(h) 递归栈深度等于树的高度h，最坏情况h=n（退化为链表）
🌲 树的遍历方式对比：
• 前序遍历：先访问根节点，再访问子节点（自顶向下）
• 中序遍历：先访问左子树，再根节点，再右子树
• 后序遍历：先访问子节点，再访问根节点（自底向上）

本题使用「后序遍历」思想，因为需要先知道子树的深度，才能计算当前节点的深度。

🔄 BFS解法（迭代）

除了递归，也可以用层序遍历（BFS）来解决。每层遍历完，深度+1。

1from collections import deque


2


3class Solution:


4    def maxDepth(self, root: Optional[TreeNode]) -> int:


5        if not root:


6            return 0


7        


8        queue = deque([root])


9        depth = 0


10        


11        while queue:


12            depth += 1


13            level_size = len(queue)


14            for _ in range(level_size):


15                node = queue.popleft()


16                if node.left:


17                    queue.append(node.left)


18                if node.right:


19                    queue.append(node.right)


20        


21        return depth
🔄 两种方法对比：
• DFS递归：代码简洁，空间复杂度O(h)，适合理解递归思想
• BFS迭代：显式使用队列，空间复杂度O(w)，w为最大宽度

对于这道题，两种方法效率相当，但DFS递归代码更简洁优雅。
