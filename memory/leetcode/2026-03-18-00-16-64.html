LeetCode #64 - 最小路径和 Medium

动态规划经典入门题，掌握二维DP的状态转移思路

📋 题目描述

给定一个包含非负整数的 m x n 网格 grid，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。

说明：每次只能向下或者向右移动一步。

示例：

输入: grid = [[1,3,1],
              [1,5,1],
              [4,2,1]]
输出: 7
解释: 路径 1→3→1→1→1 的总和最小

🧠 核心思路

这是一道经典的二维动态规划问题。关键在于发现：

  • 到达每个格子的最小路径和 = 当前格子的值 + min(从上方来的最小和, 从左方来的最小和)
  • 第一行和第一列只能从一个方向来，需要特殊处理
dp[i][j] = grid[i][j] + min(dp[i-1][j], dp[i][j-1])

空间优化 可以原地修改输入数组，空间复杂度优化到 O(1)
