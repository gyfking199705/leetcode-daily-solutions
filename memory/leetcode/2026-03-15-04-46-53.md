# 🎯 LeetCode 精讲 #53 - 最大子数组和

---

## 📋 题目描述

**原题链接**: https://leetcode.cn/problems/maximum-subarray/

**难度**: 中等

给你一个整数数组 `nums`，请你找出一个具有**最大和**的**连续子数组**（子数组最少包含一个元素），返回其最大和。

**子数组** 是数组中的一个连续部分。

### 示例

**示例 1:**
```
输入: nums = [-2,1,-3,4,-1,2,1,-5,4]
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。
```

**示例 2:**
```
输入: nums = [1]
输出: 1
```

**示例 3:**
```
输入: nums = [5,4,-1,7,8]
输出: 23
解释: 整个数组的和最大，为 23。
```

### 约束条件
- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`

**进阶**: 如果你已经实现复杂度为 `O(n)` 的解法，尝试使用更为精妙的 **分治法** 求解。

## 💡 解题思路

### 一、暴力解法（不推荐）

枚举所有可能的子数组，计算和，取最大值。

- 时间复杂度: O(n³) 或 O(n²)（用前缀和优化）
- 空间复杂度: O(1) 或 O(n)

对于 `n = 10^5`，O(n²) 会超时，需要更优解法。

---

### 二、动态规划解法（Kadane算法）⭐推荐

#### 核心思想

> **"以当前位置结尾的最大子数组和"** 只与 **"以前一个位置结尾的最大子数组和"** 有关。

#### 状态定义

设 `dp[i]` 表示**以第 i 个元素结尾**的连续子数组的最大和。

#### 状态转移方程

```
dp[i] = max(nums[i], dp[i-1] + nums[i])
```

**解释**:
- 如果 `dp[i-1] > 0`，说明前面累加和对当前有增益，连上：`dp[i-1] + nums[i]`
- 如果 `dp[i-1] <= 0`，说明前面累加和是负担，舍弃，从当前重新开始：`nums[i]`

#### 答案提取

最终答案是所有 `dp[i]` 中的最大值：
```
max_sum = max(dp[0], dp[1], ..., dp[n-1])
```

#### 空间优化

由于 `dp[i]` 只依赖于 `dp[i-1]`，可以用**滚动变量**代替数组：
```
current_max = max(nums[i], current_max + nums[i])
```

---

### 三、分治法解法（进阶）

#### 核心思想

将数组从中间分成左右两部分，最大子数组和可能出现在：
1. **左半部分**
2. **右半部分**
3. **跨越中点**（包含左半部分后缀和右半部分前缀）

#### 递归结构
```
maxSubArray(nums, left, right):
    if left == right: return nums[left]
    
    mid = (left + right) // 2
    left_max = maxSubArray(nums, left, mid)
    right_max = maxSubArray(nums, mid+1, right)
    cross_max = maxCrossingSum(nums, left, mid, right)
    
    return max(left_max, right_max, cross_max)
```

## 📝 代码实现

### 解法一：动态规划（空间优化版）

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        """
        Kadane算法 - 动态规划空间优化
        时间复杂度: O(n)
        空间复杂度: O(1)
        """
        # current_max: 以当前位置结尾的最大子数组和
        # global_max: 全局最大子数组和
        
        current_max = global_max = nums[0]
        
        for i in range(1, len(nums)):
            # 状态转移: 要么续接前面的，要么从当前重新开始
            current_max = max(nums[i], current_max + nums[i])
            # 更新全局最大值
            global_max = max(global_max, current_max)
        
        return global_max
```

**执行过程可视化（示例1）：**

| i | nums[i] | current_max (max(nums[i], prev+nums[i])) | global_max |
|---|---------|------------------------------------------|------------|
| 0 | -2      | -2                                       | -2         |
| 1 | 1       | max(1, -2+1= -1) = 1                     | 1          |
| 2 | -3      | max(-3, 1-3= -2) = -2                    | 1          |
| 3 | 4       | max(4, -2+4= 2) = 4                      | 4          |
| 4 | -1      | max(-1, 4-1= 3) = 3                      | 4          |
| 5 | 2       | max(2, 3+2= 5) = 5                       | 5          |
| 6 | 1       | max(1, 5+1= 6) = 6                       | **6** ✓    |
| 7 | -5      | max(-5, 6-5= 1) = 1                      | 6          |
| 8 | 4       | max(4, 1+4= 5) = 5                       | 6          |

**关键观察**: 在 i=6 时达到最大值 6，对应子数组 `[4,-1,2,1]`。

### 解法二：动态规划（原始DP数组版）

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        """
        动态规划 - 使用dp数组（便于理解，但空间复杂度O(n)）
        """
        n = len(nums)
        dp = [0] * n
        dp[0] = nums[0]
        
        for i in range(1, n):
            dp[i] = max(nums[i], dp[i-1] + nums[i])
        
        return max(dp)
```

### 解法三：分治法

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        """
        分治法
        时间复杂度: O(n log n)
        空间复杂度: O(log n) - 递归栈深度
        """
        def maxCrossingSum(nums, left, mid, right):
            """计算跨越中点的最大子数组和"""
            # 从中点向左扫描，找最大后缀和
            left_sum = float('-inf')
            current_sum = 0
            for i in range(mid, left - 1, -1):
                current_sum += nums[i]
                left_sum = max(left_sum, current_sum)
            
            # 从中点+1向右扫描，找最大前缀和
            right_sum = float('-inf')
            current_sum = 0
            for i in range(mid + 1, right + 1):
                current_sum += nums[i]
                right_sum = max(right_sum, current_sum)
            
            return left_sum + right_sum
        
        def maxSubArrayHelper(nums, left, right):
            """递归求解"""
            if left == right:
                return nums[left]
            
            mid = (left + right) // 2
            
            # 三种情况的最大值
            left_max = maxSubArrayHelper(nums, left, mid)
            right_max = maxSubArrayHelper(nums, mid + 1, right)
            cross_max = maxCrossingSum(nums, left, mid, right)
            
            return max(left_max, right_max, cross_max)
        
        return maxSubArrayHelper(nums, 0, len(nums) - 1)
```

### 解法四：扩展版（同时返回子数组区间）

```python
class Solution:
    def maxSubArrayWithRange(self, nums: List[int]) -> tuple:
        """
        扩展：同时返回最大子数组的起止索引
        返回: (最大和, 起始索引, 结束索引)
        """
        n = len(nums)
        
        # dp[i] 表示以i结尾的最大子数组和
        # start[i] 记录该子数组的起始位置
        dp = [0] * n
        start = [0] * n
        
        dp[0] = nums[0]
        start[0] = 0
        
        for i in range(1, n):
            if dp[i-1] > 0:
                dp[i] = dp[i-1] + nums[i]
                start[i] = start[i-1]
            else:
                dp[i] = nums[i]
                start[i] = i
        
        # 找到全局最大值
        max_sum = max(dp)
        end_idx = dp.index(max_sum)
        start_idx = start[end_idx]
        
        return max_sum, start_idx, end_idx

```

## 🎯 关键要点总结

### 1. Kadane算法的核心洞察

> **局部最优 → 全局最优**

以每个位置结尾的最大子数组和，只取决于前一个位置的结果。这种"无后效性"是动态规划的标志。

### 2. 为什么可以舍弃前面的和？

```
如果 dp[i-1] < 0:
    dp[i-1] + nums[i] < nums[i]
    前面是"累赘"，果断舍弃！
```

### 3. 处理全负数数组

题目约束 `nums.length >= 1`，所以至少选一个元素。Kadane算法天然支持全负数情况：
- `nums = [-5, -2, -8, -1]`
- 结果: `-1`（最大的那个负数）

### 💡 面试高频考点

1. **Kadane算法的时间/空间复杂度？**
   - O(n) 时间，O(1) 空间（滚动优化后）

2. **如何处理全负数情况？**
   - Kadane算法天然支持，会返回最大（最接近0）的那个负数

3. **如果要求返回子数组本身？**
   - 需要额外记录每个位置的起始索引，最后回溯

4. **分治法的核心思想？**
   - 最大子数组要么在左、要么在右、要么跨越中点

5. **与"最大子序列和"的区别？**
   - 子数组：必须连续
   - 子序列：不要求连续（所有正数之和）
