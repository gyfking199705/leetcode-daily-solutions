🌳 LeetCode #101 - 对称二叉树

Symmetric Tree · Easy · 递归 / DFS / BFS

📋 题目描述

给你一个二叉树的根节点 root，检查它是否轴对称。

对于一棵对称二叉树，其左子树和右子树必须在镜像对称的意义上相等：

  • 两个子树的根节点值相同
  • 左子树的左子树要与右子树的右子树对称
  • 左子树的右子树要与右子树的左子树对称

Python 实现（递归法）


1class TreeNode:
2    def __init__(self, val=0, left=None, right=None):
3        self.val = val
4        self.left = left
5        self.right = right
6
7class Solution:
8    def isSymmetric(self, root: TreeNode) -> bool:
9        if not root:
10            return True
11        return self.check(root.left, root.right)
12    
13    def check(self, left: TreeNode, right: TreeNode) -> bool:
14        # 两个节点都为空 → 对称
15        if not left and not right:
16            return True
17        # 只有一个为空 → 不对称
18        if not left or not right:
19            return False
20        # 值不同 → 不对称
21        if left.val != right.val:
22            return False
23        # 递归：外侧和内侧对称检查
24        return (self.check(left.left, right.right) and
25                self.check(left.right, right.left))

💡 核心思路解析

递归的核心思想： 把大问题拆解成结构相同的子问题

  • Base Case 1： 两个节点都为空 → 对称（返回 True）
  • Base Case 2： 一个空一个不空 → 不对称（返回 False）
  • Base Case 3： 值不相等 → 不对称（返回 False）
  • Recursive Case： 当前节点相等 + 外侧对称 + 内侧对称

关键细节： 递归调用时参数的配对

check(left.left, right.right)   # 外侧比较（左的左 vs 右的右）
check(left.right, right.left)   # 内侧比较（左的右 vs 右的左）

📝 迭代解法（BFS版本）


class Solution:
    def isSymmetric(self, root: TreeNode) -> bool:
        if not root:
            return True
        
        from collections import deque
        queue = deque([(root.left, root.right)])
        
        while queue:
            left, right = queue.popleft()
            
            if not left and not right:
                continue
            if not left or not right:
                return False
            if left.val != right.val:
                return False
            
            # 按对称顺序入队
            queue.append((left.left, right.right))
            queue.append((left.right, right.left))
        
        return True

迭代法使用队列存储需要比较的节点对，按对称顺序入队是关键。
