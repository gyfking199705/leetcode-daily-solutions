LeetCode #43 - 字符串相乘

Multiply Strings

🟡 Medium 模拟 字符串 数学

📋 题目描述

给定两个以字符串形式表示的非负整数 num1 和 num2，返回它们的乘积，也以字符串形式表示。

注意：不能使用任何内置的 BigInteger 库或直接将输入转换为整数。

示例 1：
输入：num1 = "2", num2 = "3"
输出："6"
示例 2：
输入：num1 = "123", num2 = "456"
输出："56088"

💡 核心思路

模拟我们在小学学习的竖式乘法过程：将 num2 的每一位与 num1 相乘，然后将所有部分积相加。关键优化是利用一个结果数组直接累加所有部分积，避免中间字符串拼接。
