LeetCode #19 - 删除链表的倒数第 N 个结点

Medium 双指针法 | 链表操作 | 快慢指针

📋 题目描述

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

示例 1：

输入：head = [1,2,3,4,5], n = 2

输出：[1,2,3,5]

解释：删除倒数第2个结点（值为4的结点），链表变为 [1,2,3,5]

示例 2：

输入：head = [1], n = 1

输出：[]

示例 3：

输入：head = [1,2], n = 1

输出：[1]

进阶：你能尝试使用一趟扫描实现吗？

💡 核心思路：使用快慢指针，让快指针先走 n 步，然后快慢指针一起走，当快指针到达末尾时，慢指针正好在倒数第 n+1 个结点（即要删除结点的前一个结点）。

Python 实现

创建虚拟头结点
→
快指针先走 n 步
→
双指针同步移动
→
快指针到末尾
→
删除目标结点
→
返回新头

📊 复杂度分析

指标 值 说明
时间复杂度 O(L) 只需遍历链表一次，L 为链表长度
空间复杂度 O(1) 只使用常数额外空间（几个指针）
遍历次数 1 次 满足进阶要求的一趟扫描

🎯 关键要点

🎭 虚拟头结点

使用 dummy 结点可以统一处理删除头结点的边界情况，避免特殊判断。

🚀 快慢指针

快指针先走 n+1 步，保证慢指针最终停在待删除结点的前一个位置。

⚠️ 边界处理

考虑 n 等于链表长度（删除头结点）和链表只有一个结点的情况。

🔄 内存释放

Python 有垃圾回收，C++需要手动 delete；Java 交给 GC 即可。
