LeetCode #20 - 有效的括号

Easy 栈 · 字符串 · 经典基础题

📋 题目描述

给定一个只包括 '('、')'、'{'、'}'、'['、']' 的字符串 s，判断字符串是否有效。

有效字符串需满足：

  1. 左括号必须用相同类型的右括号闭合。
  2. 左括号必须以正确的顺序闭合。
  3. 每个右括号都有一个对应的相同类型的左括号。
输入: s = "()[]{}"
输出: true
输入: s = "(]"
输出: false
输入: s = "([)]"
输出: false

💡 核心思路

🎯 栈的应用场景

  • 后进先出 (LIFO)：最后遇到的左括号，应该最先被闭合
  • 遇到左括号 → 压入栈中
  • 遇到右括号 → 与栈顶元素配对检查，匹配则弹出
  • 最终栈为空 → 所有括号都正确闭合
