🎯 LeetCode #10 - 正则表达式匹配

Hard 动态规划 字符串 递归

📋 题目描述

给你一个字符串 s 和一个字符规律 p，请你来实现一个支持 '.' 和 '*' 的正则表达式匹配。

  • '.' 匹配任意单个字符
  • '*' 匹配零个或多个前面的那一个元素

所谓匹配，是要涵盖整个字符串 s 的，而不是部分字符串。

示例 1:

输入：s = "aa", p = "a"

输出：false

解释："a" 无法匹配 "aa" 整个字符串。

示例 2:

输入：s = "aa", p = "a*"

输出：true

解释：因为 '*' 代表零个或多个前面的元素, 在这里前面的元素就是 'a'。因此，字符串 "aa" 可被视为 'a' 重复了一次。

💡 核心思路

使用动态规划来解决这个问题。定义 dp[i][j] 表示 s[0..i-1] 与 p[0..j-1] 是否匹配。

关键转移方程：

  • 如果 p[j-1] 是普通字符或 '.'：匹配当且仅当 s[i-1] == p[j-1]（或 p[j-1] == '.'）且 dp[i-1][j-1] 为真
  • 如果 p[j-1] == '*'：可以选择匹配0次（忽略 p[j-2]*）或匹配多次（前提是 s[i-1] 与 p[j-2] 匹配）
