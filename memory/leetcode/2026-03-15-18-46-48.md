## 一、题目描述

给定一个 n × n 的二维矩阵 matrix 表示一个图像。请你将图像顺时针旋转 90 度。

**注意**：你必须在 **原地** 旋转图像，这意味着你需要直接修改输入的二维矩阵。请不要使用另一个矩阵来旋转图像。

### 示例 1
```
输入: matrix = [[1,2,3],
                [4,5,6],
                [7,8,9]]
                
输出: [[7,4,1],
       [8,5,2],
       [9,6,3]]
```

### 示例 2
```
输入: matrix = [[ 5, 1, 9,11],
                [ 2, 4, 8,10],
                [13, 3, 6, 7],
                [15,14,12,16]]
                
输出: [[15,13, 2, 5],
       [14, 3, 4, 1],
       [12, 6, 8, 9],
       [16, 7,10,11]]
```

### 约束条件
- `n == matrix.length == matrix[i].length`
- `1 <= n <= 20`
- `-1000 <= matrix[i][j] <= 1000`

## 二、解题思路

### 方法：翻转代替旋转

这道题有两种主要解法：

#### 解法一：翻转法（推荐）⭐

**核心思想**：顺时针旋转 90° = 水平翻转 + 主对角线翻转

以一个 3×3 矩阵为例：

```
原始:           水平翻转:      主对角线翻转(转置):
1 2 3           7 8 9          7 4 1
4 5 6    -->    4 5 6    -->   8 5 2
7 8 9           1 2 3          9 6 3
```

**数学原理**：
- 水平翻转：元素 `matrix[i][j]` 移动到 `matrix[n-1-i][j]`
- 转置：元素 `matrix[i][j]` 移动到 `matrix[j][i]`
- 组合效果等价于顺时针旋转 90°

#### 解法二：原地旋转（环形替换）

**核心思想**：
- 矩阵由外向内分为若干"层"
- 每层是一个环，需要旋转
- 对于每个环，四个角落的元素循环交换位置

**旋转规律**：
```
左上角[i][j]      ←      右上角[i][n-1-j]
     ↓                       ↑
左下角[n-1-j][i]  →      右下角[n-1-i][n-1-j]
```

## 三、完整代码实现

### Python 实现（翻转法）

```python
from typing import List

class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        """
        原地顺时针旋转矩阵90度
        方法：水平翻转 + 转置
        """
        n = len(matrix)
        
        # 步骤1：水平翻转（上下交换）
        for i in range(n // 2):
            for j in range(n):
                matrix[i][j], matrix[n - 1 - i][j] = matrix[n - 1 - i][j], matrix[i][j]
        
        # 步骤2：主对角线翻转（转置）
        for i in range(n):
            for j in range(i + 1, n):
                matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
```

### Python 实现（原地旋转法）

```python
from typing import List

class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        """
        原地顺时针旋转矩阵90度
        方法：环形替换，一圈一圈处理
        """
        n = len(matrix)
        
        # 只需要处理外层到内层的每一圈
        for i in range(n // 2):
            # 对于第i圈，列从i开始到n-i-2
            for j in range(i, n - i - 1):
                # 保存左上角的值
                temp = matrix[i][j]
                
                # 左下角 → 左上角
                matrix[i][j] = matrix[n - 1 - j][i]
                
                # 右下角 → 左下角
                matrix[n - 1 - j][i] = matrix[n - 1 - i][n - 1 - j]
                
                # 右上角 → 右下角
                matrix[n - 1 - i][n - 1 - j] = matrix[j][n - 1 - i]
                
                # 左上角(暂存) → 右上角
                matrix[j][n - 1 - i] = temp
```

### Java 实现（翻转法）

```java
class Solution {
    public void rotate(int[][] matrix) {
        int n = matrix.length;
        
        // 水平翻转
        for (int i = 0; i < n / 2; i++) {
            for (int j = 0; j < n; j++) {
                int temp = matrix[i][j];
                matrix[i][j] = matrix[n - 1 - i][j];
                matrix[n - 1 - i][j] = temp;
            }
        }
        
        // 主对角线翻转（转置）
        for (int i = 0; i < n; i++) {
            for (int j = i + 1; j < n; j++) {
                int temp = matrix[i][j];
                matrix[i][j] = matrix[j][i];
                matrix[j][i] = temp;
            }
        }
    }
}
```

### C++ 实现（翻转法）

```cpp
class Solution {
public:
    void rotate(vector<vector<int>>& matrix) {
        int n = matrix.size();
        
        // 水平翻转
        for (int i = 0; i < n / 2; ++i) {
            for (int j = 0; j < n; ++j) {
                swap(matrix[i][j], matrix[n - 1 - i][j]);
            }
        }
        
        // 主对角线翻转（转置）
        for (int i = 0; i < n; ++i) {
            for (int j = i + 1; j < n; ++j) {
                swap(matrix[i][j], matrix[j][i]);
            }
        }
    }
};
```

## 四、复杂度分析

### 翻转法
| 指标 | 复杂度 | 说明 |
|------|--------|------|
| **时间复杂度** | O(n²) | 两次遍历矩阵，每次访问 n×n/2 个元素 |
| **空间复杂度** | O(1) | 原地操作，仅使用常数额外空间 |

### 原地旋转法
| 指标 | 复杂度 | 说明 |
|------|--------|------|
| **时间复杂度** | O(n²) | 遍历约 n²/4 个元素，每个元素操作4次 |
| **空间复杂度** | O(1) | 仅使用一个临时变量 |

## 五、关键要点总结

### 🔑 核心技巧

1. **翻转代替旋转**：顺时针90° = 水平翻转 + 转置
   - 这是最容易理解和实现的解法
   - 两步都是简单的对称交换操作

2. **理解坐标变换**：
   - 顺时针90°旋转：`matrix[i][j] → matrix[j][n-1-i]`
   - 逆时针90°旋转：`matrix[i][j] → matrix[n-1-j][i]`
   - 180°旋转：`matrix[i][j] → matrix[n-1-i][n-1-j]`

3. **边界处理**：
   - 水平翻转只需遍历前 n/2 行
   - 转置只需遍历上三角或下三角

### 📝 记忆口诀

> "上下翻，再转置，顺时针九十度"

### ⚠️ 常见陷阱

- 转置时注意只处理上三角（`j > i`），否则会交换两次变回原样
- 原地旋转时注意循环范围，不要重复处理同一组元素

```python
def rotateAntiClockwise(matrix):
    n = len(matrix)
    # 左右翻转 + 转置
    for i in range(n):
        for j in range(n // 2):
            matrix[i][j], matrix[i][n-1-j] = matrix[i][n-1-j], matrix[i][j]
    for i in range(n):
        for j in range(i + 1, n):
            matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
```

```python
def rotate180(matrix):
    n = len(matrix)
    # 水平翻转 + 垂直翻转
    for i in range(n // 2):
        for j in range(n):
            matrix[i][j], matrix[n-1-i][j] = matrix[n-1-i][j], matrix[i][j]
    for i in range(n):
        for j in range(n // 2):
            matrix[i][j], matrix[i][n-1-j] = matrix[i][n-1-j], matrix[i][j]
```
