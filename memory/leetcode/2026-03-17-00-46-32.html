LeetCode #32 - 最长有效括号

Hard 栈 动态规划 贪心

📋 题目描述

给你一个只包含 '(' 和 ')' 的字符串，找出最长有效（匹配）括号子串的长度。

示例

输入：s = "(()"
输出：2
解释：最长有效括号子串是 "()"
输入：s = ")()())"
输出：4
解释：最长有效括号子串是 "()()"
输入：s = ""
输出：0

提示

  • 0 <= s.length <= 3 × 10⁴
  • s[i] 为 '(' 或 ')'

💡 核心思路

这是一道经典的括号匹配问题，有三种主流解法：

解法一：栈 ⭐推荐
解法二：动态规划
解法三：双向扫描(O(1)空间)
核心思想：利用栈存储索引，通过栈底元素作为「最后一个未匹配的位置」来计算长度

🔑 关键点解析

  • 哨兵技巧：栈底始终保持 -1 或最后一个未匹配的 ')' 索引，方便计算长度
  • 遇到 '('：将索引入栈，等待匹配
  • 遇到 ')'：
    • 弹出栈顶（尝试匹配）
    • 若栈空：当前 ')' 无法匹配，将其索引入栈作为新边界
    • 若栈不空：匹配成功！长度 = 当前索引 - 栈顶索引
核心思想：dp[i] 表示以索引 i 结尾的最长有效括号长度

状态转移

  • s[i] == '('：dp[i] = 0（以左括号结尾不可能构成有效子串）
  • s[i] == ')' 且 s[i-1] == '('：形如 ...()
    dp[i] = dp[i-2] + 2
  • s[i] == ')' 且 s[i-1] == ')'：形如 ...))
    检查 i - dp[i-1] - 1 位置的字符，若为 '(' 则匹配成功：
    dp[i] = dp[i-1] + 2 + dp[i - dp[i-1] - 2]
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        n = len(s)
        dp = [0] * n
        max_len = 0
        
        for i in range(1, n):
            if s[i] == ')':
                if s[i - 1] == '(':
                    # Case 1: ...()
                    dp[i] = (dp[i - 2] if i >= 2 else 0) + 2
                elif i - dp[i - 1] - 1 >= 0 and s[i - dp[i - 1] - 1] == '(':
                    # Case 2: ...))，检查前面的 '('
                    dp[i] = dp[i - 1] + 2 + (dp[i - dp[i - 1] - 2] if i - dp[i - 1] - 2 >= 0 else 0)
                max_len = max(max_len, dp[i])
        
        return max_len
核心思想：两次扫描，分别处理右括号过多和左括号过多的情况

算法步骤

  1. 从左到右：left记录'('数量，right记录')'数量
    • left == right 时更新最大长度
    • right > left 时重置（右括号过多，无法匹配）
  2. 从右到左：镜像扫描
    • left == right 时更新最大长度
    • left > right 时重置（左括号过多）
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        left = right = max_len = 0
        
        # 从左到右
        for c in s:
            if c == '(':
                left += 1
            else:
                right += 1
            
            if left == right:
                max_len = max(max_len, 2 * right)
            elif right > left:
                left = right = 0
        
        left = right = 0
        
        # 从右到左
        for c in reversed(s):
            if c == '(':
                left += 1
            else:
                right += 1
            
            if left == right:
                max_len = max(max_len, 2 * left)
            elif left > right:
                left = right = 0
        
        return max_len
