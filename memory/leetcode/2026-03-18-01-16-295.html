LeetCode #295 - 数据流的中位数

Median Finder - 使用双堆设计实现动态中位数查询

Hard 设计 双堆 优先队列

📋 题目描述

中位数是有序整数列表中的中间值。如果列表的大小是偶数，则没有中间值，中位数是两个中间值的平均值。

实现 MedianFinder 类：

  • MedianFinder() 初始化 MedianFinder 对象
  • void addNum(int num) 将数据流中的数字 num 添加到数据结构中
  • double findMedian() 返回到目前为止所有元素的中位数

💡 核心思路

使用最大堆（左边）和最小堆（右边）维护数据：左边保存较小的一半，右边保存较大的一半。保持两堆大小平衡（差值不超过1），中位数就在堆顶。

由于 Python 的 heapq 只提供最小堆，最大堆通过存储负数实现。插入时先放入一个堆，再将堆顶元素移到另一个堆，保持平衡。

🎯 关键要点总结

1. 双堆平衡

最大堆存较小一半，最小堆存较大一半。插入时先放入对应堆，再平衡：将堆顶元素移动到另一个堆。

2. Python 最大堆技巧

Python 的 heapq 是最小堆，要实现最大堆，存入 -num，取出时再取负。

3. 大小约束

最大堆的大小 = 最小堆的大小，或者 最大堆的大小 = 最小堆的大小 + 1。这样中位数计算才有规律。
