LeetCode #106 - 从中序与后序遍历序列构造二叉树

Medium 二叉树 分治 递归

📋 题目描述

给定两个整数数组 inorder 和 postorder，其中 inorder 是二叉树的中序遍历，postorder 是同一棵树的后序遍历，请你构造并返回这棵二叉树。

示例：

输入: inorder = [9,3,15,20,7], postorder = [9,15,7,20,3]
输出: [3,9,20,null,null,15,7]

      3
     / \
    9  20
      /  \
     15   7

关键洞察：

  • 后序遍历的最后一个元素是根节点
  • 在中序遍历中找到根节点位置，左侧是左子树，右侧是右子树
  • 根据中序左子树的长度，确定后序中左右子树的范围

Python 实现


1class TreeNode:
2    def __init__(self, val=0, left=None, right=None):
3        self.val = val
4        self.left = left
5        self.right = right
6
7class Solution:
8    def buildTree(self, inorder, postorder):
9        # 用哈希表存储中序遍历的值到索引的映射
10        index_map = {val: idx for idx, val in enumerate(inorder)}
11        
12        def build(in_left, in_right, post_left, post_right):
13            # 递归终止条件
14            if in_left > in_right or post_left > post_right:
15                return None
16            
17            # 后序遍历的最后一个元素是根节点
18            root_val = postorder[post_right]
19            root = TreeNode(root_val)
20            
21            # 找到根节点在中序遍历中的位置
22            root_idx = index_map[root_val]
23            
24            # 计算左子树的节点数
25            left_tree_size = root_idx - in_left
26            
27            # 递归构建左子树
28            root.left = build(in_left, root_idx - 1, 
29                              post_left, post_left + left_tree_size - 1)
30            
31            # 递归构建右子树
32            root.right = build(root_idx + 1, in_right,
33                               post_left + left_tree_size, post_right - 1)
34            
35            return root
36        
37        n = len(inorder)
38        return build(0, n - 1, 0, n - 1)

🔍 复杂度分析

维度 值 说明
时间复杂度 O(n) 每个节点访问一次，哈希表查找 O(1)
空间复杂度 O(n) 哈希表存储 n 个节点 + 递归栈深度 O(h)

优化技巧：

  • 使用哈希表存储中序索引，避免每次 O(n) 查找
  • 后序遍历数组使用索引而非切片，减少空间开销
  • 递归改为迭代：可以使用栈模拟递归过程

🧠 核心考点总结

  • 遍历顺序理解：必须清楚三种遍历（前序、中序、后序）的访问顺序
  • 根节点定位：后序的末尾 = 根；中序的某个位置 = 根
  • 范围计算：根据中序左子树长度推导后序的左右子树范围
  • 分治思想：大问题分解为结构相同的子问题
