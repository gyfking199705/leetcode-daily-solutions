LeetCode #45 - 跳跃游戏 II

贪心算法经典题 | 最小跳跃次数 Medium

📋 题目描述

给定一个长度为 n 的 0 索引整数数组 nums。初始位置为 nums[0]。

每个元素 nums[i] 表示从索引 i 向前跳转的最大长度。

返回到达 nums[n - 1] 的最小跳跃次数。

假设你总是可以到达数组的最后一个位置。

示例 1

输入: nums = [2,3,1,1,4]

输出: 2

解释: 跳到最后一个位置的最小跳跃数是 2。
从下标为 0 跳到下标为 1 的位置，跳 1 步，然后跳 3 步到达数组的最后一个位置。

示例 2

输入: nums = [2,3,0,1,4]

输出: 2

💡 核心思路

本题使用贪心算法，核心思想是：

  • 维护 end：当前跳跃能到达的最远边界
  • 维护 farthest：在当前边界内，下一跳能到达的最远位置
  • 当遍历到 end 时，说明必须再跳一次，更新边界

关键洞察：我们不需要关心具体跳到哪里，只需要知道「在当前这一步内，能跳到的最远位置」。

# 跳跃类贪心问题通用模板
def jump_game_template(nums):
    n = len(nums)
    jumps = 0          # 已使用的跳跃次数
    end = 0            # 当前跳跃能到达的最远边界
    farthest = 0       # 下一跳能到达的最远位置
    
    for i in range(n - 1):  # 注意：不需要遍历到最后一个元素
        # 更新最远可达位置
        farthest = max(farthest, i + nums[i])
        
        # 到达当前边界，必须跳一次
        if i == end:
            jumps += 1
            end = farthest
    
    return jumps
