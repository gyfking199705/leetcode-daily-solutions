LeetCode #347 - 前K个高频元素 ● Medium

📋 题目描述

给你一个整数数组 nums 和一个整数 k，请你返回其中出现频率前 k 高的元素。可以按 任意顺序 返回答案。

哈希表 堆 / 优先队列 桶排序

🧠 核心思路

步骤：

  1. 使用哈希表统计每个元素的出现频率（O(n)）
  2. 使用最小堆维护频率最高的k个元素（O(n log k)）
  3. 堆的大小始终不超过k，将k+1个元素加入堆时，弹出最小的那个

关键点：维护一个大小为k的最小堆，堆顶是k个高频元素中频率最小的。如果新元素频率大于堆顶，替换堆顶元素。

Python 实现（最小堆优化版）

📊 复杂度分析

维度 复杂度 说明
时间复杂度 O(n log k) 统计频率O(n)，建堆O(n)，每次push/pop O(log k)，共O(n log k)
空间复杂度 O(n) 哈希表存储频率O(n)，堆存储k个元素O(k)

优化：使用桶排序可以达到O(n)时间复杂度（空间换时间）。
