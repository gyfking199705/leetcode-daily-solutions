🌳 LeetCode #104 - 二叉树的最大深度

Easy 二叉树 | DFS | 递归 | 分治

📋 题目描述

给定一个二叉树，找出其最大深度。

二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

说明：叶子节点是指没有子节点的节点。

示例二叉树结构：
        3        / \       9  20      /  /  \     null 15  7

返回其最大深度 3

🎬 算法执行过程（点击步骤或代码行查看对应关系）

Python 实现



1class TreeNode:


2    def __init__(self, val=0, left=None, right=None):


3        self.val = val


4        self.left = left


5        self.right = right


6


7class Solution:


8    def maxDepth(self, root: TreeNode) -> int:


9        # 递归终止条件：空节点深度为0


10        if not root:


11            return 0


12        


13        # 递归计算左子树深度


14        left_depth = self.maxDepth(root.left)


15        


16        # 递归计算右子树深度


17        right_depth = self.maxDepth(root.right)


18        


19        # 当前节点深度 = max(左,右) + 1


20        return max(left_depth, right_depth) + 1

🔍 核心思路解析

分治思想（Divide and Conquer）

求整棵树的最大深度，可以分解为三个子问题：

  1. 求左子树的最大深度
  2. 求右子树的最大深度
  3. 当前树的深度 = max(左深度, 右深度) + 1（加1是根节点本身）

递归三要素

要素 说明 代码体现
终止条件 空节点没有深度 if not root: return 0
递归调用 分别计算左右子树 maxDepth(root.left/right)
结果合并 取较大值+1返回 max(left, right) + 1

📊 复杂度分析

指标 复杂度 说明
时间复杂度 O(n) 每个节点只访问一次，n为节点总数
空间复杂度 O(h) 递归栈深度等于树的高度h，最坏O(n)（退化为链表）
