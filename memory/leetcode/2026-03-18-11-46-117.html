LeetCode #117 - 填充每个节点的下一个右侧节点指针 II

Populate Next Right Pointers in Each Node II

Medium 🌲 二叉树 🔗 链表 🔄 BFS ⭐ O(1)空间

📋 题目描述

给定一个二叉树，其每个节点包含：

  • val：节点的值
  • left：左子节点指针
  • right：右子节点指针
  • next：指向同一层中右侧的下一个节点（初始为 NULL）

要求：将每个节点的 next 指针指向其同一层的下一个右侧节点。如果找不到下一个右侧节点，则将 next 指针设置为 NULL。

💡 关键点

这是一个不完全二叉树，不像116题那样保证每层都是满的。这意味着不能使用基于完全二叉树的性质优化，需要更通用的解法。

⏱️ 复杂度分析

解法 时间复杂度 空间复杂度 说明
BFS（队列） O(N) O(W) W为最大层宽，需要队列存储每层节点
O(1)空间迭代 O(N) O(1) 利用已建立的next指针作为"链表"遍历，最优解 ⭐

📝 核心思路总结

🎯 解法一：BFS层序遍历（直观）

使用队列进行层序遍历，每层遍历时将前一个节点的next指向当前节点。空间复杂度O(W)，W为树的最大宽度。

🌟 解法二：O(1)空间迭代（最优）

将每一层看作一个链表，利用上一层已经建立好的next指针来遍历并连接下一层。核心是维护三个指针：当前层节点、下一层的头部、下一层的尾部。

🔑 关键技巧

dummy = Node(0) 作为下一层的"虚拟头节点"，避免处理下一层第一个节点的特殊逻辑，简化代码。
