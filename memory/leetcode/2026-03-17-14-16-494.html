🎯 LeetCode #494 - 目标和

📋 题目描述

给你一个整数数组 nums 和一个整数 target。

向数组中的每个整数前添加 '+' 或 '-'，然后串联起所有整数，可以构造一个 表达式 ：

  • 例如，nums = [2, 1]，可以在 2 之前添加 '+'，在 1 之前添加 '-'，然后串联起来得到表达式 "+2-1"。

返回可以通过上述方法构造的、运算结果等于 target 的不同 表达式 的数目。

💡 核心思想：数学转换

设正数之和为 P，负数绝对值之和为 N：
• 总和关系：P + N = sum(nums)
• 目标关系：P - N = target

两式相加得：2P = target + sum
因此：P = (target + sum) / 2

问题转化为：从数组中选择若干个数，使其和等于 P 的方案数（01背包问题）

示例 1：

输入：nums = [1,1,1,1,1], target = 3
输出：5
解释：-1+1+1+1+1 = 3
      +1-1+1+1+1 = 3
      +1+1-1+1+1 = 3
      +1+1+1-1+1 = 3
      +1+1+1+1-1 = 3

示例 2：

输入：nums = [1], target = 1
输出：1

📊 复杂度分析

维度 时间复杂度 空间复杂度 说明
基础实现 O(n × target) O(n × target) 二维DP数组，n为数组长度
空间优化 O(n × target) O(target) 滚动数组，一维DP
最坏情况 O(n × sum) O(sum) target可能接近sum

🔑 关键点总结

  • 数学转换是核心：将符号分配问题转化为子集和问题
  • 边界情况：如果 (target + sum) % 2 != 0，直接返回0（无法整除）
  • 负数处理：如果 abs(target) > sum，直接返回0（不可能实现）
  • DP定义：dp[i][j] 表示前i个数中选若干个，和为j的方案数
  • 状态转移：dp[j] = dp[j] + dp[j - num]（滚动数组优化）
🎯 01背包问题的本质

这是一个组合计数类型的01背包：求装满背包的方案数，而不是最大价值。
状态转移时采用累加而非取最大值。
