LeetCode #64

最小路径和

Medium 动态规划 矩阵DP

📋 题目描述

给定一个包含非负整数的 m x n 网格 grid，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。

说明：每次只能向下或者向右移动一步。

示例：
输入：grid = [[1,3,1],[1,5,1],[4,2,1]]
输出：7
解释：因为路径 1→3→1→1→1 的总和最小。

提示：

  • 1 <= m, n <= 200
  • 0 <= grid[i][j] <= 200

💡 核心思路

这是一个经典的动态规划问题。关键观察：

  • 到达某个格子 (i,j) 只能从 上方 (i-1,j) 或 左方 (i,j-1) 过来
  • 因此，到 (i,j) 的最小路径和 = grid[i][j] + min(从上面来的最小和, 从左面来的最小和)
状态转移方程：
dp[i][j] = grid[i][j] + min(dp[i-1][j], dp[i][j-1])
