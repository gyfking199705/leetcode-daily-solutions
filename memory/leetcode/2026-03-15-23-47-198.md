# LeetCode 198 - 打家劫舍 (House Robber)

## 📋 题目描述

你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，**如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警**。

给定一个代表每个房屋存放金额的非负整数数组 `nums`，计算你在 **不触动警报装置的情况下**，一夜之内能够偷窃到的最高金额。

### 示例

**示例 1：**
```
输入: nums = [1,2,3,1]
输出: 4
解释: 偷窃 1 号房屋 (金额 = 1)，然后偷窃 3 号房屋 (金额 = 3)。
     偷窃到的最高金额 = 1 + 3 = 4
```

**示例 2：**
```
输入: nums = [2,7,9,3,1]
输出: 12
解释: 偷窃 1 号房屋 (金额 = 2)，偷窃 3 号房屋 (金额 = 9)，接着偷窃 5 号房屋 (金额 = 1)。
     偷窃到的最高金额 = 2 + 9 + 1 = 12
```

**约束条件：**
- `1 <= nums.length <= 100`
- `0 <= nums[i] <= 400`

## 💡 解题思路

### 核心思想：动态规划

这是一个经典的**动态规划**问题。关键在于理解**状态转移**的逻辑：

对于第 `i` 个房子，小偷有两个选择：
1. **偷第 `i` 个房子**：那么第 `i-1` 个房子不能偷，最大金额 = 前 `i-2` 个房子的最大金额 + `nums[i]`
2. **不偷第 `i` 个房子**：最大金额 = 前 `i-1` 个房子的最大金额

### 状态定义

`dp[i]` = 偷窃前 `i` 个房子（下标 0 到 i-1）能获得的最高金额

### 状态转移方程

```
dp[i] = max(dp[i-1], dp[i-2] + nums[i-1])
```

**解释：**
- `dp[i-1]`：不偷第 `i-1` 号房子（对应原数组 `nums[i-1]`）
- `dp[i-2] + nums[i-1]`：偷第 `i-1` 号房子

### 初始条件

```
dp[0] = 0          // 偷 0 个房子，金额为 0
dp[1] = nums[0]    // 偷 1 个房子，金额就是第一个房子的钱
```

## 📝 代码实现

### 方法一：标准动态规划（空间 O(n)）

```python
from typing import List

class Solution:
    def rob(self, nums: List[int]) -> int:
        n = len(nums)
        if n == 0:
            return 0
        if n == 1:
            return nums[0]
        
        # dp[i] 表示前 i 个房子能偷到的最大金额
        dp = [0] * (n + 1)
        dp[0] = 0           # 0 个房子
        dp[1] = nums[0]     # 1 个房子
        
        for i in range(2, n + 1):
            # 选择：偷当前房子 vs 不偷当前房子
            dp[i] = max(dp[i-1], dp[i-2] + nums[i-1])
        
        return dp[n]
```

### 方法二：空间优化版（空间 O(1)）

由于 `dp[i]` 只依赖于 `dp[i-1]` 和 `dp[i-2]`，我们可以用两个变量代替整个数组：

```python
from typing import List

class Solution:
    def rob(self, nums: List[int]) -> int:
        n = len(nums)
        if n == 0:
            return 0
        if n == 1:
            return nums[0]
        
        # 只用两个变量滚动更新
        prev2 = 0           # dp[i-2]
        prev1 = nums[0]     # dp[i-1]
        
        for i in range(1, n):
            # 当前最大金额 = max(不偷当前, 偷当前)
            curr = max(prev1, prev2 + nums[i])
            prev2 = prev1
            prev1 = curr
        
        return prev1
```

### 方法三：状态机 DP（更通用的思路）

```python
from typing import List

class Solution:
    def rob(self, nums: List[int]) -> int:
        n = len(nums)
        if n == 0:
            return 0
        
        # dp[i][0]: 不偷第 i 个房子的最大金额
        # dp[i][1]: 偷第 i 个房子的最大金额
        dp = [[0, 0] for _ in range(n)]
        dp[0][0] = 0
        dp[0][1] = nums[0]
        
        for i in range(1, n):
            dp[i][0] = max(dp[i-1][0], dp[i-1][1])  # 不偷：前一间可偷可不偷
            dp[i][1] = dp[i-1][0] + nums[i]         # 偷：前一间必须不偷
        
        return max(dp[n-1][0], dp[n-1][1])
```

## 🎯 关键要点总结

1. **DP 的核心逻辑**：每个位置的状态只依赖于前两个位置
2. **选择模型**：对于每个房子，只有"偷"或"不偷"两种选择
3. **空间优化技巧**：滚动数组思想，将 O(n) 空间降至 O(1)
4. **边界处理**：注意处理空数组和单元素数组的情况

## 💬 面试高频考点

1. **如何想到用 DP？**
   - 最优化问题 + 重叠子问题
   - 每个房子的最优解依赖于之前房子的最优解

2. **空间优化是怎么想到的？**
   - 观察状态转移方程，发现只依赖前两个状态
   - 用滚动变量代替数组

3. **如果房子排成环形怎么办？**（进阶）
   - 分两种情况：偷第一家不偷最后一家，或不偷第一家偷最后一家
   - 取两种情况的最大值
