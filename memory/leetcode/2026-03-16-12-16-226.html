LeetCode #226 - 翻转二叉树

Invert Binary Tree · Easy · 递归 / DFS / BFS

📋 题目描述

给你一棵二叉树的根节点 root，翻转这棵二叉树，并返回其根节点。

翻转定义：将二叉树中每个节点的左子树和右子树进行交换。

🎯 示例

输入：root = [4,2,7,1,3,6,9]
输出：[4,7,2,9,6,3,1]

解释：每一层的左右子树都被交换了位置

树 递归 DFS BFS

🧠 核心思路

翻转二叉树的核心思想是递归地交换每个节点的左右子节点。

🔑 递归三部曲

1. 终止条件：当前节点为空，直接返回
2. 单层逻辑：交换当前节点的左右子节点
3. 递归调用：分别对左右子树进行翻转

💡 关键理解：翻转操作是「自底向上」还是「自顶向下」？
实际上两者都可以！先交换再递归，或者先递归再交换，结果是一样的。

💡 面试技巧

🎤 高频追问

• 「你能用非递归的方式实现吗？」→ BFS/栈
• 「翻转前序遍历和中序遍历结果有什么变化？」→ 前序：左右互换；中序：完全颠倒
• 「如何判断两棵树互为翻转？」→ 同时遍历，一个左优先一个右优先

⚠️ 常见错误

• 忘记处理空节点
• 交换时写成 root.left = root.right 导致右子树丢失（应该先保存）
