📋 题目描述

给定一个二叉树的根节点 root，返回它的 中序 遍历结果。

中序遍历顺序：左子树 → 根节点 → 右子树

        1
       / \
      2   3
     / \
    4   5

示例输出：[4, 2, 5, 1, 3]

📊 复杂度分析

方法 时间复杂度 空间复杂度 特点
递归 O(n) O(h) - 栈空间 最直观，h为树高度
迭代（栈） O(n) O(h) 显式管理栈，可控
Morris O(n) O(1) threaded tree，空间最优

💡 核心思路

  • 中序遍历口诀：左根右 —— 先遍历左子树，再访问根节点，最后遍历右子树
  • 递归本质：将大问题分解为「左子树遍历 + 访问根 + 右子树遍历」三个子问题
  • 迭代要点：用栈模拟递归，先一路向左压栈，再弹出访问，最后转向右子树
  • Morris精髓：利用叶子节点的空指针，建立临时链接，实现O(1)空间
