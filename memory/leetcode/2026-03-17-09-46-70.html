LeetCode #70 - 爬楼梯

Easy 动态规划经典入门题 | 斐波那契数列 | 空间优化技巧

📋 题目描述

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

示例 1： 
输入：n = 2
输出：2
解释：有两种方法可以爬到楼顶。
1. 1 阶 + 1 阶
2. 2 阶
示例 2： 
输入：n = 3
输出：3
解释：有三种方法可以爬到楼顶。
1. 1 阶 + 1 阶 + 1 阶
2. 1 阶 + 2 阶
3. 2 阶 + 1 阶

提示：1 <= n <= 45

💡 核心思路

这道题的精髓在于发现状态转移方程：

dp[i] = dp[i-1] + dp[i-2]

含义：到达第 i 阶的方法数 = 从 i-1 阶跨1步的方法数 + 从 i-2 阶跨2步的方法数

dp[i-2]
+
dp[i-1]
=
dp[i]

这正是斐波那契数列的定义！所以这道题本质上是求斐波那契数列的第 n 项。

边界条件：
  • dp[0] = 1：到达地面有1种方法（不爬）
  • dp[1] = 1：到达第1阶只有1种方法（跨1步）

✨ 空间优化技巧

由于 dp[i] 只依赖 dp[i-1] 和 dp[i-2]，我们不需要保存整个数组， 只需用两个变量滚动更新，将空间复杂度从 O(n) 降到 O(1)！

📝 解题套路总结

1️⃣ 找状态

定义 dp[i] 为「到达第 i 阶的方法数」，这是问题的子结构

2️⃣ 找转移

思考最后一步是怎么来的：可能是从 i-1 跨1步，或从 i-2 跨2步

3️⃣ 找边界

dp[0]=1, dp[1]=1，递推从这里开始

4️⃣ 空间优化

如果转移只依赖前 k 个状态，考虑用滚动变量代替数组
