LeetCode #100 - 相同的树 (Same Tree)

📋 题目描述

给你两棵二叉树的根节点 p 和 q，编写一个函数来检验这两棵树是否相同。

如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

Easy 二叉树 递归 DFS

🎯 核心思路

递归分治思想

两棵树相同 ⟺ 当前节点值相同 AND 左子树相同 AND 右子树相同

这是一个典型的递归问题，遵循「分解-解决-合并」的思路：

  • 分解：判断当前节点 + 递归判断左右子树
  • 解决：基本情况（空节点判断）
  • 合并：三个条件同时满足才返回 true

✓ 结构相同 + 值相同 = 相同的树

🎬 算法执行过程（点击步骤或代码行查看对应关系）

步骤 1: 基本情况处理✓

如果 p 和 q 都为空，返回 True。两棵空树是相同的。

步骤 2: 单一空节点判断✗

如果只有一个为空，返回 False。结构不同。

步骤 3: 节点值比较✗

如果当前节点值不同，返回 False。

步骤 4: 递归左子树

递归判断 p 的左子树和 q 的左子树是否相同。

步骤 5: 递归右子树

递归判断 p 的右子树和 q 的右子树是否相同。

步骤 6: 合并结果✓

左右子树都相同，整棵树才相同。

📊 复杂度分析

方法 时间复杂度 空间复杂度 说明
递归 (DFS) O(min(n,m)) O(min(h1,h2)) h 为树的高度，递归栈空间
迭代 (BFS) O(min(n,m)) O(min(n,m)) 队列存储节点对

其中 n, m 分别是两棵树的节点数。一旦发现不同会提前返回，所以是 min(n,m)。

📝 关键要点总结

  • 递归三要素：基本情况（终止条件）、递归条件、返回值
  • 空节点处理：都为空时返回 true，一个为空返回 false
  • 短路求值：Python 的 and/or 可以简化条件判断
  • 提前返回：值不同立即返回 false，不需要继续递归
