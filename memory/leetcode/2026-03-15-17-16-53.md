# 🎯 LeetCode 精讲 #53 - 最大子数组和

### 题目原文
> 给你一个整数数组 `nums` ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

**子数组**：是数组中的一个连续部分。

### 示例

**示例 1:**
```
输入: nums = [-2,1,-3,4,-1,2,1,-5,4]
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。
```

**示例 2:**
```
输入: nums = [1]
输出: 1
```

**示例 3:**
```
输入: nums = [5,4,-1,7,8]
输出: 23
```

### 约束条件
- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`

**进阶**: 如果你已经实现复杂度为 `O(n)` 的解法，尝试使用更为精妙的 **分治法** 求解。

## 🧠 解题思路

### 核心思想：动态规划（Dynamic Programming）

这道题是**动态规划**的入门经典，被称为「Kadane算法」。核心思想是：

> **以第 i 个元素结尾的最大子数组和 = max(第 i 个元素本身, 以第 i-1 个元素结尾的最大子数组和 + 第 i 个元素)**

#### 状态定义

设 `dp[i]` 表示**以第 i 个元素结尾**的连续子数组的最大和。

#### 状态转移方程

```
dp[i] = max(nums[i], dp[i-1] + nums[i])
```

**解释**:
- 如果 `dp[i-1] < 0`：前面的和对当前元素是"拖累"，不如从当前元素重新开始
- 如果 `dp[i-1] >= 0`：前面的和对当前元素有"增益"，应该接上

#### 滚动数组优化

观察到 `dp[i]` 只依赖 `dp[i-1]`，可以用一个变量代替整个数组，将空间复杂度从 O(n) 降到 O(1)。

## 💻 代码实现

### 方法一：动态规划（空间优化版）⭐推荐

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        # 初始化：当前子数组和、全局最大和
        curr_sum = max_sum = nums[0]
        
        for i in range(1, len(nums)):
            # 关键决策：是延续前面的子数组，还是从当前元素重新开始？
            curr_sum = max(nums[i], curr_sum + nums[i])
            # 更新全局最大值
            max_sum = max(max_sum, curr_sum)
        
        return max_sum
```

**执行过程图解**:
```
数组: [-2, 1, -3, 4, -1, 2, 1, -5, 4]

i=0: curr_sum = -2, max_sum = -2
i=1: curr_sum = max(1, -2+1) = 1,  max_sum = max(-2, 1) = 1
i=2: curr_sum = max(-3, 1-3) = -2, max_sum = max(1, -2) = 1
i=3: curr_sum = max(4, -2+4) = 4,  max_sum = max(1, 4) = 4
i=4: curr_sum = max(-1, 4-1) = 3,  max_sum = max(4, 3) = 4
i=5: curr_sum = max(2, 3+2) = 5,   max_sum = max(4, 5) = 5
i=6: curr_sum = max(1, 5+1) = 6,   max_sum = max(5, 6) = 6  ⭐找到最大和！
i=7: curr_sum = max(-5, 6-5) = 1,  max_sum = max(6, 1) = 6
i=8: curr_sum = max(4, 1+4) = 5,   max_sum = max(6, 5) = 6

结果: 6 (子数组 [4,-1,2,1])
```

### 方法二：动态规划（标准版）

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        n = len(nums)
        # dp[i] 表示以第 i 个元素结尾的最大子数组和
        dp = [0] * n
        dp[0] = nums[0]
        
        for i in range(1, n):
            dp[i] = max(nums[i], dp[i-1] + nums[i])
        
        return max(dp)
```

### 方法三：分治法（进阶）

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        def divide_conquer(left, right):
            if left == right:
                return nums[left]
            
            mid = (left + right) // 2
            
            # 递归求解左半部分和右半部分
            left_max = divide_conquer(left, mid)
            right_max = divide_conquer(mid + 1, right)
            
            # 求解跨越中点的最大子数组和
            # 从中点向左扩展的最大和
            cross_left_max = float('-inf')
            curr_sum = 0
            for i in range(mid, left - 1, -1):
                curr_sum += nums[i]
                cross_left_max = max(cross_left_max, curr_sum)
            
            # 从中点向右扩展的最大和
            cross_right_max = float('-inf')
            curr_sum = 0
            for i in range(mid + 1, right + 1):
                curr_sum += nums[i]
                cross_right_max = max(cross_right_max, curr_sum)
            
            # 跨越中点的总和
            cross_max = cross_left_max + cross_right_max
            
            return max(left_max, right_max, cross_max)
        
        return divide_conquer(0, len(nums) - 1)
```

**分治思想**:
```
最大子数组和可能出现在三个位置之一：
1. 完全在左半部分
2. 完全在右半部分  
3. 跨越中点（一部分在左，一部分在右）

递归求解，时间复杂度 O(n log n)
```

### 方法四：前缀和

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        # 前缀和数组
        prefix_sum = 0
        min_prefix = 0  # 前缀和的最小值
        max_sum = float('-inf')
        
        for num in nums:
            prefix_sum += num
            # 当前子数组和 = 当前前缀和 - 之前最小的前缀和
            max_sum = max(max_sum, prefix_sum - min_prefix)
            min_prefix = min(min_prefix, prefix_sum)
        
        return max_sum
```

**核心思想**:
```
subarray(i,j) = prefix[j+1] - prefix[i]
要使 subarray(i,j) 最大，对于每个 j，需要找到最小的 prefix[i] (i <= j)
```

## 🎯 关键要点总结

### 1. 决策点
```python
# 核心：当前元素是"另起炉灶"还是"接续前人"？
curr_sum = max(nums[i], curr_sum + nums[i])
```

### 2. 为什么这样决策？
- **如果 curr_sum < 0**：前面的子数组对当前元素是负贡献，抛弃它重新开始更优
- **如果 curr_sum >= 0**：前面的子数组对当前元素是正贡献，应该接上

### 3. 常见陷阱
- ❌ 初始化 `curr_sum = 0` —— 如果数组全负数会出错
- ✅ 正确初始化 `curr_sum = max_sum = nums[0]`

### 4. 扩展问题：如何输出具体的子数组？

```python
def maxSubArrayWithIndices(nums):
    curr_sum = max_sum = nums[0]
    start = end = temp_start = 0
    
    for i in range(1, len(nums)):
        if curr_sum + nums[i] < nums[i]:
            # 重新开始
            curr_sum = nums[i]
            temp_start = i
        else:
            # 继续延伸
            curr_sum += nums[i]
        
        if curr_sum > max_sum:
            max_sum = curr_sum
            start = temp_start
            end = i
    
    return max_sum, nums[start:end+1]
```

## 🧠 面试高频考点

### Q1: 这道题为什么不能用滑动窗口？
> 滑动窗口适用于"满足某条件的最短/最长子数组"，但最大和没有单调性，无法用双指针维护窗口。

### Q2: 如果数组全是负数，算法还正确吗？
> 正确。因为我们初始化 `curr_sum = nums[0]`，且每次决策取 `max(nums[i], curr_sum + nums[i])`，即使全负数也会选择最大的那个负数。

### Q3: 分治法和DP的优劣对比？
> DP是O(n)时间O(1)空间，最优。分治O(n log n)主要是为了练习分治思想，实际面试中DP解法即可。

### Q4: 如何证明贪心策略的正确性？
> 数学归纳法：假设对于前i-1个元素，我们已经找到以i-1结尾的最大子数组和。对于第i个元素，只有两种选择：重新开始 或 接上，取较大者必然最优。
