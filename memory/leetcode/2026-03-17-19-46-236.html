LeetCode #236 - 二叉树的最近公共祖先

Medium 树 | 递归 | 分治

📋 题目描述

给定一个二叉树，找到该树中两个指定节点的最近公共祖先（Lowest Common Ancestor, LCA）。

对于两个节点 p 和 q，其最近公共祖先是指：在二叉树中，p 和 q 的最近公共祖先节点 x 满足 x 是 p 和 q 的祖先，且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。

关键性质：所有节点的值都是唯一的；p、q 为不同节点且均存在于二叉树中。

🎯 核心思路：递归分治

最近公共祖先有以下三种情况：

  1. p 和 q 分别位于当前节点的左右子树中 → 当前节点就是 LCA
  2. p 和 q 都在当前节点的左子树中 → LCA 在左子树
  3. p 和 q 都在当前节点的右子树中 → LCA 在右子树
递归终止条件：如果当前节点是 null，或是 p，或是 q，直接返回当前节点。
