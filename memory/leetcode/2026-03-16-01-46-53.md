## 一、题目描述

给定一个整数数组 `nums`，请你找出一个具有最大和的**连续子数组**（子数组最少包含一个元素），返回其最大和。

**子数组**：数组中的一个连续部分。

### 示例

```
输入: nums = [-2,1,-3,4,-1,2,1,-5,4]
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。
```

```
输入: nums = [1]
输出: 1
```

```
输入: nums = [5,4,-1,7,8]
输出: 23
解释: 整个数组的和最大。
```

## 二、解题思路

### 2.1 核心思想

这道题的关键在于理解**局部最优与全局最优**的关系。

想象一下股票投资：你有每日涨跌数据，想找出哪段连续交易日能获得最大收益。

**关键洞察**：
- 如果之前的"资产"是正数，不管多少，都值得**继续累积**
- 如果之前的"资产"是负数（负债），不如**重新开始**

### 2.2 状态定义

设 `dp[i]` 表示**以第 i 个元素结尾**的最大子数组和。

### 2.3 状态转移方程

```
dp[i] = max(dp[i-1] + nums[i], nums[i])
```

**含义**：
- 接续前面的子数组：`dp[i-1] + nums[i]`
- 从当前元素重新开始：`nums[i]`
- 取两者较大值

等价写法（更直观）：
```
dp[i] = max(dp[i-1], 0) + nums[i]
```

如果前面的和是负数，就丢弃（取0），从当前重新开始。

### 2.4 最终答案

`max(dp[0], dp[1], ..., dp[n-1])`

## 三、代码实现

### 3.1 基础动态规划（O(n)空间）

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        n = len(nums)
        dp = [0] * n
        dp[0] = nums[0]
        max_sum = dp[0]
        
        for i in range(1, n):
            # 状态转移：要么接续，要么重新开始
            dp[i] = max(dp[i-1] + nums[i], nums[i])
            # 更新全局最大值
            max_sum = max(max_sum, dp[i])
        
        return max_sum
```

### 3.2 空间优化版 - Kadane算法（推荐）⭐

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        # current_max: 以当前元素结尾的最大子数组和
        # max_so_far: 全局最大子数组和
        current_max = max_so_far = nums[0]
        
        for num in nums[1:]:
            # 关键决策：继续当前子数组，还是重新开始
            current_max = max(num, current_max + num)
            # 更新全局最优
            max_so_far = max(max_so_far, current_max)
        
        return max_so_far
```

### 3.3 Java实现

```java
class Solution {
    public int maxSubArray(int[] nums) {
        int currentMax = nums[0];
        int maxSoFar = nums[0];
        
        for (int i = 1; i < nums.length; i++) {
            // 状态转移
            currentMax = Math.max(nums[i], currentMax + nums[i]);
            // 更新全局最优
            maxSoFar = Math.max(maxSoFar, currentMax);
        }
        
        return maxSoFar;
    }
}
```

### 3.4 C++实现

class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int currentMax = nums[0];
        int maxSoFar = nums[0];
        
        for (int i = 1; i < nums.size(); i++) {
            currentMax = max(nums[i], currentMax + nums[i]);
            maxSoFar = max(maxSoFar, currentMax);
        }

## 四、算法执行过程可视化

以 `nums = [-2, 1, -3, 4, -1, 2, 1, -5]` 为例：

| i | nums[i] | current_max | max_so_far | 决策说明 |
|---|---------|-------------|------------|----------|
| 0 | -2 | -2 | -2 | 初始值 |
| 1 | 1 | 1 | 1 | -2+1=-1<1，重新开始 |
| 2 | -3 | -2 | 1 | 1-3=-2，继续但和变小 |
| 3 | 4 | 4 | 4 | -2+4=2<4，重新开始 ✅ |
| 4 | -1 | 3 | 4 | 4-1=3，继续 |
| 5 | 2 | 5 | 5 | 3+2=5，继续 ✅ |
| 6 | 1 | 6 | 6 | 5+1=6，继续 ✅ |
| 7 | -5 | 1 | 6 | 6-5=1，继续但最优仍是6 |

**最优子数组**: `[4, -1, 2, 1]`，和为 **6**

## 六、关键要点总结

### 6.1 面试必考点
1. **状态定义**：`dp[i]` 表示以第i个元素结尾的最大子数组和
2. **状态转移**：`dp[i] = max(dp[i-1] + nums[i], nums[i])`
