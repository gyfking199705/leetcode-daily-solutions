LeetCode #169 - 多数元素

Majority Element · Boyer-Moore 投票算法 · 难度: Easy

📋 题目描述

给定一个大小为 n 的数组 nums，返回其中的多数元素。

多数元素是指在数组中出现次数大于 ⌊ n/2 ⌋ 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。

示例 1：
输入：nums = [3,2,3]
输出：3
示例 2：
输入：nums = [2,2,1,1,1,2,2]
输出：2
数组 计数 Boyer-Moore O(1)空间

💡 核心思想

Boyer-Moore投票算法的核心思想：把众数想象成"多数派"，其他数想象成"少数派"，两两抵消，最后剩下的就是多数派。

算法维护一个候选元素 candidate 和计数器 count：

  • 遇到相同元素，计数 +1
  • 遇到不同元素，计数 -1
  • 计数归零时，更换候选元素

由于多数元素出现次数 > n/2，它一定能"存活"到最后。

  • 核心直觉：多数元素的"票数"一定多于其他所有元素之和
