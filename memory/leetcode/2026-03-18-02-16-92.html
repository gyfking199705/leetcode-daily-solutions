LeetCode #92 - 反转链表 II

Medium 链表 | 模拟 | 重点掌握：如何定位到反转起点并处理边界情况

📋 题目描述

给你单链表的头指针 head 和两个整数 left 和 right ，其中 left <= right。请你反转从位置 left 到位置 right 的链表节点，返回反转后的链表。

示例：
输入: head = [1,2,3,4,5], left = 2, right = 4
输出: [1,4,3,2,5]
反转前: 1 → 2 → 3 → 4 → 5 → null
反转区间 [2,4]:
          ↑_________↓
反转后: 1 → 4 → 3 → 2 → 5 → null

核心考点：这道题考察链表操作的「边界意识」——如何处理 left=1（从头开始反转）的情况，以及如何正确连接反转后的子链表。

💡 解题技巧总结

四指针法：
1. pre - 反转区间前一个节点（用于连接反转后的头）
2. curr - 当前处理的节点
3. tail - 反转前的第一个节点（反转后变成尾部）
4. next - 临时保存下一个节点

关键思维：链表反转的本质是改变 next 指针的方向。对于局部反转，我们需要记录「反转前的一个节点」和「反转后的第一个节点」，以便正确拼接。

边界处理：当 left=1 时，pre 为 null，需要特殊处理返回的新头节点。
