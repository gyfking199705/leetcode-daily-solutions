LeetCode #416 - 分割等和子集

Partition Equal Subset Sum MEDIUM

📋 题目描述

给定一个只包含正整数的非空数组 nums 。请判断是否可以将这个数组分割成两个子集，使得两个子集的元素和相等。

📝 示例

输入：nums = [1, 5, 11, 5]

输出：true

解释：数组可以分割成 [1, 5, 5] 和 [11]，两者和都是 11

💡 核心思路

  • 如果数组总和 sum 为奇数，直接返回 false
  • 问题转化为：能否从数组中选出若干个数，使其和等于 sum / 2
  • 这是经典的 01背包问题：背包容量为 sum/2，物品重量和价值都是 nums[i]

🎬 算法执行过程（点击步骤或代码行互相高亮）

■ 绿色表示该和可以被组成   ■ 黄色表示目标和

Python 实现


1class Solution:
2    def canPartition(self, nums: List[int]) -> bool:
3        total = sum(nums)
4        # 总和为奇数，无法平分
5        if total % 2 != 0:
6            return False
7        
8        target = total // 2
9        n = len(nums)
10        
11        # dp[i] 表示和为 i 是否可以被组成
12        dp = [False] * (target + 1)
13        dp[0] = True  # 和为0总是可以被组成（不选任何数）
14        
15        for num in nums:
16            # 倒序遍历，避免重复使用同一个数
17            for j in range(target, num - 1, -1):
18                if dp[j - num]:
19                    dp[j] = True
20                    if j == target:  # 提前退出
21                        return True
22        
23        return dp[target]

📊 复杂度分析

维度 值 说明
时间复杂度 O(n × target) target = sum/2，需要遍历每个数字和每个可能的和
空间复杂度 O(target) 一维DP数组，通过倒序遍历实现空间优化

🔍 关键点解析

为什么要「倒序遍历」？

01背包问题的核心是每个物品只能使用一次。如果正序遍历，当我们更新 dp[j] 时， 可能会用到本轮已经更新过的 dp[j-num]，导致同一个数字被重复使用。 倒序遍历确保每个数字只被使用一次。

空间优化原理

二维DP dp[i][j] 表示前i个数字能否组成和j。但实际上，每一行只依赖上一行的值， 因此可以压缩为一维，通过倒序更新避免覆盖还需要使用的旧值。
