📈 买卖股票的最佳时机

Best Time to Buy and Sell Stock

LeetCode #121 Easy 动态规划 / 贪心

📋 题目描述

给定一个数组 prices，它的第 i 个元素 prices[i] 表示一支给定股票第 i 天的价格。

你只能选择 某一天 买入这只股票，并选择在 未来的某一个不同的日子 卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 0。

示例 1：

输入：prices = [7, 1, 5, 3, 6, 4]

输出：5

解释：在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5。

注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格；同时，你不能在买入前卖出股票。

示例 2：

输入：prices = [7, 6, 4, 3, 1]

输出：0

解释：在这种情况下, 没有交易完成, 所以最大利润为 0。

💡 解题思路

💡 核心思路

  • 我们需要找到 最低的买入价格 和 之后的最高卖出价格
  • 遍历数组时，维护一个变量记录 历史最低价
  • 每天计算「当天卖出」的利润，更新最大利润
max_profit = max(max_profit, price[i] - min_price)
min_price = min(min_price, price[i])

📝 代码实现

方法一：单次遍历（最优解）

▼ 
1class Solution:


2    def maxProfit(self, prices: List[int]) -> int:


3        # 边界情况：空数组或只有一个价格


4        if len(prices) < 2:


5            return 0


6        


7        # 初始化：最低买入价格为第一天


8        min_price = prices[0]


9        max_profit = 0


10        


11        # 遍历每一天的价格


12        for price in prices:


13            # 更新历史最低买入价


14            if price < min_price:


15                min_price = price


16            


17            # 计算当天卖出的利润，更新最大利润


18            profit = price - min_price


19            if profit > max_profit:


20                max_profit = profit


21        


22        return max_profit

方法二：Pythonic 简洁写法

▼ 
1class Solution:


2    def maxProfit(self, prices: List[int]) -> int:


3        min_price = float('inf')


4        max_profit = 0


5        


6        for price in prices:


7            min_price = min(min_price, price)


8            max_profit = max(max_profit, price - min_price)


9        


10        return max_profit

方法三：动态规划（完整 DP 思路）

▼ 
1class Solution:


2    def maxProfit(self, prices: List[int]) -> int:


3        n = len(prices)


4        if n < 2:


5            return 0


6        


7        # dp[i][0]: 第 i 天持有股票的最大收益


8        # dp[i][1]: 第 i 天不持有股票的最大收益


9        dp = [[0, 0] for _ in range(n)]


10        


11        # 初始状态


12        dp[0][0] = -prices[0]  # 第0天买入


13        dp[0][1] = 0           # 第0天不买


14        


15        for i in range(1, n):


16            # 持有：昨天就持有 vs 今天买入


17            dp[i][0] = max(dp[i-1][0], -prices[i])


18            # 不持有：昨天就不持有 vs 今天卖出


19            dp[i][1] = max(dp[i-1][1], dp[i-1][0] + prices[i])


20        


21        return dp[n-1][1]
