LeetCode #78 - 子集 Medium

回溯算法的经典入门题，理解「选或不选」的决策树

📋 题目描述

给你一个整数数组 nums，数组中的元素 互不相同。返回该数组所有可能的子集（幂集）。

解集不能包含重复的子集。你可以按 任意顺序 返回解集。

示例 1：
输入：nums = [1,2,3]
输出：[[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]

示例 2：
输入：nums = [0]
输出：[[],[0]]

提示：

  • 1 <= nums.length <= 10
  • -10 <= nums[i] <= 10
  • nums 中的所有元素 互不相同
💡 核心思路
对于每个元素，我们都有两种选择：选 或 不选。
n个元素就有 2n 种组合，正好对应所有子集的数量。

🌲 决策树可视化

以 nums = [1, 2, 3] 为例，每个节点的决策如下：

[] / \ 选1 / \ 不选1 / \ [1] [] / \ / \ 选2/ \不选2 选2/ \不选2 / \ / \ [1,2] [1] [2] [] / \ / \ / \ / \ 选3/不\不选3 选3/不\不选3 选3/不\不选3 选3/不\不选3 / \ / \ / \ / \ [1,2,3][1,2][1,3][1][2,3][2] [3] []

每个叶子节点代表一个子集，共 23 = 8 个子集。

📊 复杂度分析

维度 复杂度 说明
时间复杂度 O(n × 2n) 共 2n 个子集，每个子集平均长度 n/2，复制到结果需要 O(n)
空间复杂度 O(n) 递归栈深度为 n（不考虑存储结果的 O(n × 2n) 空间）
🔍 为什么是 O(n × 2n)？
子集数量：每个元素选或不选，共 2n 个
每个子集平均长度：n/2
总时间 = 2n × O(n) = O(n × 2n)

🎯 解题技巧总结

  • 回溯三要素：路径(path)、选择列表(nums[start:])、结束条件
  • 做选择：path.append(nums[i])
  • 撤销选择：path.pop()（回溯的关键）
  • 剪枝优化：本题无需剪枝，所有路径都是有效解
  • 迭代解法：也可以用迭代，逐个添加元素扩展已有子集

💻 其他解法

迭代法（非递归）

1def subsets(self, nums: List[int]) -> List[List[int]]:
2    result = [[]]  # 初始：空集
3    for num in nums:
4        # 将当前数字加入所有已有子集，生成新子集
5        new_subsets = [subset + [num] for subset in result]
6        result.extend(new_subsets)
7    return result

位运算法（最优雅）

1def subsets(self, nums: List[int]) -> List[List[int]]:
2    n = len(nums)
3    result = []
4    # 枚举 0 到 2^n - 1，每个数字代表一个子集
5    for mask in range(1 << n):  # 1 << n = 2^n
6        subset = []
7        for i in range(n):
8            if mask & (1 << i):  # 第 i 位是否为 1
9                subset.append(nums[i])
10        result.append(subset)
11    return result
