有效的括号 · Valid Parentheses
Easy 栈 字符串

📋 题目描述

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s，判断字符串是否有效。


有效字符串需满足：

  1. 左括号必须用相同类型的右括号闭合。
  2. 左括号必须以正确的顺序闭合。
  3. 每个右括号都有一个对应的相同类型的左括号。
示例 1:

输入: s = "()"

输出: true

示例 2:

输入: s = "()[]{}"

输出: true

示例 3:

输入: s = "(]"

输出: false

解释: 左括号 '(' 没有被正确闭合（应该是 ')' 而不是 ']'）

示例 4:

输入: s = "([)]"

输出: false

解释: 括号交叉，顺序不正确

示例 5:

输入: s = "{[]}"

输出: true

解释: 嵌套括号，正确闭合

💡 解题思路

🎯 核心思想

遇到左括号就入栈，遇到右括号就与栈顶匹配。栈的「后进先出」特性完美契合括号的「就近匹配」规则。

详细算法步骤 ▼
步骤 1: 初始化一个空栈，建立括号映射表：')'→'(', ']'→'[', '}'→'{'
步骤 2: 遍历字符串中的每个字符
步骤 3: 如果遇到左括号 '(' '[' '{'，直接入栈
步骤 4: 如果遇到右括号：
  • 若栈为空 → 返回 false（没有匹配的左括号）
  • 弹出栈顶元素，检查是否与当前右括号匹配
  • 若不匹配 → 返回 false
步骤 5: 遍历结束后，检查栈是否为空
  • 栈为空 → 所有括号都匹配成功，返回 true
  • 栈非空 → 有左括号未闭合，返回 false

📝 代码实现

Python3 时间: O(n) | 空间: O(n) 
1class Solution:
2    def isValid(self, s: str) -> bool:
3        # 括号映射表：右括号 -> 对应的左括号
4        bracket_map = {')': '(', ']': '[', '}': '{'}
5        
6        # 用列表模拟栈
7        stack = []
8        
9        # 遍历每个字符
10        for char in s:
11            if char in bracket_map:
12                # 遇到右括号
13                if not stack:
14                    # 栈为空，没有可匹配的左括号
15                    return False
16                
17                # 弹出栈顶并检查是否匹配
18                top = stack.pop()
19                if top != bracket_map[char]:
20                    # 不匹配
21                    return False
22            else:
23                # 遇到左括号，入栈
24                stack.append(char)
25        
26        # 栈为空说明全部匹配成功
27        return len(stack) == 0
💡 代码技巧
  • 哈希表优化：用字典存储右括号到左括号的映射，避免多个if判断
  • 提前返回：发现不匹配立即返回false，无需继续遍历
  • 边界处理：检查栈是否为空（避免pop空栈错误）

🎓 面试要点

常考追问 ▼
Q1: 如果只有小括号，可以优化吗？
A: 可以用计数器替代栈，空间优化到 O(1)。遇到 '(' 计数+1，')' 计数-1，过程中不能为负，最终必须为0。
Q2: 如何找到不匹配的位置？
A: 记录遍历索引，失败时返回当前索引。或用栈存储 (索引, 字符) 元组。
Q3: 字符串很长时的优化？
A: 可以先检查长度是否为偶数（奇数直接返回false）；也可以预分配栈空间避免动态扩容。
