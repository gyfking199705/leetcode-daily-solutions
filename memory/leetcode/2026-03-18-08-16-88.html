LeetCode #88 - 合并两个有序数组

Merge Sorted Array · Easy · 数组 · 双指针

Easy 数组 双指针 原地算法

📋 题目描述

给你两个按 非递减顺序 排列的整数数组 nums1 和 nums2，另有两个整数 m 和 n，分别表示 nums1 和 nums2 中的元素数目。

请你 合并 nums2 到 nums1 中，使合并后的数组同样按 非递减顺序 排列。

注意：最终，合并后数组不应由函数返回，而是存储在数组 nums1 中。为了应对这种情况，nums1 的初始长度为 m + n，其中前 m 个元素表示应合并的元素，后 n 个元素为 0，应忽略。nums2 的长度为 n。

💡 核心思想

从后往前填充！ 常规思路是从前往后合并，但会覆盖 nums1 的元素。改为从后往前，从两个数组的末尾开始比较，将较大的元素放到 nums1 的末尾，这样可以避免覆盖问题。

🎬 算法执行过程（点击步骤或代码行查看对应关系）

示例：nums1 = [1,2,3,0,0,0], m=3 | nums2 = [2,5,6], n=3

指针 q (nums2末尾) 2

Python 实现


1class Solution:
2    def merge(self, nums1: list[int], m: int, nums2: list[int], n: int) -> None:
3        # 初始化三个指针：p指向nums1有效元素末尾，q指向nums2末尾，k指向填充位置
4        p, q, k = m - 1, n - 1, m + n - 1
5        
6        # 当两个数组都还有元素时，比较并填充
7        while p >= 0 and q >= 0:
8            if nums1[p] > nums2[q]:
9                nums1[k] = nums1[p]  # nums1[p]更大，放到末尾
10                p -= 1
11            else:
12                nums1[k] = nums2[q]  # nums2[q]更大或相等，放到末尾
13                q -= 1
14            k -= 1
15        
16        # 如果nums2还有剩余，全部复制到nums1前面
17        while q >= 0:
18            nums1[k] = nums2[q]
19            q -= 1
20            k -= 1
21        # 不需要处理nums1剩余，因为已经在原地了

时间复杂度 O(m + n) 只需要遍历两个数组各一次

🔍 易错点 & 面试要点

  • 为什么从后往前？ 如果从前往后，合并时会覆盖 nums1 中尚未处理的元素。从后往前利用 nums1 尾部的空闲空间，避免覆盖。
  • 边界情况： nums1 或 nums2 为空；所有 nums2 元素都大于 nums1；所有 nums2 元素都小于 nums1。
  • 为什么最后只检查 q >= 0？ 如果 p >= 0（nums1 有剩余），这些元素已经在正确位置，无需移动。
  • 变体： 如果不允许修改 nums1 呢？那就需要从前往后，使用额外数组存储结果。
