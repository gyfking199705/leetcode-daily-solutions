LeetCode #5

最长回文子串 · Longest Palindromic Substring

📋 题目描述

给你一个字符串 s，找到 s 中最长的回文子串。

回文串是正着读和反着读都一样的字符串。如果字符串的反序与原始字符串相同，则该字符串称为回文字符串。

示例 1：

输入：s = "babad"

输出："bab"

解释："aba" 同样是符合题意的答案。

示例 2：

输入：s = "cbbd"

输出："bb"

约束条件：

  • 1 <= s.length <= 1000
  • s 仅由数字和英文字母组成

💡 解题思路

回文串的核心特性是对称。对于每个字符（或字符间隙），我们可以将其视为"中心"，然后向两边扩展，直到不再满足回文条件。

🎯 关键观察

  • 回文串的中心可能是一个字符（奇数长度，如 "aba"）
  • 也可能是两个相同字符之间（偶数长度，如 "abba"）

💡 核心技巧：对于长度为 n 的字符串，我们需要检查 2n-1 个中心位置（n 个字符中心 + n-1 个间隙中心）。

设 dp[i][j] 表示子串 s[i:j+1] 是否为回文串。

状态转移方程

dp[i][j] = (s[i] == s[j]) && (j - i < 3 || dp[i+1][j-1])
解释：
• s[i] == s[j]：首尾字符相同
• j - i < 3：子串长度 ≤ 3，无需再检查内部
• dp[i+1][j-1]：内部子串也是回文

按子串长度从小到大填充 DP 表，记录最长回文子串。

Manacher 算法是线性时间求解最长回文子串的经典算法，通过以下技巧达到 O(n)：

  • 字符预处理：在每对字符间插入特殊字符（如 #），统一奇偶长度处理
  • 对称性利用：维护当前最右回文边界，利用对称性避免重复计算
  • 半径数组：p[i] 表示以 i 为中心的回文半径
面试建议：掌握中心扩展法即可应对大部分面试，Manacher 作为加分项了解原理。

class Solution:
    def longestPalindrome(self, s: str) -> str:
        if not s or len(s) < 2:
            return s
        
        start, max_len = 0, 1
        
        for i in range(len(s)):
            # 奇数长度扩展（中心是单个字符）
            len1 = self.expandAroundCenter(s, i, i)
            # 偶数长度扩展（中心是两个字符之间）
            len2 = self.expandAroundCenter(s, i, i + 1)
            
            curr_len = max(len1, len2)
            
            if curr_len > max_len:
                max_len = curr_len
                # 计算起始位置
                # 当前中心 i, 总长度 curr_len
                # 左边界 = i - (curr_len - 1) // 2
                start = i - (curr_len - 1) // 2
        
        return s[start:start + max_len]
    
    def expandAroundCenter(self, s: str, left: int, right: int) -> int:
        # 从中心向两边扩展，返回回文串长度
        while left >= 0 and right < len(s) and s[left] == s[right]:
            left -= 1
            right += 1
        
        # 退出时 left 和 right 都多走了一步
        return right - left - 1

class Solution:
    def longestPalindrome(self, s: str) -> str:
        n = len(s)
        if n < 2:
            return s
        
        # dp[i][j] 表示 s[i:j+1] 是否为回文
        dp = [[False] * n for _ in range(n)]
        start, max_len = 0, 1
        
        # 单个字符都是回文
        for i in range(n):
            dp[i][i] = True
        
        # 按子串长度从小到大遍历
        for length in range(2, n + 1):
            for i in range(n - length + 1):
                j = i + length - 1
                if s[i] == s[j] and (length <= 3 or dp[i + 1][j - 1]):
                    dp[i][j] = True
                    if length > max_len:
                        max_len = length
                        start = i
        
        return s[start:start + max_len]

🎓 核心要点总结

关键技巧速记：

1️⃣ 中心扩展：每个位置作为中心，向两边扩散，注意处理奇偶两种情况

2️⃣ 状态定义：DP[i][j] = s[i:j+1] 是否为回文

3️⃣ 转移方程：dp[i][j] = (s[i]==s[j]) && (j-i<3 || dp[i+1][j-1])

4️⃣ 边界处理：单个字符必为回文，两个字符需判断是否相等

5️⃣ 优化思路：对称性利用、Manacher 的镜像优化
