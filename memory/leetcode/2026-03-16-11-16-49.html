🎯 LeetCode #49 - 字母异位词分组

Medium 哈希表 · 字符串 · 排序

📋 题目描述

给你一个字符串数组 strs，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

字母异位词 是由重新排列源单词的所有字母得到的一个新单词。

示例 1： 
输入: strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
输出: [["bat"],["nat","tan"],["ate","eat","tea"]]
示例 2： 
输入: strs = [""]
输出: [[""]]
示例 3： 
输入: strs = ["a"]
输出: [["a"]]

💡 核心思路

字母异位词的特点是：排序后的字符串相同。

例如："eat"、"tea"、"ate" 排序后都是 "aet"。

因此，我们可以：

  1. 遍历每个字符串，将其排序后作为 哈希表的键（key）
  2. 将原字符串添加到对应键的值列表中
  3. 最后返回哈希表的所有值

📊 复杂度分析

维度 复杂度 说明
时间复杂度 O(N × K log K) N为字符串数量，K为字符串最大长度。对每个字符串排序耗时O(K log K)
空间复杂度 O(N × K) 哈希表存储所有字符串

🚀 优化思路

计数法优化（O(N × K)）：

用字符计数代替排序。将每个字符串的26个字母计数组成元组作为键，避免排序的 log K 开销。

def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
    groups = defaultdict(list)
    for s in strs:
        count = [0] * 26
        for c in s:
            count[ord(c) - ord('a')] += 1
        groups[tuple(count)].append(s)
    return list(groups.values())
