LeetCode #34 - 在排序数组中查找元素的第一个和最后一个位置

Medium 二分查找经典变种 | 时间复杂度 O(log n) | 空间复杂度 O(1)

📋 题目描述

给你一个按照非递减顺序排列的整数数组 nums，和一个目标值 target。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 target，返回 [-1, -1]。

你必须设计并实现时间复杂度为 O(log n) 的算法解决此问题。

💡 核心考点：二分查找的变形应用——分别查找左边界和右边界。这是面试中最常考的二分查找变种之一。

示例

输入：nums = [5,7,7,8,8,10], target = 8
输出：[3,4]

输入：nums = [5,7,7,8,8,10], target = 6
输出：[-1,-1]

输入：nums = [], target = 0
输出：[-1,-1]

📊 复杂度分析

算法 时间复杂度 空间复杂度 说明
暴力解法 O(n) O(1) 线性扫描，不满足题目要求
二分查找（推荐） O(log n) O(1) 两次二分，分别找左右边界

🎯 关键思路

二分查找左边界：当 nums[mid] == target 时，不立即返回，而是收缩右边界，继续向左查找更小的索引。
二分查找右边界：当 nums[mid] == target 时，不立即返回，而是收缩左边界，继续向右查找更大的索引。

通过两次二分查找，分别找到 target 的最左出现位置和最右出现位置。

# 左边界查找：找第一个等于 target 的位置
def findLeftBound(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = left + (right - left) // 2
        if nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return left if left < len(nums) and nums[left] == target else -1

# 右边界查找：找最后一个等于 target 的位置
def findRightBound(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = left + (right - left) // 2
        if nums[mid] <= target:
            left = mid + 1
        else:
            right = mid - 1
    return right if right >= 0 and nums[right] == target else -1
