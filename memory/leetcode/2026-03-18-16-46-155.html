LeetCode #155 - 最小栈

Medium 栈 | 设计 | 辅助数据结构

📋 题目描述

设计一个支持 push、pop、top 操作，并能在常数时间内检索到最小元素的栈。

实现 MinStack 类：

  • MinStack() 初始化堆栈对象
  • void push(int val) 将元素 val 推入堆栈
  • void pop() 删除堆栈顶部的元素
  • int top() 获取堆栈顶部的元素
  • int getMin() 获取堆栈中的最小元素

示例：

MinStack minStack = new MinStack();
minStack.push(-2);
minStack.push(0);
minStack.push(-3);
minStack.getMin();   --> 返回 -3
minStack.pop();
minStack.top();      --> 返回 0
minStack.getMin();   --> 返回 -2

💡 核心思路

问题的关键在于：当栈顶元素被弹出后，如何知道剩余元素中的最小值？

解决方案：辅助栈

使用两个栈：

  • 主栈 (stack)：存储所有元素，支持常规的 push/pop/top
  • 辅助栈 (min_stack)：存储每个状态下的最小值

关键技巧：辅助栈的栈顶永远是当前所有元素的最小值。push 时，将当前元素与辅助栈顶比较，较小者入辅助栈；pop 时，两栈同时出栈。

📊 复杂度分析

操作 时间复杂度 空间复杂度 说明
push O(1) O(1) 两栈各执行一次入栈
pop O(1) O(1) 两栈各执行一次出栈
top O(1) O(1) 直接返回主栈栈顶
getMin O(1) O(1) 直接返回辅助栈栈顶

总体空间复杂度：O(n) - 辅助栈最坏情况下存储 n 个元素（当输入递减时）

🔧 空间优化版本

当遇到重复的最小值时，可以只在辅助栈中存储一个计数，进一步优化空间：

1class MinStackOptimized:


2    def __init__(self):


3        self.stack = []      # 主栈


4        self.min_stack = []  # 辅助栈存储 (最小值, 计数)


5


6    def push(self, val: int) -> None:


7        self.stack.append(val)


8        if not self.min_stack or val < self.min_stack[-1][0]:


9            self.min_stack.append((val, 1))  # 新最小值


10        elif val == self.min_stack[-1][0]:


11            self.min_stack[-1] = (val, self.min_stack[-1][1] + 1)  # 计数+1


12


13    def pop(self) -> None:


14        val = self.stack.pop()


15        if val == self.min_stack[-1][0]:


16            count = self.min_stack[-1][1] - 1


17            if count == 0:


18                self.min_stack.pop()


19            else:


20                self.min_stack[-1] = (val, count)
