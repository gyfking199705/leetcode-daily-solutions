LeetCode #148 - 排序链表

在 O(n log n) 时间复杂度和常数级空间复杂度下，对链表进行排序

Medium 归并排序 O(1) 空间

📋 题目描述

给你链表的头结点 head ，请将其按 升序 排列并返回 排序后的链表 。

进阶要求： 你可以在 O(n log n) 时间复杂度和 O(1) 空间复杂度下完成此题吗？

示例 1：

输入：head = [4,2,1,3]
输出：[1,2,3,4]

示例 2：

输入：head = [-1,5,3,4,0]
输出：[-1,0,3,4,5]

约束条件：

  • 链表中节点的数目在范围 [0, 5 * 10^4] 内
  • -10^5 <= Node.val <= 10^5

💡 核心思路

链表排序的最佳选择是归并排序，原因：

  • 时间复杂度 O(n log n) - 满足题目要求
  • 空间复杂度 O(1) - 使用迭代版本的归并排序可以达到
  • 适合链表 - 归并排序不需要随机访问，天然适合链表结构

🎯 关键技巧总结

  1. 快慢指针找中点：fast 走两步，slow 走一步，当 fast 到末尾时，slow 就在中点
  2. 断开链表：找到中点后，将链表分成两个独立的部分，避免循环引用
  3. 合并有序链表：使用虚拟头节点简化边界处理，双指针逐个比较
  4. 递归终止条件：当链表只有一个节点或为空时直接返回
