LeetCode #91 - 解码方法 Medium

动态规划经典题目 | 字符串解码 | 斐波那契思想

📋 题目描述

一条包含字母 A-Z 的消息通过以下映射进行了 编码：

'A' -> "1"
'B' -> "2"
...
'Z' -> "26"

要 解码 已编码的消息，所有数字必须基于上述映射的方法，反向映射回字母（可能有多种方法）。例如，"11106" 可以映射为：

  • "AAJF"，将消息分组为 (1 1 10 6)
  • "KJF"，将消息分组为 (11 10 6)

注意，消息不能分组为 (1 11 06)，因为 "06" 不能映射为 'F'，这是由于 '6' 和 '06' 在映射中并不等价。

给你一个只含数字的 非空 字符串 s ，请计算并返回 解码 方法的 总数 。

步骤1: 初始化DP数组

dp[0] = 1（空字符串有1种解码方式）

dp = [1, 1, ?, ?]

步骤2: 进入迭代（i=2）

处理子串"22"，考虑最后一位"2"单独解码

i=2, one_digit=2, dp[2] = 0

步骤3: 单独解码检查

检查 s[i-1]='2' 是否有效（1-9），有效则继承dp[1]

s[1]='2' ∈ [1,9] ✓
dp[2] += dp[1] = 1

步骤4: 组合解码检查

检查 s[0:2]="22" 是否在10-26范围内

two_digits=22 ∈ [10,26] ✓
dp[2] += dp[0] = 2

步骤5: 进入下一轮（i=3）

处理子串"226"，考虑最后一位"6"

i=3, dp[3] = 0

步骤6: 单独解码检查

s[2]='6' 单独有效，继承dp[2]

dp[3] += dp[2] = 2

步骤7: 组合解码检查

"26"在有效范围内，组合解码有效

two_digits=26 ∈ [10,26] ✓
dp[3] += dp[1] = 3

步骤8: 返回结果

遍历完成，dp[3]=3 即为答案

结果: 3种解码方式
(BBF, BV, ZBF)

Python 实现



1class Solution:


2    def numDecodings(self, s: str) -> int:


3        if not s or s[0] == '0':


4            return 0


5        n = len(s)


6        dp = [0] * (n + 1)


7        dp[0], dp[1] = 1, 1  # 初始化


8        for i in range(2, n + 1):


9            # 单独解码：s[i-1] 是 '1'-'9'


10            if '1' <= s[i-1] <= '9':


11                dp[i] += dp[i-1]


12            # 组合解码：s[i-2:i] 是 '10'-'26'


13            two = int(s[i-2:i])


14            if 10 <= two <= 26:


15                dp[i] += dp[i-2]


16        return dp[n]

🧠 算法思路解析

状态定义

dp[i] 表示字符串 s[0:i]（前i个字符）的解码方法总数。

状态转移

对于位置 i，有两种情况可以形成有效解码：

  1. 单独解码：如果 s[i-1] 是 '1'~'9'，则可以单独解码为一个字母，方法数 += dp[i-1]
  2. 组合解码：如果 s[i-2:i] 组成的数字在 10~26 范围内，则可以组合解码，方法数 += dp[i-2]

边界条件

  • dp[0] = 1：空字符串有1种解码方式（什么都不做）
  • dp[1] = 1：如果第一个字符不是 '0'，则有1种解码方式
  • 如果字符串以 '0' 开头，直接返回 0

空间优化

由于 dp[i] 只依赖于 dp[i-1] 和 dp[i-2]，可以用滚动变量将空间复杂度优化到 O(1)：

def numDecodings(self, s: str) -> int:
    if not s or s[0] == '0': return 0
    prev2, prev1 = 1, 1  # dp[i-2], dp[i-1]
    for i in range(1, len(s)):
        curr = 0
        if s[i] != '0': curr += prev1
        if 10 <= int(s[i-1:i+1]) <= 26: curr += prev2
        prev2, prev1 = prev1, curr
    return prev1

💡 面试要点

  • '0'的处理：'0'不能单独解码，也不能作为组合的高位（如"01"无效）
  • 边界陷阱：注意字符串长度为1的情况，以及"10"、"20"这样的特殊情况
  • 模块化思维：单独解码和组合解码是两个独立的条件判断，不要耦合
  • 空间优化：面试官通常会追问能否优化空间，提前准备好滚动变量的写法
  • 大数问题：Python整数无上限，但其他语言需注意溢出（本题不需要取模）
