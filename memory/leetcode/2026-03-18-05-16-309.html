最佳买卖股票时机含冷冻期 | Best Time to Buy and Sell Stock with Cooldown

📋 题目描述

给定一个整数数组 prices，其中第 i 个元素代表了某只股票第 i 天的价格。

设计一个算法来计算你所能获取的最大利润。你可以尽可能地完成更多的交易（多次买卖一支股票），但有以下限制：

  • 卖出股票后，你无法在第二天买入股票（即冷冻期为1天）
  • 你不能同时参与多笔交易（必须在再次购买前出售掉之前的股票）
示例

输入：prices = [1, 2, 3, 0, 2]

输出：3

解释：

  • 第1天买入，第2天卖出 → 利润 = 2-1 = 1
  • 第2天（卖出后）进入冷冻期，第3天不能买入
  • 第4天买入，第5天卖出 → 利润 = 2-0 = 2
  • 总利润 = 1 + 2 = 3

🧠 核心思路：状态机DP

这道题的关键在于理解「冷冻期」带来的状态转移变化。我们需要用状态机的思想来建模每一天的可能状态。

三种状态定义：
  • 持有股票 (hold)：当天结束时持有股票
  • 卖出股票 (sold)：当天卖出股票（下一天进入冷冻期）
  • 冷冻期/不持有 (rest)：当天不持有股票且不在冷冻期
hold 持有
→ 卖出
sold 卖出
→ 冷却
rest 不持有

状态转移方程

状态 转移来源 方程
hold[i] 前一天持有 或 前一天不持有今天买入 max(hold[i-1], rest[i-1] - prices[i])
sold[i] 前一天持有，今天卖出 hold[i-1] + prices[i]
rest[i] 前一天卖出(冷冻结束) 或 前一天就不持有 max(sold[i-1], rest[i-1])

⏱️ 复杂度分析

指标 复杂度 说明
时间复杂度 O(n) 只需遍历一次价格数组，每天的状态转移是 O(1)
空间复杂度 O(1) 只需保存前一天的状态，可以优化为常数空间

💡 关键点总结

  1. 状态机建模：将问题抽象为有限状态机是解题核心
  2. 冷冻期处理：sold 状态只能转移到 rest，不能直接买入
  3. 空间优化：由于只依赖前一天状态，可用滚动变量优化到 O(1)
  4. 初始化：第0天 hold = -prices[0]，sold = 0（不可能），rest = 0
