Hard 动态规划

题目描述

给你两个单词 word1 和 word2， 请返回将 word1 转换成 word2 所使用的最少操作数。

你可以对一个单词进行如下三种操作：

➕
插入一个字符
Insert
➖
删除一个字符
Delete
🔄
替换一个字符
Replace
示例 1
输入: word1 = "horse", word2 = "ros"
输出: 3
解释:
horse → rorse (将 'h' 替换为 'r')
rorse → rose (删除 'r')
rose → ros (删除 'e')
示例 2
输入: word1 = "intention", word2 = "execution"
输出: 5
解释:
intention → inention (删除 't')
inention → enention (将 'i' 替换为 'e')
enention → exention (将 'n' 替换为 'x')
exention → exection (将 'n' 替换为 'c')
exection → execution (插入 'u')

🎯 核心思想

动态规划状态定义

定义 dp[i][j] 表示：

将 word1[0..i-1] 转换为 word2[0..j-1] 所需的最少操作数

1 最后字符相同：dp[i][j] = dp[i-1][j-1]
2 最后字符不同：取三种操作的最小值

状态转移方程

if word1[i-1] == word2[j-1]:
    # 字符相同，无需操作
    dp[i][j] = dp[i-1][j-1]
else:
    # 字符不同，取三种操作的最小值
    dp[i][j] = 1 + min(
        dp[i-1][j],      # 删除 word1[i-1]
        dp[i][j-1],      # 在 word1 插入 word2[j-1]
        dp[i-1][j-1]    # 替换 word1[i-1] 为 word2[j-1]
    )

💻 代码实现

Python3 - 二维DP Time: O(mn) | Space: O(mn) 
class Solution:
    def minDistance(self, word1: str, word2: str) -> int:
        m, n = len(word1), len(word2)
        
        # dp[i][j] = word1[0:i] 转 word2[0:j] 的最小操作数
        dp = [[0] * (n + 1) for _ in range(m + 1)]
        
        # 初始化边界：一个空串变成另一个串需要插入/删除操作
        for i in range(m + 1):
            dp[i][0] = i  # 删除i个字符
        for j in range(n + 1):
            dp[0][j] = j  # 插入j个字符
        
        # 状态转移
        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if word1[i - 1] == word2[j - 1]:
                    # 字符相同，无需操作
                    dp[i][j] = dp[i - 1][j - 1]
                else:
                    # 字符不同，取三种操作的最小值
                    dp[i][j] = 1 + min(
                        dp[i - 1][j],      # 删除
                        dp[i][j - 1],      # 插入
                        dp[i - 1][j - 1]   # 替换
                    )
        
        return dp[m][n]

    int minDistance(string word1, string word2) {
        int m = word1.size(), n = word2.size();
        // dp[i][j] = word1[0:i] 转 word2[0:j] 的最小操作数
        vector<vector<int>> dp(m + 1, vector<int>(n + 1, 0));

        for (int i = 0; i <= m; ++i) dp[i][0] = i;
        for (int j = 0; j <= n; ++j) dp[0][j] = j;
        
        // 状态转移
        for (int i = 1; i <= m; ++i) {

Java - 二维DP Time: O(mn) | Space: O(mn) 
class Solution {
    public int minDistance(String word1, String word2) {
        int m = word1.length(), n = word2.length();
        int[][] dp = new int[m + 1][n + 1];
        
        // 初始化边界
        for (int i = 0; i <= m; i++) dp[i][0] = i;
        for (int j = 0; j <= n; j++) dp[0][j] = j;
        
        // 状态转移
        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (word1.charAt(i - 1) == word2.charAt(j - 1)) {
                    dp[i][j] = dp[i - 1][j - 1];
                } else {
                    dp[i][j] = 1 + Math.min(
                        dp[i - 1][j],      // 删除
                        Math.min(
                            dp[i][j - 1],      // 插入
                            dp[i - 1][j - 1]   // 替换
                        )
                    );
                }
            }
        }
        return dp[m][n];
    }
}
Python - 滚动数组优化 Time: O(mn) | Space: O(min(m,n)) ⭐ 
class Solution:
    def minDistance(self, word1: str, word2: str) -> int:
        # 确保 word1 是较短的，节省空间
        if len(word1) > len(word2):
            word1, word2 = word2, word1
        
        m, n = len(word1), len(word2)
        
        # 只保留两行：当前行和前一行
        prev = list(range(m + 1))  # dp[i-1][*]
        curr = [0] * (m + 1)          # dp[i][*]
        
        for j in range(1, n + 1):
            curr[0] = j  # 边界：空串转 word2[0:j] 需 j 次插入
            for i in range(1, m + 1):
                if word1[i - 1] == word2[j - 1]:
                    curr[i] = prev[i - 1]  # 字符相同
                else:
                    curr[i] = 1 + min(
                        prev[i],      # 删除 (dp[i-1][j])
                        curr[i - 1],  # 插入 (dp[i][j-1])
                        prev[i - 1]   # 替换 (dp[i-1][j-1])
                    )
            # 交换两行
            prev, curr = curr, prev
        
        return prev[m]

📊 复杂度分析

时间复杂度

O(m × n)

需要填充一个 m × n 的二维DP表格，每个状态计算时间为 O(1)。 其中 m = len(word1), n = len(word2)。

空间复杂度

二维数组： O(m × n)
滚动数组优化： O(min(m, n)) ⭐
