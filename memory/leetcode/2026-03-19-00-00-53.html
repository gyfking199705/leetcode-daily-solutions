最大子数组和 (Maximum Subarray)

📋 题目描述

给你一个整数数组 nums，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

子数组 是数组中的一个连续部分。

📝 Problem Statement

Given an integer array nums, find the contiguous subarray (containing at least one number) which has the largest sum and return its sum.

A subarray is a contiguous part of an array.

示例 1 / Example 1
输入: nums = [-2,1,-3,4,-1,2,1,-5,4]
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。
Explanation: The subarray [4,-1,2,1] has the largest sum = 6.
示例 2 / Example 2
输入: nums = [1]
输出: 1
示例 3 / Example 3
输入: nums = [5,4,-1,7,8]
输出: 23
解释: 整个数组的和最大，为 23。
Explanation: The entire array sum is 23.
💡 提示 / Constraints:
• 1 ≤ nums.length ≤ 10⁵
• -10⁴ ≤ nums[i] ≤ 10⁴
• 进阶: 如果你已经实现复杂度为 O(n) 的解法，尝试使用更为精妙的 分治法 求解。

🔨 暴力解法思路

最直观的方法是枚举所有可能的子数组，计算它们的和，然后找出最大值。

  • 使用两层循环：外层循环确定子数组的起始位置
  • 内层循环从起始位置向后遍历，累加元素值
  • 记录遍历过程中遇到的最大和
时间复杂度
O(n²)
空间复杂度
O(1)
⚠️ 注意: 暴力解法在 n = 10⁵ 时会超时，仅用于理解问题。
💻 代码实现 
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        n = len(nums)
        max_sum = float('-inf')
        
        # 枚举所有子数组的起始位置
        for i in range(n):
            current_sum = 0
            # 从位置i开始向后扩展
            for j in range(i, n):
                current_sum += nums[j]
                max_sum = max(max_sum, current_sum)
        
        return max_sum

⚡ Kadane算法 (动态规划)

Kadane算法是一种高效的动态规划解法，核心思想是：如果前面的和对当前元素有增益（即前面和为正），就继续累加；否则从当前元素重新开始。

dp[i] = max(nums[i], dp[i-1] + nums[i])
情况 1: 从当前元素重新开始
dp[i] = nums[i]
当前面子数组和为负数时
情况 2: 延续前面子数组
dp[i] = dp[i-1] + nums[i]
当前面子数组和为正数时

🔄 状态转移过程图解

数组: [-2, 1, -3, 4, -1, 2, 1, -5, 4]
i 0 1 2 3 4 5 6 7 8
nums[i] -2 1 -3 4 -1 2 1 -5 4
dp[i] -2 1 -2 4 3 5 6 1 5
max -2 1 1 4 4 5 6 6 6
dp[i] 表示以第 i 个元素结尾的最大子数组和
时间复杂度
O(n)
空间复杂度
O(1)
✅ 优化: 由于 dp[i] 只依赖于 dp[i-1]，可以用一个变量代替数组，将空间复杂度优化到 O(1)。
💻 代码实现 (优化版) 
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        # 初始化：当前子数组和与最大和都设为第一个元素
        current_sum = max_sum = nums[0]
        
        # 从第二个元素开始遍历
        for i in range(1, len(nums)):
            # 决策：是延续前面的子数组，还是从当前重新开始
            # 如果 current_sum > 0，说明对后续有增益，继续累加
            # 如果 current_sum <= 0，说明会拖累后续，从当前重新开始
            current_sum = max(nums[i], current_sum + nums[i])
            
            # 更新全局最大值
            max_sum = max(max_sum, current_sum)
        
        return max_sum

🔄 分治法思路

分治法的核心思想是将数组分成左右两部分，最大子数组和可能出现在三个位置：

  • 完全在左半部分：递归求解左半部分的最大子数组和
  • 完全在右半部分：递归求解右半部分的最大子数组和
  • 跨越中点：从中点向左找最大和，向右找最大和，两者相加

📊 跨越中点的子数组计算

对于跨越中点的子数组，我们需要：

  1. 从中点向左遍历，找到以中点结尾的最大子数组和
  2. 从中点+1向右遍历，找到以中点+1开始的最大子数组和
  3. 两者相加即为跨越中点的最大子数组和
时间复杂度
O(n log n)
空间复杂度
O(log n)
💡 说明: 时间复杂度满足递推式 T(n) = 2T(n/2) + O(n)，根据主定理为 O(n log n)。空间复杂度主要是递归栈空间。
💻 代码实现 
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        return self.divide_conquer(nums, 0, len(nums) - 1)
    
    def divide_conquer(self, nums: List[int], left: int, right: int) -> int:
        # 递归终止条件：只有一个元素
        if left == right:
            return nums[left]
        
        # 计算中点
        mid = left + (right - left) // 2
        
        # 递归求解左半部分的最大子数组和
        left_max = self.divide_conquer(nums, left, mid)
        
        # 递归求解右半部分的最大子数组和
        right_max = self.divide_conquer(nums, mid + 1, right)
        
        # 计算跨越中点的最大子数组和
        cross_max = self.max_crossing_sum(nums, left, mid, right)
        
        # 返回三者中的最大值
        return max(left_max, right_max, cross_max)
    
    def max_crossing_sum(self, nums: List[int], left: int, mid: int, right: int) -> int:
        # 从中点向左找最大和
        left_sum = float('-inf')
        current_sum = 0
        for i in range(mid, left - 1, -1):
            current_sum += nums[i]
            left_sum = max(left_sum, current_sum)
        
        # 从中点向右找最大和
        right_sum = float('-inf')
        current_sum = 0
        for i in range(mid + 1, right + 1):
            current_sum += nums[i]
            right_sum = max(right_sum, current_sum)
        
        # 跨越中点的子数组和 = 左边最大和 + 右边最大和
        return left_sum + right_sum
class Solution {
    public int maxSubArray(int[] nums) {
        return divideConquer(nums, 0, nums.length - 1);
    }
    
    private int divideConquer(int[] nums, int left, int right) {
        // 递归终止条件：只有一个元素
        if (left == right) {
            return nums[left];
        }
        
        // 计算中点
        int mid = left + (right - left) / 2;
        
        // 递归求解左半部分的最大子数组和
        int leftMax = divideConquer(nums, left, mid);
        
        // 递归求解右半部分的最大子数组和
        int rightMax = divideConquer(nums, mid + 1, right);
        
        // 计算跨越中点的最大子数组和
        int crossMax = maxCrossingSum(nums, left, mid, right);
        
        // 返回三者中的最大值
        return Math.max(Math.max(leftMax, rightMax), crossMax);
    }
    
    private int maxCrossingSum(int[] nums, int left, int mid, int right) {
        // 从中点向左找最大和
        int leftSum = Integer.MIN_VALUE;
        int currentSum = 0;
        for (int i = mid; i >= left; i--) {
            currentSum += nums[i];
            leftSum = Math.max(leftSum, currentSum);
        }
        
        // 从中点向右找最大和
        int rightSum = Integer.MIN_VALUE;
        currentSum = 0;
        for (int i = mid + 1; i <= right; i++) {
            currentSum += nums[i];
            rightSum = Math.max(rightSum, currentSum);
        }
        
        // 跨越中点的子数组和
        return leftSum + rightSum;
    }
}

function divideConquer(nums, left, right) {
    // 递归终止条件：只有一个元素
    if (left === right) {
        return nums[left];
    }
    
    // 计算中点
    const mid = left + Math.floor((right - left) / 2);
    
    // 递归求解左半部分的最大子数组和
    const leftMax = divideConquer(nums, left, mid);
    
    // 递归求解右半部分的最大子数组和
    const rightMax = divideConquer(nums, mid + 1, right);
    
    // 计算跨越中点的最大子数组和
    const crossMax = maxCrossingSum(nums, left, mid, right);
    
    // 返回三者中的最大值
    return Math.max(leftMax, rightMax, crossMax);
}

function maxCrossingSum(nums, left, mid, right) {
    // 从中点向左找最大和
    let leftSum = -Infinity;
    let currentSum = 0;
    for (let i = mid; i >= left; i--) {
        currentSum += nums[i];
        leftSum = Math.max(leftSum, currentSum);
    }
    
    // 从中点向右找最大和
    let rightSum = -Infinity;
    currentSum = 0;
    for (let i = mid + 1; i <= right; i++) {
        currentSum += nums[i];
        rightSum = Math.max(rightSum, currentSum);
    }
    
    // 跨越中点的子数组和
    return leftSum + rightSum;
}

⚠️ 边界条件讨论

  • 全负数数组：如 [-3, -2, -5]，最大子数组是其中最大的元素 [-2]，结果为 -2
  • 单元素数组：直接返回该元素
  • 全正数数组：最大子数组是整个数组
  • 大数溢出：在静态类型语言中需考虑整数溢出，可使用 long long (C++) 或 BigInteger (Java)
  • 空数组：根据题目要求，本题保证至少有一个元素
