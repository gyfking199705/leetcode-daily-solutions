LeetCode #151 - 反转字符串中的单词

Reverse Words in a String

中等 双指针 字符串 原地操作

📋 题目描述

给你一个字符串 s ，请你反转字符串中 单词 的顺序。

单词 是由非空格字符组成的字符串。s 中使用至少一个空格将字符串中的 单词 分隔开。

返回 单词 顺序颠倒且 单词 之间用单个空格连接的结果字符串。

注意：输入字符串 s 中可能会存在前导空格、尾随空格或者单词间的多个空格。返回的结果字符串中，单词间应当仅用单个空格分隔，且不包含任何额外的空格。

示例 1：

输入：s = "the sky is blue"

输出："blue is sky the"

示例 2：

输入：s = " hello world "

输出："world hello"

解释：反转后的字符串中不能存在前导空格和尾随空格。

示例 3：

输入：s = "a good example"

输出："example good a"

解释：如果两个单词间有多余的空格，反转后的字符串需要将单词间的空格减少到仅有一个。

🧠 算法思路

这道题的经典解法是「双指针法」，整体思路分为三步：

1
去除首尾空格
→
2
翻转整个字符串
→
3
翻转每个单词

核心思想：两次翻转法

  • 将整个字符串翻转后，单词的顺序就对了，但每个单词内部的字符是反的
  • 再将每个单词内部翻转一次，就得到了最终结果
  • 同时需要处理多余空格：通过双指针将有效字符移动到数组前面
演示："the sky is blue"
theskyisblue
↓ 翻转整个字符串
eulbsiykseht
↓ 翻转每个单词
blueisskythe

🎬 算法执行过程

Python 实现


1class Solution:
2    def reverseWords(self, s: str) -> str:
3        # 去除首尾空格并转为字符列表
4        chars = list(s.strip())
5        n = len(chars)
6        
7        # 步骤1：清理多余空格（双指针）
8        i, j = 0, 0
9        while j < n:
10            if chars[j] != ' ':
11                # 不是空格，直接复制
12                chars[i] = chars[j]
13                i += 1
14            elif i > 0 and chars[i-1] != ' ':
15                # 是空格，但前一个不是空格（单词间单个空格）
16                chars[i] = chars[j]
17                i += 1
18            j += 1
19        
20        # 截取有效部分
21        chars = chars[:i]
22        n = len(chars)
23        
24        # 步骤2：翻转整个字符串
25        self._reverse(chars, 0, n - 1)
26        
27        # 步骤3：翻转每个单词
28        start = 0
29        while start < n:
30            # 找单词结尾（空格或字符串末尾）
31            end = start
32            while end < n and chars[end] != ' ':
33                end += 1
34            # 翻转当前单词
35            self._reverse(chars, start, end - 1)
36            start = end + 1
37        
38        return ''.join(chars)
39    
40    def _reverse(self, chars: list, left: int, right: int) -> None:
41        # 双指针翻转字符数组的[left, right]区间
42        while left < right:
43            chars[left], chars[right] = chars[right], chars[left]
44            left += 1
45            right -= 1

📊 复杂度分析

指标 值 说明
时间复杂度 O(n) 遍历字符串3次（清理空格、整体翻转、单词翻转）
空间复杂度 O(n) 需要字符数组存储结果（Python字符串不可变）

优化思考：

  • 在C++/Java中可以实现 O(1) 额外空间，因为可以直接修改输入字符数组
  • Python由于字符串不可变，必须创建新列表，所以空间复杂度为 O(n)

🎯 关键要点总结

  • 双指针清理多余空格时，使用快慢指针技巧，将有效字符前移
  • 两次翻转先整体翻转，再逐个单词翻转，是字符串单词反转的经典技巧
  • 边界处理注意处理首尾空格、连续多个空格的情况
  • 单词识别通过空格位置识别单词边界，翻转指定区间
