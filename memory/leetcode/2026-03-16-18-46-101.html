🌳 LeetCode #101 - 对称二叉树

难度：Easy | 方法：递归 / 迭代 | 核心：镜像比较

📋 题目描述

给你一个二叉树的根节点 root，检查它是否轴对称。

对称的定义：一棵二叉树如果满足以下条件，则称其是对称的：

  • 根节点的左子树与右子树镜像对称
  • 即：左子树的左节点 = 右子树的右节点，左子树的右节点 = 右子树的左节点
示例：对称二叉树
1
2 ─ 2
3 4 4 3

🎬 算法执行过程（点击步骤或代码行查看对应关系）

Python 实现


1# Definition for a binary tree node.
2class TreeNode:
3    def __init__(self, val=0, left=None, right=None):
4        self.val = val
5        self.left = left
6        self.right = right
7
8class Solution:
9    def isSymmetric(self, root: TreeNode) -> bool:
10        if not root:
11            return True
12        return self.check(root.left, root.right)
13    
14    def check(self, left: TreeNode, right: TreeNode) -> bool:
15        # 两个都为空，对称
16        if not left and not right:
17            return True
18        # 只有一个为空，不对称
19        if not left or not right:
20            return False
21        # 值不同，不对称
22        if left.val != right.val:
23            return False
24        # 递归检查：左的左 vs 右的右，左的右 vs 右的左
25        return self.check(left.left, right.right) and \
26               self.check(left.right, right.left)

空间复杂度 O(h) 递归栈深度，h为树的高度，最坏O(n)，平衡树O(log n)

💡 解题思路

核心思想：镜像比较

判断二叉树是否对称，不是简单比较左子树和右子树，而是：

  • 比较左子树的左节点 与 右子树的右节点
  • 比较左子树的右节点 与 右子树的左节点

这是典型的递归分治思想，同时也可以用迭代（队列/栈）实现。
