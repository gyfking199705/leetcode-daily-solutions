LeetCode #739 - 每日温度

Medium 单调栈 数组

📋 题目描述

给定一个数组 temperatures，表示每天的温度，返回一个数组 answer，其中 answer[i] 是指对于第 i 天，下一个更高温度出现在几天后。如果气温在这之后都不会升高，在该位置用 0 来代替。

💡 核心思路

利用单调递减栈，栈中存储的是索引而非温度值。当遇到比栈顶温度高的新温度时，就找到了栈顶元素的"下一个更高温度"，计算天数差并出栈。这样保证栈内温度始终递减，每个元素最多入栈出栈一次。

示例：

输入: temperatures = [73, 74, 75, 71, 69, 72, 76, 73]
输出: [1, 1, 4, 2, 1, 1, 0, 0]

解释：第0天73度，第1天74度比它高，所以是1天后；第3天71度，第6天76度比它高，相差3天...

🧠 解题技巧总结

  • 何时用单调栈？ 找"下一个更大/更小元素"、"左侧/右侧第一个满足条件的元素"这类问题
  • 递增还是递减？ 找下一个更大用递减栈（栈顶最大），找下一个更小用递增栈
  • 存值还是存索引？ 需要计算距离时存索引，只需要比较值时可存值
  • 本题的变形： 若问"前一个更高温度"，只需改为从右向左遍历
