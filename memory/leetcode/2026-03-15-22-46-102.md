## 一、题目描述

**链接**: https://leetcode.cn/problems/binary-tree-level-order-traversal/

### 题目

给你二叉树的根节点 `root`，返回其节点值的**层序遍历**。（即逐层地，从左到右访问所有节点）。

### 示例

```
输入：root = [3,9,20,null,null,15,7]
输出：[[3],[9,20],[15,7]]

输入：root = [1]
输出：[[1]]

输入：root = []
输出：[]
```

### 约束条件
- 树中节点数目在范围 `[0, 2000]` 内
- `-1000 <= Node.val <= 1000`

## 二、解题思路

### 核心思想

层序遍历是**广度优先搜索（BFS）**的经典应用。

与**深度优先搜索（DFS）**的前/中/后序遍历不同，层序遍历：
- **BFS**：先遍历完当前层所有节点，再进入下一层（像"剥洋葱"）
- **DFS**：沿着一条路径走到底，再回溯

### 算法框架（BFS）

```
初始化队列，根节点入队

while 队列不为空:
    记录当前队列长度 = 当前层节点数
    创建临时数组存储当前层节点值
    
    for i in range(当前层节点数):
        节点出队，值加入临时数组
        左孩子不为空则入队
        右孩子不为空则入队
    
    临时数组加入结果
```

**关键技巧**：记录当前层的节点数 `levelSize`，确保每次循环只处理当前层节点。

## 三、完整代码实现

### Python 实现（BFS）

```python
from collections import deque

# Definition for a binary tree node.
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def levelOrder(self, root: TreeNode) -> List[List[int]]:
        if not root:
            return []
        
        result = []
        queue = deque([root])
        
        while queue:
            level_size = len(queue)  # 当前层节点数
            level = []
            
            for _ in range(level_size):
                node = queue.popleft()
                level.append(node.val)
                
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)
            
            result.append(level)
        
        return result
```

### Python 实现（DFS）

```python
class Solution:
    def levelOrder(self, root: TreeNode) -> List[List[int]]:
        result = []
        self.dfs(root, 0, result)
        return result
    
    def dfs(self, node: TreeNode, depth: int, result: List[List[int]]):
        if not node:
            return
        
        # 第一次访问该深度，创建新列表
        if len(result) == depth:
            result.append([])
        
        result[depth].append(node.val)
        
        self.dfs(node.left, depth + 1, result)
        self.dfs(node.right, depth + 1, result)
```

### Java 实现（BFS）

```java
class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> result = new ArrayList<>();
        if (root == null) return result;
        
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        
        while (!queue.isEmpty()) {
            int levelSize = queue.size();
            List<Integer> level = new ArrayList<>();
            
            for (int i = 0; i < levelSize; i++) {
                TreeNode node = queue.poll();
                level.add(node.val);
                
                if (node.left != null) queue.offer(node.left);
                if (node.right != null) queue.offer(node.right);
            }
            
            result.add(level);
        }
        
        return result;
    }
}
```

### C++ 实现（BFS）

```cpp
class Solution {
public:
    vector<vector<int>> levelOrder(TreeNode* root) {
        vector<vector<int>> result;
        if (!root) return result;
        
        queue<TreeNode*> q;
        q.push(root);
        
        while (!q.empty()) {
            int levelSize = q.size();
            vector<int> level;
            
            for (int i = 0; i < levelSize; i++) {
                TreeNode* node = q.front();
                q.pop();
                level.push_back(node->val);
                
                if (node->left) q.push(node->left);
                if (node->right) q.push(node->right);
            }
            
            result.push_back(level);
        }
        
        return result;
    }
};
```

## 四、复杂度分析

| 方法 | 时间复杂度 | 空间复杂度 | 说明 |
|------|-----------|-----------|------|
| **BFS** | O(n) | O(n) | 队列最多存储一层节点，最坏情况为满树最后一层 |
| **DFS** | O(n) | O(h) | h为树的高度，递归栈深度 |

**说明**：
- n 为树中节点总数
- 两种方法时间复杂度相同，每个节点恰好访问一次
- BFS空间复杂度在最坏情况下（完全二叉树）也是O(n)

## 五、关键要点与易错点

### ✅ 核心技巧

**记录当前层节点数 `levelSize`** 是区分每层的关键：
- 队列在循环中会不断加入下一层节点
- 若不记录 `levelSize`，直接使用 `queue.size()` 会导致层级混乱

### ⚠️ 常见错误

1. **忘记空树检查**
   ```python
   if not root: return []  # 必须处理
   ```

2. **层级划分错误**
   - 错误：`for i in range(len(queue))`（队列长度动态变化）
   - 正确：`level_size = len(queue); for i in range(level_size)`

3. **子节点入队顺序**
   - 题目要求"从左到右"，必须先左后右

4. **DFS深度计算**
   - 新层判断：`if len(result) == depth`
   - 递归时深度+1

## 七、面试常问问题

**Q1: BFS和DFS都可以解决这道题，如何选择？**
> - BFS更符合层序遍历的直观理解，是标准解法
> - DFS代码更简洁，适合递归思维者
> - 面试中推荐写BFS版本

**Q2: 如果要求"自底向上"层序遍历怎么做？**> 在BFS完成后，将结果列表反转即可：`result.reverse()`

**Q3: 二叉树的层序遍历在实际应用中有什么用途？**> - 计算树的高度、宽度
> - 寻找最短路径（无权图）> - 序列化/反序列化二叉树
> - 打印树结构

**Q4: 空间复杂度能否优化？**> - 对于BFS，必须用队列存储一层节点，无法优化
> - 对于DFS，递归栈深度最坏O(n)，平衡树时为O(log n)

## 八、总结

层序遍历是二叉树的基础操作，掌握以下三点即可轻松应对：

1. **队列存节点** —— BFS的核心数据结构
2. **levelSize分层次** —— 区分每层的关键技巧  
3. **左孩子先入队** —— 保证"从左到右"的顺序

**一句话口诀**：
> 根入队，记层数，出队取值子入队，层层推进直到空。
