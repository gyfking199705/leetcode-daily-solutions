## 题目描述

给你一个整数 `n`，对于 `0 <= i <= n` 中的每个 `i`，计算其二进制表示中 `1` 的个数，返回一个长度为 `n + 1` 的数组 `ans` 作为答案。

### 示例

**示例 1:**
```
输入: n = 2
输出: [0,1,1]
解释:
0 --> 0
1 --> 1
2 --> 10
```

**示例 2:**
```
输入: n = 5
输出: [0,1,1,2,1,2]
解释:
0 --> 0
1 --> 1
2 --> 10
3 --> 11
4 --> 100
5 --> 101
```

**进阶要求:**
- 时间复杂度：O(n)
- 空间复杂度：O(n)
- 不使用内置函数

---

## 解题思路

### 核心规律发现

这道题的关键在于发现**奇偶数的规律**：

| 数字 | 二进制 | 1的个数 | 规律 |
|-----|-------|--------|-----|
| 0 | 0 | 0 | 基准 |
| 1 | 1 | 1 | 奇数 = 前一个偶数 + 1 |
| 2 | 10 | 1 | 偶数 = 除以2后的数 |
| 3 | 11 | 2 | 奇数 = 前一个偶数 + 1 |
| 4 | 100 | 1 | 偶数 = 除以2后的数 |
| 5 | 101 | 2 | 奇数 = 前一个偶数 + 1 |
| 6 | 110 | 2 | 偶数 = 除以2后的数 |
| 7 | 111 | 3 | 奇数 = 前一个偶数 + 1 |
| 8 | 1000 | 1 | 偶数 = 除以2后的数 |

### 规律总结

1. **偶数**（最低位为0）：`dp[i] = dp[i/2]`
   - 偶数右移一位（除以2）只是在末尾去掉一个0
   - 1的个数不变

2. **奇数**（最低位为1）：`dp[i] = dp[i/2] + 1` 或 `dp[i] = dp[i-1] + 1`
   - 奇数 = 前一个偶数 + 1
   - 比对应的偶数多一个最低位的1

3. **统一公式**：`dp[i] = dp[i >> 1] + (i & 1)`
   - `i >> 1` = `i / 2`（右移一位）
   - `i & 1` = `i % 2`（判断奇偶）

## 代码实现

### 方法一：动态规划 + 奇偶判断

```python
class Solution:
    def countBits(self, n: int) -> List[int]:
        dp = [0] * (n + 1)
        
        for i in range(1, n + 1):
            if i % 2 == 0:
                # 偶数：1的个数与 i/2 相同
                dp[i] = dp[i // 2]
            else:
                # 奇数：比前一个偶数多一个1
                dp[i] = dp[i - 1] + 1
        
        return dp
```

### 方法二：动态规划 + 统一公式（推荐）

```python
class Solution:
    def countBits(self, n: int) -> List[int]:
        dp = [0] * (n + 1)
        
        for i in range(1, n + 1):
            # dp[i] = dp[i // 2] + (i % 2)
            dp[i] = dp[i >> 1] + (i & 1)
        
        return dp
```

### 方法三：Brian Kernighan 算法（理解位运算）

```python
class Solution:
    def countBits(self, n: int) -> List[int]:
        def countOnes(x: int) -> int:
            ones = 0
            while x > 0:
                x &= (x - 1)  # 消除最低位的1
                ones += 1
            return ones
        
        return [countOnes(i) for i in range(n + 1)]
```

### 方法四：动态规划 + 最低位1消除

```python
class Solution:
    def countBits(self, n: int) -> List[int]:
        dp = [0] * (n + 1)
        
        for i in range(1, n + 1):
            # i & (i-1) 消除i的最低位1
            # dp[i] = dp[消除最低位1后的数] + 1
            dp[i] = dp[i & (i - 1)] + 1
        
        return dp
```

### C++ 实现（位运算优化版）

```cpp
class Solution {
public:
    vector<int> countBits(int n) {
        vector<int> dp(n + 1, 0);
        
        for (int i = 1; i <= n; i++) {
            // i >> 1 等价于 i / 2
            // i & 1 等价于 i % 2
            dp[i] = dp[i >> 1] + (i & 1);
        }
        
        return dp;
    }
};
```

### Java 实现

```java
class Solution {
    public int[] countBits(int n) {
```

## 复杂度分析

| 方法 | 时间复杂度 | 空间复杂度 |
|-----|-----------|-----------|
| 暴力法 | O(n × log n) | O(n) |
| DP + 奇偶 | O(n) | O(n) |
| DP + 位运算 | O(n) | O(n) |

**说明：**
- 时间复杂度 O(n)：只需遍历一次 1 到 n
- 空间复杂度 O(n)：返回结果数组的大小
- 如果不算返回结果的空间，额外空间是 O(1)

---

## 关键要点

1. **找规律**：通过观察奇偶数的二进制特征发现递推关系
2. **状态转移**：`dp[i] = dp[i/2] + (i%2)`
3. **位运算优化**：`i >> 1` 和 `i & 1` 比除法和取模更快
4. **初始化**：`dp[0] = 0`（0的二进制没有1）
