LeetCode #98 - 验证二叉搜索树

📋 题目描述

给你一个二叉树的根节点 root，判断其是否是一个有效的二叉搜索树（BST）。

有效 BST 的定义：

  • 节点的左子树只包含 小于 当前节点的数
  • 节点的右子树只包含 大于 当前节点的数
  • 所有左子树和右子树自身必须也是二叉搜索树

关键陷阱：只比较当前节点与直接左右子节点是不够的，需要考虑整棵子树的取值范围。

❌ 常见错误：只判断 node.val > node.left.val 且 node.val < node.right.val 是不够的！
例如：右子树的左孩子必须大于根节点，而不仅仅是大于其父节点。

示例树：[5,1,4,null,null,3,6] — 不是BST（4的左孩子3小于根节点5）

Python 实现（递归法 + 边界传递）

1class Solution:


2    def isValidBST(self, root: TreeNode) -> bool:


3        # 递归辅助函数，维护当前节点的有效范围


4        def validate(node, low, high):


5            # 空节点是合法的BST


6            if not node:


7                return True


8            


9            # 当前节点值必须在 (low, high) 开区间内


10            if node.val <= low or node.val >= high:


11                return False


12            


13            # 左子树: 所有值必须 < 当前节点值


14            # 右子树: 所有值必须 > 当前节点值


15            return (validate(node.left, low, node.val) and


16                    validate(node.right, node.val, high))


17        


18        # 初始范围: 负无穷到正无穷


19        return validate(root, float('-inf'), float('inf'))

🔍 中序遍历解法（替代方案）

核心洞察：BST的中序遍历结果一定是递增的！利用这一性质，我们可以在遍历过程中检查是否严格递增。 
1class Solution:


2    def isValidBST(self, root: TreeNode) -> bool:


3        self.prev = float('-inf')


4        


5        def inorder(node):


6            if not node:


7                return True


8            


9            # 遍历左子树


10            if not inorder(node.left):


11                return False


12            


13            # 检查当前节点（必须严格大于前一个）


14            if node.val <= self.prev:


15                return False


16            self.prev = node.val


17            


18            # 遍历右子树


19            return inorder(node.right)


20        


21        return inorder(root)

中序遍历法 O(n) O(h) 同样递归栈空间，可改用迭代优化

💡 关键要点总结

  • 不能只比较直接子节点：必须维护上下界范围，确保整个子树都满足条件
  • 递归边界传递：左子树的上界是当前节点值，右子树的下界是当前节点值
  • 注意等于的情况：BST要求严格小于/大于，不是小于等于/大于等于
  • 中序遍历特性：BST的中序遍历结果严格递增，可利用这一性质验证
  • 处理极端值：使用 float('-inf') 和 float('inf') 作为初始边界
