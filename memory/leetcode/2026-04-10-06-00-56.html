LeetCode 56. 合并区间

Medium 🕒 2026-04-10 06:00 (Asia/Shanghai) 📌 排序 + 贪心算法

📝 题目描述

以数组 intervals 表示若干个区间的集合，其中单个区间为 intervals[i] = [starti, endi] 。

请你合并所有重叠的区间，并返回一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间。

💡 示例

输入：intervals = [[1,3],[2,6],[8,10],[15,18]]

输出：[[1,6],[8,10],[15,18]]

解释：区间 [1,3] 和 [2,6] 重叠, 将它们合并为 [1,6].

🔍 解题思路

🎯 核心思想

排序是这类区间问题的通用解法。先按起点排序，然后遍历，维护一个当前区间，如果下一个区间与当前区间重叠就合并，不重叠就将当前区间加入结果。

为什么先排序？

  • 排序后，所有可能重叠的区间都会相邻
  • 只需检查当前区间与下一个区间是否重叠（O(1)判断）
  • 无需考虑交叉、跳跃等复杂情况

🎨 重叠判断条件

区间A [a1, a2] 与 区间B [b1, b2] 重叠 的条件是：

a1 <= b2 && b1 <= a2
// 排序后简化为：b1 <= a2 (下一个区间的起点 <= 当前区间的终点)

🎬 交互式演示

观察区间合并的过程（示例：[1,3],[2,6],[8,10],[15,18]）

Step 0: 初始化
原始区间：[[1,3], [2,6], [8,10], [15,18]]
按起点排序后：[[1,3], [2,6], [8,10], [15,18]]
Step 1: 处理 [1,3]
当前区间 current = [1,3]
下一个区间 [2,6] 的起点 2 <= 当前终点 3，重叠！
合并后：current = [1, max(3,6)] = [1,6]
Step 2: 处理 [2,6] 后
当前区间 current = [1,6]
下一个区间 [8,10] 的起点 8 > 当前终点 6，不重叠
将 [1,6] 加入结果，current 更新为 [8,10]
Step 3: 处理 [8,10]
当前区间 current = [8,10]
下一个区间 [15,18] 的起点 15 > 当前终点 10，不重叠
将 [8,10] 加入结果，current 更新为 [15,18]
Step 4: 完成
遍历结束，将最后的 [15,18] 加入结果
最终结果：[[1,6], [8,10], [15,18]]

💻 代码实现

class Solution: def merge(self, intervals: List[List[int]]) -> List[List[int]]: # 1. 按区间起点升序排序 intervals.sort(key=lambda x: x[0]) merged = [] for interval in intervals: # 2. 如果当前区间与merged中最后一个不重叠，直接添加 if not merged or merged[-1][1] < interval[0]: merged.append(interval) else: # 3. 重叠，合并区间（更新终点） merged[-1][1] = max(merged[-1][1], interval[1]) return merged
class Solution { public int[][] merge(int[][] intervals) { // 1. 按起点排序 Arrays.sort(intervals, (a, b) -> Integer.compare(a[0], b[0])); List<int[]> merged = new ArrayList<>(); for (int[] interval : intervals) { // 2. 不重叠则添加 if (merged.isEmpty() || merged.get(merged.size() - 1)[1] < interval[0]) { merged.add(interval); } else { // 3. 重叠则合并 int[] last = merged.get(merged.size() - 1); last[1] = Math.max(last[1], interval[1]); } } return merged.toArray(new int[merged.size()][]); } }
class Solution { public: vectorint>> merge(vectorint>>& intervals) { if (intervals.empty()) return {}; // 1. 按起点排序 sort(intervals.begin(), intervals.end()); vectorint>> merged; for (auto& interval : intervals) { // 2. 不重叠则添加 if (merged.empty() || merged.back()[1] < interval[0]) { merged.push_back(interval); } else { // 3. 重叠则合并 merged.back()[1] = max(merged.back()[1], interval[1]); } } return merged; } };

🎓 解题模板总结

区间问题通用解题框架：

  1. 排序 - 按起点（或终点）排序
  2. 遍历 - 维护当前区间
  3. 判断 - 重叠条件（通常是比较终点与下一个起点）
  4. 合并/更新 - 根据题目要求处理

⚠️ 常见坑点

  • 忘记排序 - 这是最关键的一步，未排序的区间无法直接合并
  • 重叠条件错误 - 注意是 <= 还是 <，题目要求闭区间时边界相等也算重叠
  • 合并时只更新终点 - 起点不需要更新（因为已排序，当前区间起点一定更大或相等）
  • 最后一个区间遗漏 - 遍历结束后记得把最后一个区间加入结果
