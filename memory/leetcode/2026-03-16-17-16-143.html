🔀 LeetCode #143 - 重排链表

Medium 链表 双指针 链表反转

📋 题目描述

给定一个单链表 L 的头节点 head ，单链表 L 表示为：

L0 → L1 → L2 → ... → Ln-2 → Ln-1

将其重新排列后变为：

L0 → Ln-1 → L1 → Ln-2 → L2 → Ln-3 → ...

要求：不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

示例 1：

输入: 1 → 2 → 3 → 4

输出: 1 → 4 → 2 → 3

示例 2：

输入: 1 → 2 → 3 → 4 → 5

输出: 1 → 5 → 2 → 4 → 3

💡 算法思路解析

这道题的核心是将链表从中间分成两半，然后把后半部分反转，最后交错合并两个链表。

🎯 三步走策略

  • Step 1 - 找中点：使用快慢指针，快指针走2步，慢指针走1步，当快指针到达末尾时，慢指针正好在中间
  • Step 2 - 反转后半段：将链表后半部分原地反转（经典三指针法）
  • Step 3 - 交错合并：类似拉链一样，交替从两个链表取节点连接

时间复杂度：O(n) — 每个节点最多被访问3次（找中点、反转、合并各1次）

空间复杂度：O(1) — 只使用了几个指针变量，原地操作

📊 复杂度分析

算法 时间复杂度 空间复杂度 说明
三步法（最优） O(n) O(1) 原地操作，不借助额外数据结构
线性表辅助 O(n) O(n) 用数组存储节点再重连，不符合进阶要求
递归（栈） O(n) O(n) 利用递归隐式栈找到末尾节点
