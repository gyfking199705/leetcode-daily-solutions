LeetCode #10 - 正则表达式匹配

📋 题目描述

给你一个字符串 s 和一个字符规律 p，请你来实现一个支持 '.' 和 '*' 的正则表达式匹配。

  • '.' 匹配任意单个字符
  • '*' 匹配零个或多个前面的那一个元素

所谓匹配，是要涵盖 整个 字符串 s 的，而不是部分字符串。

示例 1：
输入：s = "aa", p = "a"
输出：false
解释："a" 无法匹配 "aa" 整个字符串。
示例 2：
输入：s = "aa", p = "a*"
输出：true
解释：因为 '*' 代表可以匹配零个或多个前面的那一个元素，即 'a'。因此，"aa" 可被 'a' 重复两次匹配。
示例 3：
输入：s = "ab", p = ".*"
输出：true
解释：".*" 表示可匹配零个或多个任意字符。

🧠 解题思路

这是一道经典的 二维动态规划 题目。定义 dp[i][j] 表示 s[0:i]（前i个字符）能否被 p[0:j]（前j个字符）匹配。

状态转移方程

对于 dp[i][j]，考虑 p[j-1] 的情况：

  • 如果 p[j-1] 是普通字符或 '.'：
    dp[i][j] = dp[i-1][j-1] && (s[i-1] == p[j-1] || p[j-1] == '.')
  • 如果 p[j-1] 是 '*'：
    需要考虑 '*' 匹配 0次 或 多次：
    • 匹配0次：dp[i][j] = dp[i][j-2]（跳过 'x*'）
    • 匹配多次：dp[i][j] = dp[i-1][j] && (s[i-1] == p[j-2] || p[j-2] == '.')

💡 关键点总结

  • DP定义：dp[i][j] 表示 s前i个 与 p前j个 是否匹配
  • 初始化：dp[0][0] = true（空串匹配空串）；注意处理 p中以*结尾 的模式可以匹配空串
  • 星号处理：'*' 可以匹配0次或多次，这是本题的核心难点
  • 下标映射：dp[i][j] 对应 s[i-1] 和 p[j-1]，注意索引偏移
