LeetCode #70 - 爬楼梯

📋 题目描述

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

示例 1：
输入：n = 2
输出：2
解释：有两种方法可以爬到楼顶。
1. 1 阶 + 1 阶
2. 2 阶
示例 2：
输入：n = 3
输出：3
解释：有三种方法可以爬到楼顶。
1. 1 阶 + 1 阶 + 1 阶
2. 1 阶 + 2 阶
3. 2 阶 + 1 阶

约束条件：1 ≤ n ≤ 45

💡 解题思路

这是一道经典的动态规划入门题。核心在于发现状态转移方程：

状态转移方程： dp[i] = dp[i-1] + dp[i-2]
到达第 i 阶的方法数 = 从 i-1 阶迈1步上来 + 从 i-2 阶迈2步上来

这与斐波那契数列相同！因此我们可以用空间优化的方式，只保留前两个状态。

Python 实现（空间优化版）


1class Solution:
2    def climbStairs(self, n: int) -> int:
3        # 边界条件处理
4        if n <= 2:
5            return n
6        
7        # 只保留前两个状态：prev2=dp[i-2], prev1=dp[i-1]
8        prev2, prev1 = 1, 2  # dp[1]=1, dp[2]=2
9        
10        # 从第3阶开始迭代计算
11        for i in range(3, n + 1):
12            # 当前阶数的方法数 = 前两阶方法数之和
13            curr = prev1 + prev2
14            prev2 = prev1  # 更新前前阶
15            prev1 = curr   # 更新前一阶
16        
17        return prev1

Python 实现（标准DP版）


20class Solution:
21    def climbStairs(self, n: int) -> int:
22        # dp[i] 表示爬到第i阶的方法数
23        dp = [0] * (n + 1)
24        dp[0], dp[1] = 1, 1  # 初始条件
25        
26        for i in range(2, n + 1):
27            dp[i] = dp[i - 1] + dp[i - 2]
28        
29        return dp[n]

📊 复杂度分析

解法 时间复杂度 空间复杂度 说明
标准DP O(n) O(n) 需要dp数组存储所有状态
空间优化DP O(n) O(1) 只保留前两个状态，推荐
矩阵快速幂 O(log n) O(1) 利用斐波那契性质，面试加分项

🔍 常见变形与扩展

  • 每次可爬1/2/3阶： dp[i] = dp[i-1] + dp[i-2] + dp[i-3]
  • 带花费的爬楼梯： 每阶有cost，求最小花费
  • 不能连续爬2阶： 增加状态维度表示上一步爬了几阶
