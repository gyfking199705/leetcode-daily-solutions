LeetCode #108 - 将有序数组转换为二叉搜索树

Convert Sorted Array to Binary Search Tree · 分治 · 二叉树

📋 题目描述

给你一个整数数组 nums ，其中元素已经按 升序 排列，请你将其转换为一棵 高度平衡 的二叉搜索树。

高度平衡 二叉树是一棵满足「每个节点的左右两个子树的高度差的绝对值不超过 1」的二叉树。

二叉搜索树（BST）性质：左子树所有节点值 < 根节点值 < 右子树所有节点值

💡 核心思路

有序数组转平衡 BST 的关键在于：每次选择数组的中间元素作为根节点，这样左右子树的节点数量差不超过 1，自然满足高度平衡。

  • 数组中间元素 → 当前子树的根节点
  • 中间元素左边的子数组 → 构建左子树
  • 中间元素右边的子数组 → 构建右子树
  • 递归处理，直到子数组为空
