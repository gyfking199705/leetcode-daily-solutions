LeetCode #79 - 单词搜索

Medium DFS Backtracking

📋 题目描述

给定一个 m x n 二维字符网格 board 和一个字符串单词 word 。如果 word 存在于网格中，返回 true；否则，返回 false。

单词必须按照字母顺序，通过相邻的单元格内的字母构成，其中「相邻」单元格是那些水平相邻或垂直相邻的单元格。同一个单元格内的字母不允许被重复使用。

核心考点：二维回溯 + DFS + 剪枝优化。这是回溯算法在二维网格上的经典应用，也是面试高频题。

示例：

board = [
  ['A','B','C','E'],
  ['S','F','C','S'],
  ['A','D','E','E']
]
word = "ABCCED"

输出: true
解释: 单词 "ABCCED" 可以从 board[0][0] -> [0][1] -> [0][2] -> [1][2] -> [2][2] -> [2][1] 找到

⏱️ 复杂度分析

复杂度类型 值 说明
时间复杂度 O(m × n × 4^L) L为单词长度，每个格子最多4个方向探索
空间复杂度 O(L) 递归栈深度，最坏为单词长度
剪枝优化技巧：如果单词首字母在board中出现次数很少，可以从这些位置开始；搜索前先判断board中各字符数量是否足够组成word。
