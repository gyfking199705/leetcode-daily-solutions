LeetCode #72 - 编辑距离

Medium 动态规划经典问题 · 字符串DP

📋 题目描述

给你两个单词 word1 和 word2，请返回将 word1 转换成 word2 所使用的最少操作数。

你可以对一个单词进行如下三种操作：

  • 插入一个字符
  • 删除一个字符
  • 替换一个字符

示例：word1 = "horse", word2 = "ros" → 输出：3
horse → rorse (将 'h' 替换为 'r')
rorse → rose (删除 'r')
rose → ros (删除 'e')

💡 核心思路

定义 dp[i][j] 表示将 word1[0..i-1] 转换为 word2[0..j-1] 的最小操作数。

状态定义 状态转移 边界条件

状态转移方程：

if word1[i-1] == word2[j-1]:
    dp[i][j] = dp[i-1][j-1]  # 字符相同，无需操作
else:
    dp[i][j] = 1 + min(dp[i-1][j],      # 删除
                       dp[i][j-1],      # 插入
                       dp[i-1][j-1])    # 替换

⏱️ 复杂度分析

复杂度类型 值 说明
时间复杂度 O(m × n) 需要遍历 m×n 的DP表格
空间复杂度 O(m × n) 二维DP数组，可优化至 O(min(m,n))

空间优化：由于每次只用到上一行的数据，可以将二维数组压缩为一维滚动数组，空间复杂度降至 O(n)。
