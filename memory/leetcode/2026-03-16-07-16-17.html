📋 题目描述

给定一个仅包含数字 2-9 的字符串，返回所有它能表示的字母组合。答案可以按 任意顺序 返回。

给出数字到字母的映射如下（与电话按键相同）。注意 1 不对应任何字母。

2 ABC
4 GHI
5 JKL
6 MNO
7 PQRS
8 TUV
9 WXYZ
3 DEF
示例 1：
输入：digits = "23"
输出：["ad","ae","af","bd","be","bf","cd","ce","cf"]
示例 2：
输入：digits = ""
输出：[]
示例 3：
输入：digits = "2"
输出：["a","b","c"]

💡 关键约束

• 0 <= digits.length <= 4
• digits[i] 是范围 ['2', '9'] 的一个数字

💡 解题思路

这道题的本质是枚举所有可能的组合。假设输入是 "23"：

  • 数字 2 对应 "abc"（3个选择）
  • 数字 3 对应 "def"（3个选择）
  • 总共有 3 × 3 = 9 种组合

这种"需要尝试所有可能路径"的问题，正是回溯算法的典型应用场景。

36class Solution:
    def letterCombinations(self, digits: str) -> List[str]:
        if not digits:
            return []
        
        # 数字到字母的映射表
        phone_map = {
            '2': 'abc', '3': 'def',
            '4': 'ghi', '5': 'jkl',
            '6': 'mno', '7': 'pqrs',
            '8': 'tuv', '9': 'wxyz'
        }
        
        result = []
        
        def backtrack(index: int, current: str):
            # 结束条件：已经处理完所有数字
            if index == len(digits):
                result.append(current)
                return
            
            # 获取当前数字对应的所有字母
            digit = digits[index]
            for letter in phone_map[digit]:
                # 做选择
                current += letter
                
                # 进入下一层决策树
                backtrack(index + 1, current)
                
                # 撤销选择（回溯）
                current = current[:-1]
        
        backtrack(0, '')
        return result

22class Solution:
    def letterCombinations(self, digits: str) -> List[str]:
        if not digits:
            return []
        
        phone_map = ['', '', 'abc', 'def', 'ghi', 
                     'jkl', 'mno', 'pqrs', 'tuv', 'wxyz']
        
        result = []
        path = []  # 使用列表存储字符，避免字符串拼接开销
        
        def backtrack(index: int):
            if index == len(digits):
                result.append(''.join(path))
                return
            
            for letter in phone_map[int(digits[index])]:
                path.append(letter)      # 做选择
                backtrack(index + 1)    # 递归
                path.pop()               # 撤销选择
        
        backtrack(0)
        return result

📊 复杂度分析

指标 复杂度 说明
时间 O(3^N × 4^M) N 是映射为3个字母的数字个数，M 是映射为4个字母的数字个数（7和9）。最坏情况下所有数字都是7或9。
空间 O(N) 递归调用栈的深度为 N（digits 长度），不包括存储结果的数组。

📈 为什么是 3^N × 4^M？

数字 2-6、8 对应 3 个字母，数字 7、9 对应 4 个字母。
输入 "23" → 3 × 3 = 9 种组合
输入 "79" → 4 × 4 = 16 种组合
输入 "29" → 3 × 4 = 12 种组合
