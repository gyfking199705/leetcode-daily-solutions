LeetCode #162 - 寻找峰值 Medium

二分查找的经典变体：利用题目条件缩小搜索范围

📋 题目描述

峰值元素是指其值严格大于左右相邻值的元素。

给你一个整数数组 nums，找到峰值元素并返回其索引。数组可能包含多个峰值，在这种情况下，返回 任何一个峰值 所在位置即可。

你可以假设 nums[-1] = nums[n] = -∞ 。也就是说，数组两端的元素都可以视为峰值（只要它比相邻元素大）。

你必须实现时间复杂度为 O(log n) 的算法来解决此问题。

示例：
输入：nums = [1,2,3,1]
输出：2
解释：3 是峰值元素，你的函数应该返回其索引 2。

💡 核心思想

这道题的关键在于理解「只要找到一个上升的方向，就一定能找到一个峰值」。

如果 nums[mid] < nums[mid+1]，说明右边存在上升，右边一定有峰值。
如果 nums[mid] > nums[mid+1]，说明左边更大或相等，左边一定有峰值。

left = 0
→
right = n-1
→
取 mid
→
比较 mid 与 mid+1
→
缩小区间
→
返回 left

📊 复杂度分析

复杂度类型 值 说明
时间复杂度 O(log n) 二分查找，每次将搜索区间减半
空间复杂度 O(1) 只使用了常数额外空间
比较次数 ⌈log₂n⌉ 最多进行 log₂n 次比较

🎯 为什么这种方法有效？

关键证明：

假设 nums[mid] < nums[mid+1]，我们选择向右边搜索。

1. 如果右边一直上升直到数组末尾，由于 nums[n] = -∞，最后一个元素就是峰值。

2. 如果右边某处开始下降，那个转折点就是峰值。

因此，只要存在上升，就必然存在峰值。
