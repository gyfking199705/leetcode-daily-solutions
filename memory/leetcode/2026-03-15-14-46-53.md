# 🎯 LeetCode 精讲 #53 - 最大子数组和

## 📋 题目描述

**来源**: LeetCode 第53题  
**难度**: Medium

### 题目原文
> 给你一个整数数组 `nums`，请你找出一个具有最大和的**连续子数组**（子数组最少包含一个元素），返回其最大和。
> 
> **子数组**是数组中的一个连续部分。

### 示例

**示例 1:**
```
输入: nums = [-2,1,-3,4,-1,2,1,-5,4]
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。
```

**示例 2:**
```
输入: nums = [1]
输出: 1
```

**示例 3:**
```
输入: nums = [5,4,-1,7,8]
输出: 23
解释: 整个数组的和最大，为 23。
```

### 约束条件
- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`

## 💡 解题思路

这道题是**动态规划**的入门经典，也是理解「状态转移」的绝佳例题。同时，它还有分治解法，值得一起掌握。

### 方法一：动态规划（Kadane算法）⭐推荐

#### 核心思想

> **「当前元素要不要加入前面的子数组？」**

对于每个位置 `i`，我们面临一个选择：
- **加入前面的子数组**：如果前面子数组的和为正，加入能让当前和更大
- **从自己开始新子数组**：如果前面子数组的和为负，不如重新开始

#### 状态定义

`dp[i]` 表示「以第 `i` 个元素结尾的连续子数组的最大和」

#### 状态转移方程

```
dp[i] = max(nums[i], dp[i-1] + nums[i])
      ↑            ↑
   自己开始    加入前面
```

#### 图解过程

```
数组:    [-2,  1, -3,  4, -1,  2,  1, -5,  4]
         
dp[0]:   -2   (只有自己，-2)
dp[1]:    1   (max(1, -2+1) = 1，前面是负的，自己开始)
dp[2]:   -2   (max(-3, 1-3) = -2，加入前面)
dp[3]:    4   (max(4, -2+4) = 4，前面是负的，自己开始)
dp[4]:    3   (max(-1, 4-1) = 3，加入前面)
dp[5]:    5   (max(2, 3+2) = 5，加入前面)
dp[6]:    6   (max(1, 5+1) = 6，加入前面) ← 最大值！
dp[7]:    1   (max(-5, 6-5) = 1，加入前面)
dp[8]:    5   (max(4, 1+4) = 5，加入前面)

最大子数组: [4, -1, 2, 1]，和为 6
```

#### 空间优化

注意到 `dp[i]` 只依赖于 `dp[i-1]`，可以用一个变量代替数组：

```
current_max = max(nums[i], current_max + nums[i])
global_max = max(global_max, current_max)
```

#### 代码实现

**Python:**
```python
class Solution:
    def maxSubArray(self, nums: list[int]) -> int:
        # 初始化：第一个元素
        current_max = global_max = nums[0]
        
        # 从第二个元素开始遍历
        for i in range(1, len(nums)):
            # 决策：自己开始 还是 加入前面
            current_max = max(nums[i], current_max + nums[i])
            # 更新全局最大值
            global_max = max(global_max, current_max)
        
        return global_max
```

### 方法二：分治法

#### 核心思想

将数组分成左右两部分，最大子数组可能出现在：
1. **左半部分**
2. **右半部分**
3. **跨越中点**（包含左半部分后缀 + 右半部分前缀）

#### 图解

```
数组: [left_part] | [right_part]
            ↑
          中点

最大子数组有三种情况:
┌─────────────────────────────────────┐
│  [  左半部分的最大子数组  ]           │  ← 递归求解
├─────────────────────────────────────┤
│              [  右半部分的最大子数组 ] │  ← 递归求解
├─────────────────────────────────────┤
│  [左后缀...] | [...右前缀]            │  ← 从中点向两边扩展求最大
└─────────────────────────────────────┘
```

#### 代码实现

```python
class Solution:
    def maxSubArray(self, nums: list[int]) -> int:
        def divide_conquer(left, right):
            if left == right:
                return nums[left]
            
            mid = (left + right) // 2
            
            # 左半部分最大和
            left_max = divide_conquer(left, mid)
            # 右半部分最大和
            right_max = divide_conquer(mid + 1, right)
            
            # 跨越中点的最大和
            # 从中点向左扩展
            cross_left_max = float('-inf')
            cross_sum = 0
            for i in range(mid, left - 1, -1):
                cross_sum += nums[i]
                cross_left_max = max(cross_left_max, cross_sum)
            
            # 从中点向右扩展
            cross_right_max = float('-inf')
            cross_sum = 0
            for i in range(mid + 1, right + 1):
                cross_sum += nums[i]
                cross_right_max = max(cross_right_max, cross_sum)
            
            cross_max = cross_left_max + cross_right_max
            
            return max(left_max, right_max, cross_max)
        
        return divide_conquer(0, len(nums) - 1)
```

## ⏱️ 复杂度分析

| 方法 | 时间复杂度 | 空间复杂度 | 说明 |
|:---|:---:|:---:|:---|
| 动态规划（Kadane） | O(n) | O(1) | ⭐最优解法，推荐掌握 |
| 分治法 | O(n log n) | O(log n) | 递归栈空间，思想重要 |

### 复杂度详解

**动态规划：**
- 时间 O(n)：只需遍历数组一次
- 空间 O(1)：只用两个变量，常数空间

**分治法：**
- 时间 O(n log n)：递归树高度 log n，每层处理需要 O(n)
- 空间 O(log n)：递归调用栈的深度

## 🎯 关键要点总结

### 1. 动态规划核心

```
状态定义：dp[i] = 以i结尾的最大子数组和
转移方程：dp[i] = max(nums[i], dp[i-1] + nums[i])

关键理解：
- 如果 dp[i-1] > 0，加入前面更优
- 如果 dp[i-1] < 0，自己重新开始更优
```

### 2. 边界情况

```
[-1]       → -1      (单元素负数)
[-2, -1]   → -1      (全负数，取最大那个)
[1, 2, 3]  → 6       (全正数，取全部)
```

### 3. 扩展问题：如何返回子数组本身？

```python
def maxSubArrayWithIndices(nums):
    current_max = global_max = nums[0]
    current_start = 0  # 当前子数组起始位置
    best_start = best_end = 0  # 最优子数组起止位置
    
    for i in range(1, len(nums)):
        if current_max + nums[i] < nums[i]:
            # 重新开始
            current_max = nums[i]
            current_start = i
        else:
            # 加入前面
            current_max += nums[i]
        
        # 更新全局最优
        if current_max > global_max:
            global_max = current_max
            best_start = current_start
            best_end = i
    
    return global_max, best_start, best_end, nums[best_start:best_end+1]
```

## 🧠 面试技巧

**高频追问：**

1. **Q: 如果数组全是负数怎么办？**
   A: 算法依然适用，返回最大的那个负数。

2. **Q: 如何找出具体的子数组（不只是和）？**
   A: 额外记录起始位置，更新最大值时同步更新起止索引。

3. **Q: 如果改成乘积最大怎么做？**
   A: 需要同时维护最大值和最小值，因为负数×负数=正数。

4. **Q: 时间复杂度还能优化吗？**
   A: 已经是最优 O(n)，必须遍历每个元素至少一次。
