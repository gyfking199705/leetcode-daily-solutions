🔄 LeetCode #7 - 整数反转

Reverse Integer | 数学方法 | 溢出检测

📋 题目描述

给你一个 32 位的有符号整数 x，返回将 x 中的数字部分反转后的结果。

⚠️ 关键约束：如果反转后整数超过 32 位有符号整数的范围 [−2³¹, 2³¹ − 1]，则返回 0。
示例 1：

输入：x = 123

输出：321

示例 2：

输入：x = -123

输出：-321

示例 3：

输入：x = 120

输出：21

示例 4：

输入：x = 1534236469

输出：0 (反转后溢出)

🎬 算法执行过程（点击步骤或代码行查看对应关系）

输入: x = 123，观察每次循环的变化

Python 实现



1class Solution:


2    def reverse(self, x: int) -> int:


3        INT_MIN, INT_MAX = -2**31, 2**31 - 1  # -2147483648 ~ 2147483647


4        result = 0


5        while x != 0:


6            # Python取模特殊处理：%运算结果符号与除数一致


7            digit = x % 10 if x > 0 else x % -10


8            x = x // 10 if x > 0 else (x - digit) // -10


9            # 溢出检测：在乘10加digit之前判断


10            if result > INT_MAX // 10 or (result == INT_MAX // 10 and digit > 7):


11                return 0


12            if result < INT_MIN // 10 or (result == INT_MIN // 10 and digit < -8):


13                return 0


14            result = result * 10 + digit


15        return result

💡 核心思路解析

🎯 数学方法

通过循环每次取出整数的最后一位，然后将其添加到结果中：

result = result * 10 + digit

🔍 关键问题：溢出检测

32位有符号整数范围：[-2147483648, 2147483647]

在 result * 10 + digit 之前进行溢出检查：

  • 正溢出：如果 result > 214748364，或者 result == 214748364 且 digit > 7
  • 负溢出：如果 result < -214748364，或者 result == -214748364 且 digit < -8

🐍 Python 特殊处理

Python的 % 和 // 运算符结果符号与除数一致（而不是被除数），需要特殊处理负数情况：

  • -123 % 10 在Python中等于 7，而不是 -3
  • 解决方案：负数时取模用 x % -10 得到 -3

空间复杂度 O(1) 只使用常数额外空间
