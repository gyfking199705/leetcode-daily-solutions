LeetCode #2 - 两数相加

Add Two Numbers | 链表 | Medium

📋 题目描述

给你两个 非空 的链表，表示两个非负的整数。它们每位数字都是按照 逆序 的方式存储的，并且每个节点只能存储 一位 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

💡 核心考点：链表遍历 + 进位处理。需要同时处理两个链表的遍历、进位的计算与传递、以及最终链表末尾可能多出的进位。

示例 1：

输入：l1 = [2,4,3], l2 = [5,6,4]

输出：[7,0,8]

解释：342 + 465 = 807

📊 复杂度分析

维度值说明
时间复杂度O(max(m, n))m 和 n 分别是两个链表的长度，遍历较长链表的每一位
空间复杂度O(max(m, n))新链表的长度，可能需要多一位存储进位

🎯 关键思路

  1. 逆序存储的优势：链表头就是个位，正好符合加法从低位到高位的运算顺序，无需反转链表。
  2. 同步遍历：同时遍历两个链表，位数不足时用0补齐。
  3. 进位处理：当前位和 = val1 + val2 + carry，新节点值 = sum % 10，新进位 = sum // 10。
  4. 收尾检查：循环结束后若还有进位，需额外创建一个节点。
