LeetCode #234 - 回文链表

快慢指针找中点 + 链表反转的经典组合题

📋 题目描述

给你一个单链表的头节点 head ，请你判断该链表是否为回文链表。如果是，返回 true；否则，返回 false。

示例 1：
输入：head = [1,2,2,1]
输出：true

示例 2：
输入：head = [1,2]
输出：false

中等

💡 核心思路（O(1)空间解法）：
  1. 找中点：使用快慢指针找到链表的中点（慢指针走一步，快指针走两步）
  2. 反转后半段：将链表的后半部分进行反转
  3. 比较：将前半段和反转后的后半段逐一比较
  4. 恢复（可选）：再次反转后半段，恢复原链表结构

📊 复杂度分析

指标 复杂度 说明
时间复杂度 O(n) 需要遍历链表三次：找中点、反转、比较
空间复杂度 O(1) 只使用了几个指针，没有使用额外数据结构

🎯 关键技巧总结

  1. 快慢指针技巧：快指针走两步、慢指针走一步，当快指针到达末尾时，慢指针正好在中间
  2. 链表反转技巧：使用三个指针（prev, curr, next）迭代完成反转
  3. 奇偶节点处理：对于奇数长度的链表，慢指针会停在中点，需要额外向前一步
  4. 边界情况：空链表和单节点链表天然是回文
