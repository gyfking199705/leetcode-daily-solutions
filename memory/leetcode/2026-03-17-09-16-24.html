🔄 LeetCode #24 - 两两交换链表中的节点

📋 题目描述

给你一个链表，两两交换其中相邻的节点，并返回交换后链表的头节点。

你必须在不修改节点内部的值的情况下完成本题（即，只能进行节点交换）。

示例 1：
输入：head = [1,2,3,4]
输出：[2,1,4,3]
示例 2：
输入：head = [1,2,3]
输出：[2,1,3]
示例 3：
输入：head = [1]
输出：[1]

提示：链表中节点的数目在范围 [0, 100] 内；0 <= Node.val <= 100

Python 实现（迭代法）

📊 复杂度分析

指标值说明
时间复杂度 O(n) 遍历链表一次，n为链表长度
空间复杂度 O(1) 只使用常数个指针变量

💡 核心思路解析

这道题的难点在于指针操作的顺序。交换两个相邻节点涉及4个指针的重新指向：

  1. prev：待交换节点对的前一个节点
  2. first：第一个节点（要放到后面）
  3. second：第二个节点（要放到前面）
  4. next_pair：下一对待交换节点的第一个
原始: prev → first → second → next_pair
⬇️ 交换后
结果: prev → second → first → next_pair

操作顺序至关重要：

  1. 先保存 next_pair（防止断链后找不到）
  2. 将 second 接到 prev 后面
  3. 将 first 接到 second 后面
  4. 将 next_pair 接到 first 后面
  5. 移动 prev 到 first，准备下一对交换

🔄 递归解法（一行代码）

# 递归法：优雅但空间复杂度为O(n)
class Solution:
    def swapPairs(self, head: Optional[ListNode]) -> Optional[ListNode]:
        # 递归终止条件：不足两个节点
        if not head or not head.next:
            return head
        
        # 保存第二个节点（新头节点）
        new_head = head.next
        
        # 递归处理后面的链表
        head.next = self.swapPairs(new_head.next)
        
        # 交换当前这对
        new_head.next = head
        
        return new_head
