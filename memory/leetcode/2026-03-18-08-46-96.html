🌲 LeetCode #96 - 不同的二叉搜索树

Medium 动态规划 卡特兰数 二叉树

📋 题目描述

给你一个整数 n，求恰由 n 个节点组成且节点值从 1 到 n 互不相同的 二叉搜索树 有多少种？返回满足题意的二叉搜索树的种数。

示例 1：
输入：n = 3
输出：5

示例 2：
输入：n = 1
输出：1

二叉搜索树定义：对于任意节点，左子树所有节点的值小于当前节点，右子树所有节点的值大于当前节点。

💡 核心思路

这道题的本质是计算卡特兰数（Catalan Number）。关键在于发现以下规律：

G(n) = G(0)×G(n-1) + G(1)×G(n-2) + ... + G(n-1)×G(0)

其中 G(i) 表示 i 个节点能构成的不同 BST 数量。对于以 i 为根节点的 BST：

  • 左子树有 i-1 个节点（值为 1 到 i-1），方案数为 G(i-1)
  • 右子树有 n-i 个节点（值为 i+1 到 n），方案数为 G(n-i)
  • 以 i 为根的方案数 = G(i-1) × G(n-i)
