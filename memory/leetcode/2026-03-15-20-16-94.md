# LeetCode 94. 二叉树的中序遍历

## 一、题目描述

给定一个二叉树的根节点 `root`，返回它的**中序遍历**结果。

**中序遍历顺序**：左子树 → 根节点 → 右子树

### 示例

```
输入: root = [1,null,2,3]
    1
     \
      2
     /
    3
输出: [1,3,2]
```

### 约束条件
- 树中节点数目在范围 `[0, 100]` 内
- `-100 <= Node.val <= 100`

## 三、解法一：递归法

### 思路
按照中序遍历定义直接实现：先递归左子树，再访问根节点，最后递归右子树。

### 代码

```python
# Definition for a binary tree node.
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def inorderTraversal(self, root: TreeNode) -> List[int]:
        result = []
        self.inorder_helper(root, result)
        return result
    
    def inorder_helper(self, node: TreeNode, result: List[int]):
        if not node:
            return
        # 1. 遍历左子树
        self.inorder_helper(node.left, result)
        # 2. 访问根节点
        result.append(node.val)
        # 3. 遍历右子树
        self.inorder_helper(node.right, result)
```

### 复杂度分析
- **时间复杂度**: O(n) - 每个节点访问一次
- **空间复杂度**: O(h) - h为树的高度，递归栈的深度

## 四、解法二：迭代法（栈模拟）

### 思路
递归本质是使用了系统调用栈，我们可以用显式栈来模拟：
1. 一路向左，将所有左节点入栈
2. 出栈访问节点
3. 转向右子树，重复上述过程

### 代码

```python
class Solution:
    def inorderTraversal(self, root: TreeNode) -> List[int]:
        result = []
        stack = []
        current = root
        
        while current or stack:
            # 一路向左，全部入栈
            while current:
                stack.append(current)
                current = current.left
            
            # 出栈访问
            current = stack.pop()
            result.append(current.val)
            
            # 转向右子树
            current = current.right
        
        return result
```

### 复杂度分析
- **时间复杂度**: O(n)
- **空间复杂度**: O(h) - 栈的最大深度

### 关键点
- 迭代法避免了递归的栈溢出风险
- 适用于树很深的场景

## 五、解法三：Morris遍历（面试加分项）

### 思路
利用树中大量指向null的指针，**临时修改树的结构**来实现O(1)空间遍历：
1. 如果当前节点左子树为空：访问当前节点，转向右子树
2. 如果左子树不为空：找到左子树最右节点（前驱）
   - 若前驱右指针为null：将其指向当前节点（建立线索），当前节点向左走
   - 若前驱右指针指向当前节点：断开线索，访问当前节点，向右走

### 代码

```python
class Solution:
    def inorderTraversal(self, root: TreeNode) -> List[int]:
        result = []
        current = root
        
        while current:
            if not current.left:
                # 左子树为空，直接访问，向右走
                result.append(current.val)
                current = current.right
            else:
                # 找到左子树最右节点（前驱）
                predecessor = current.left
                while predecessor.right and predecessor.right != current:
                    predecessor = predecessor.right
                
                if not predecessor.right:
                    # 第一次访问，建立线索
                    predecessor.right = current
                    current = current.left
                else:
                    # 第二次访问，断开线索，访问当前节点
                    predecessor.right = None
                    result.append(current.val)
                    current = current.right
        
        return result
```

### 复杂度分析
- **时间复杂度**: O(n) - 每个节点被访问常数次
- **空间复杂度**: O(1) - 只使用常数额外空间

### 关键点
- 面试中展示Morris遍历是加分项
- 会临时修改树的结构，但遍历完成后恢复原状
- 适用于内存极度受限的场景

## 七、核心要点总结

### 🔑 必会点
1. **遍历顺序**：左 → 根 → 右，不要与前序（根左右）、后序（左右根）混淆
2. **迭代实现**：用栈模拟递归，记住"左入栈 → 出栈访问 → 转右"的套路
3. **边界处理**：空树、单节点树、只有左/右子树的树

### ⚠️ 常见错误
1. 混淆三种遍历顺序
2. 迭代法中栈操作顺序错误
3. Morris遍历中线索的建立与断开时机错误

### 💡 面试技巧
- 先写递归版本（简洁易懂）
- 主动提及迭代版本（展示基本功）
- 能讲Morris遍历（加分亮点）
