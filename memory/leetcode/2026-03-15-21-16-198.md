## 题目描述

你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，**如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警**。

给定一个代表每个房屋存放金额的非负整数数组 `nums`，计算你**在不触动警报装置的情况下**，今晚能够偷窃到的最高金额。

### 示例

```
输入：nums = [1,2,3,1]
输出：4
解释：偷窃 1 号房屋 (金额 = 1) ，然后偷窃 3 号房屋 (金额 = 3)。
     偷窃到的最高金额 = 1 + 3 = 4 。
```

```
输入：nums = [2,7,9,3,1]
输出：12
解释：偷窃 1 号房屋 (金额 = 2), 偷窃 3 号房屋 (金额 = 9)，接着偷窃 5 号房屋 (金额 = 1)。
     偷窃到的最高金额 = 2 + 9 + 1 = 12 。
```

---

## 解题思路

### 核心分析

这是一个经典的 **动态规划入门题**，完美体现了 DP 的核心思想：
> **状态定义** + **状态转移** + **最优子结构**

对于第 `i` 间房屋，小偷只有两种选择：
1. **偷第 i 间**：那么第 `i-1` 间不能偷，最大金额 = `dp[i-2] + nums[i]`
2. **不偷第 i 间**：最大金额 = `dp[i-1]`

### 状态转移方程

```
dp[i] = max(dp[i-1], dp[i-2] + nums[i])
```

其中 `dp[i]` 表示偷窃前 `i` 间房屋能获得的最大金额。

### 边界条件

- `dp[0] = nums[0]`（只有一间房，只能偷它）
- `dp[1] = max(nums[0], nums[1])`（两间房，选金额大的）

## 代码实现

### 解法一：标准 DP 数组

```python
class Solution:
    def rob(self, nums: List[int]) -> int:
        n = len(nums)
        if n == 0:
            return 0
        if n == 1:
            return nums[0]
        
        # dp[i] 表示偷窃前 i+1 间房屋的最大金额
        dp = [0] * n
        dp[0] = nums[0]
        dp[1] = max(nums[0], nums[1])
        
        for i in range(2, n):
            dp[i] = max(dp[i-1], dp[i-2] + nums[i])
        
        return dp[n-1]
```

### 解法二：空间优化 O(1)

```python
class Solution:
    def rob(self, nums: List[int]) -> int:
        n = len(nums)
        if n == 0:
            return 0
        if n == 1:
            return nums[0]
        
        # 只需要保存前两个状态
        prev2 = nums[0]  # dp[i-2]
        prev1 = max(nums[0], nums[1])  # dp[i-1]
        
        for i in range(2, n):
            curr = max(prev1, prev2 + nums[i])
            prev2 = prev1
            prev1 = curr
        
        return prev1
```

### 解法三：Java 实现（面试高频）

```java
class Solution {
    public int rob(int[] nums) {
        int n = nums.length;
        if (n == 0) return 0;
        if (n == 1) return nums[0];
        
        int prev2 = nums[0];
        int prev1 = Math.max(nums[0], nums[1]);
        
        for (int i = 2; i < n; i++) {
            int curr = Math.max(prev1, prev2 + nums[i]);
            prev2 = prev1;
            prev1 = curr;
        }
        
        return prev1;
    }
}
```

### 解法四：记忆化搜索（递归思路）

```python
from functools import lru_cache

class Solution:
    def rob(self, nums: List[int]) -> int:
        @lru_cache(maxsize=None)
        def dfs(i):
            if i < 0:
                return 0
            # 不偷第i间 或 偷第i间
            return max(dfs(i-1), dfs(i-2) + nums[i])
        
        return dfs(len(nums) - 1)
```

| 解法 | 时间复杂度 | 空间复杂度 | 说明 |

| 标准 DP | O(n) | O(n) | 需要 dp 数组存储所有状态 |

| 记忆化搜索 | O(n) | O(n) | 递归栈空间，理解用 |

## 关键点总结

1. **最优子结构**：偷前 `i` 间房屋的最优解，可以由前 `i-1` 和 `i-2` 间的最优推导

2. **状态压缩**：由于只依赖前两个状态，可将空间优化至 O(1)

3. **记忆化搜索**：从递归角度理解，帮助建立 DP 思维

4. **边界处理**：
   - 空数组返回 0
   - 单元素数组直接返回该元素
   - 两元素数组返回较大者

5. **面试常问**：
   - 为什么能用 DP？（最优子结构 + 无后效性）
   - 能否继续优化？（已经 O(n) 时间，空间可优化至 O(1)）

## 举一反三

**核心思想迁移**：
- 状态只有两种：「选」或「不选」当前元素
- 状态转移：dp[i] 由 dp[i-1] 和 dp[i-2] 决定
- 这是 **线性 DP** 最基础的模式，掌握后可解决 70% 的入门 DP 题
