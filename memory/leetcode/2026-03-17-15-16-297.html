LeetCode #297 - 二叉树的序列化与反序列化

Hard

📋 题目描述

请你设计一个算法，可以将一棵二叉树序列化成一个字符串，并且可以将这个字符串反序列化成原始的树结构。


序列化是将一个数据结构或对象转化为连续的比特位的过程，进而可以将其存储在一个文件或内存缓冲区中，同时通过网络传输到另一个计算机环境，并采取相反方式重构得到原数据。


你需要完成以下方法：

  • serialize(root) - 将二叉树序列化为一个字符串
  • deserialize(data) - 将字符串反序列化为二叉树

编码格式：使用前序遍历（根-左-右），用 'X' 表示空节点，节点间用 ',' 分隔。

示例：树 [1,2,3,null,null,4,5] 序列化为 "1,2,X,X,3,4,X,X,5,X,X"

1class Codec:

3    def serialize(self, root: TreeNode) -> str:
4        vals = []
5        def preorder(node):
6            if not node:
7                vals.append('X')
8                return
9            vals.append(str(node.val))
10            preorder(node.left)
11            preorder(node.right)
12        preorder(root)
13        return ','.join(vals)
14
15    def deserialize(self, data: str) -> TreeNode:
16        vals = iter(data.split(','))
17        def build():
18            val = next(vals)
19            if val == 'X':
20                return None
21            node = TreeNode(int(val))
22            node.left = build()
23            node.right = build()
24            return node
25        return build()

🧠 核心思路

前序遍历 + 标记空节点是本题的经典解法。


序列化流程：

  1. 对二叉树进行前序遍历（根 → 左 → 右）
  2. 遇到空节点时，用特殊标记（如'X'）记录
  3. 将所有节点值用分隔符（如','）连接成字符串

反序列化流程：

  1. 将字符串按分隔符分割成数组
  2. 使用迭代器按顺序读取元素
  3. 递归构建树：读到'X'返回None；否则创建节点，递归构建左右子树

⏱️ 复杂度分析

操作 时间复杂度 空间复杂度 说明
序列化 (serialize) O(n) O(n) 遍历所有n个节点，存储字符串
反序列化 (deserialize) O(n) O(n) 递归深度最坏为n（链状树）

💡 关键技巧

  • 空节点标记：必须用特殊值标记空节点，否则无法区分不同结构的树
  • 使用迭代器：反序列化时用迭代器替代数组下标，代码更简洁优雅
  • 递归与迭代的映射：序列化时的递归顺序，与反序列化时读取顺序完全一致
