LeetCode #62 - 不同路径

Medium 动态规划 组合数学 网格遍历

📋 题目描述

一个机器人位于一个 m x n 网格的左上角（起始点在下图中标记为 "Start" ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 "Finish" ）。

问总共有多少条不同的路径？

💡 核心思路

到达任意格子 (i,j) 的路径数 = 从上方格子 (i-1,j) 下来的路径数 + 从左方格子 (i,j-1) 过来的路径数。这就是典型的 状态转移方程。

示例： m = 3, n = 7 → 输出: 28
