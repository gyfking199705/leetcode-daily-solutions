LeetCode #110 - 平衡二叉树

EASY

📋 题目描述

给定一个二叉树，判断它是否是高度平衡的二叉树。

本题中，一棵高度平衡二叉树定义为：

一个二叉树每个节点的左右两个子树的高度差的绝对值不超过 1。

示例 1:

3 / \ 9 20 / \ 15 7

输入: root = [3,9,20,null,null,15,7]

输出: true

示例 2:

1 / \ 2 2 / \ 3 3 / \ 4 4

输入: root = [1,2,2,3,3,null,null,4,4]

输出: false（节点2的左右子树高度差为2）

📊 复杂度分析

指标 复杂度 说明
时间复杂度 O(n) 每个节点最多访问一次，剪枝后提前返回
空间复杂度 O(h) 递归栈深度，h为树的高度，最坏O(n)，平衡时O(log n)

🧠 核心思路

方法一：自顶向下（不推荐）

对每个节点计算左右子树高度，然后比较。时间复杂度 O(n²) — 每个节点被重复计算多次。

方法二：自底向上（推荐 ✓）

核心思想：后序遍历，从叶子节点向上计算高度。

  • 如果子树不平衡，返回 -1（剪枝）
  • 如果子树平衡，返回实际高度
  • 父节点根据子节点高度判断是否平衡

关键点

  • 🎯 空节点高度为 0
  • 🎯 用 -1 作为「不平衡」的特殊标记
  • 🎯 一旦发现不平衡，立即剪枝返回，不再计算其他节点
