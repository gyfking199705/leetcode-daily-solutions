克隆图

📋 题目描述

给你无向 连通 图中一个节点的引用，请你返回该图的 深拷贝（克隆）。

图中的每个节点都包含它的值 val（int） 和其邻居的列表（list[Node]）。

测试用例格式：

简单起见，每个节点的值都和它的索引相同。例如，第一个节点值为 1（val = 1），第二个节点值为 2（val = 2），以此类推。该图在测试用例中使用邻接列表表示。

邻接列表是用于表示有限图的无序列表的集合。每个列表都描述了图中节点的邻居集。

给定节点将始终是图中的第一个节点（值为 1）。你必须返回 给定节点的拷贝 作为对克隆图的引用。

🎬 算法执行过程（点击步骤或代码行查看对应关系）

步骤1: 入口检查

检查输入节点是否为None，边界情况处理

步骤2: 创建映射表

初始化visited字典，存储原节点到新节点的映射

步骤3: 初始化队列

从起始节点开始BFS，将原节点加入队列

步骤4: 克隆首节点

克隆起始节点，存入映射表

visited[1] = Node(1)

步骤5: BFS循环开始

取出队列中的原节点进行处理

步骤6: 遍历邻居

遍历当前节点的所有邻居节点

步骤7: 检查是否已访问

如果邻居未被访问，创建克隆并加入队列

步骤8: 克隆邻居节点

为新发现的邻居创建克隆节点

步骤9: 加入队列

将原邻居节点加入队列，后续处理其连接

步骤10: 添加邻居连接

在克隆节点上添加克隆邻居的引用

步骤11: 返回结果

返回起始节点的克隆，图克隆完成

🧠 核心思路

深拷贝意味着创建全新的节点对象，而不是引用原节点。图可能包含环，因此必须记录已访问节点避免无限循环。

方法数据结构核心思想
BFS队列 + 哈希表层序遍历，克隆节点并建立连接
DFS递归 + 哈希表递归遍历，回溯时建立连接

空间复杂度O(N)visited哈希表存储N个克隆节点，队列最多O(N)

📝 DFS 实现（替代方案）

class Solution:
    def cloneGraph(self, node: 'Node') -> 'Node':
        if not node:
            return None
        
        visited = {}
        
        def dfs(original):
            if original in visited:
                return visited[original]
            
            # 克隆当前节点（不处理邻居）
            clone = Node(original.val)
            visited[original] = clone
            
            # 递归克隆所有邻居
            for neighbor in original.neighbors:
                clone.neighbors.append(dfs(neighbor))
            
            return clone
        
        return dfs(node)
