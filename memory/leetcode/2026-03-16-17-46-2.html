LeetCode #2 - 两数相加 Medium

模拟加法运算的链表经典题目，考察数位处理与进位逻辑

📋 题目描述

给你两个 非空 的链表，表示两个非负的整数。它们每位数字都是按照 逆序 的方式存储的，并且每个节点只能存储 一位 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

💡 关键理解：链表的头部是数字的个位，这与我们手算加法的顺序一致（从右往左），所以可以直接遍历链表进行加法运算。
示例 1：
输入：l1 = [2,4,3], l2 = [5,6,4]
输出：[7,0,8]
解释：342 + 465 = 807
示例 2：
输入：l1 = [0], l2 = [0]
输出：[0]
示例 3：
输入：l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
输出：[8,9,9,9,0,0,0,1]

🎯 核心考点

🔗
链表遍历与构建
熟练操作链表指针，同时遍历两个链表并构建新链表

进位处理
理解进位的产生条件（和 ≥ 10），以及如何传递到高位
⚖️
边界情况
链表长度不同、最后一位产生进位（结果多一位）

📊 复杂度分析

维度 值 说明
时间复杂度 O(max(m, n)) 只需遍历较长的链表一次，m 和 n 分别是两个链表的长度
空间复杂度 O(max(m, n)) 新链表长度最多为 max(m, n) + 1（考虑最后的进位）

💭 解题思路详解

核心思想：模拟小学竖式加法。从个位开始，逐位相加，处理进位。

  1. 初始化：创建哑节点（dummy）作为结果链表的头节点，方便处理边界情况
  2. 遍历相加：同时遍历 l1 和 l2，取对应节点的值相加，再加上进位
  3. 计算新节点值：新值 = 总和 % 10，新进位 = 总和 // 10
  4. 链表长度不等：如果其中一个链表先遍历完，继续处理另一个链表剩余部分
  5. 最终进位：如果遍历结束后还有进位，需要再添加一个节点
🎨 设计技巧：使用「哑节点」（dummy node）可以简化代码逻辑，避免对头节点的特殊判断。最终返回 dummy.next 即可。
