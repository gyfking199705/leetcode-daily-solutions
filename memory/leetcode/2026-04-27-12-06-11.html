11. 盛最多水的容器

Container With Most Water · 高频面试题

题目描述

给定一个长度为 n 的整数数组 height。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i])。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。返回容器可以储存的最大水量。

注意：你不能倾斜容器。
核心思路
面积 = 底边宽度 × 最小高度 = (right − left) × min(height[left], height[right])
关键观察 🔑

双指针从数组两端向中间移动。底边宽度逐渐减小，那为什么要移动较短的那一侧？

1. 假设 height[left] < height[right]，此时面积由 height[left] 决定（短板效应）。
2. 如果固定 left，无论 right 怎么向左移动，底边宽度只会变小，而高度最大就是 height[left]。
3. 因此以 left 为边界的所有可能中，当前面积已经是最大的了。left 不可能再参与构成更大的面积。
4. 所以可以安全地丢弃 left（left++），继续搜索更大的可能性。

代码实现 (Python)
class Solution: def maxArea(self, height: List[int]) -> int: left, right = 0, len(height) - 1 max_area = 0 while left < right: # 计算当前面积 width = right - left h = min(height[left], height[right]) max_area = max(max_area, width * h) # 移动较短的一侧 if height[left] < height[right]: left += 1 else: right -= 1 return max_area

必会问题
  • 为什么移动短板而不是长板？—— 给出正确性证明
  • 暴力解法是什么？—— 双重循环 O(n²)
  • 边界情况？—— n=2 直接返回 min(h[0], h[1])
  • 如果要求返回具体的索引对？—— 记录最大面积时的 left/right

  • 如果数组中有负数高度？—— 取绝对值或题目不允许
