LeetCode #540 - 有序数组中的单一元素

Single Element in a Sorted Array | 二分查找 · 位运算

Medium 二分查找 位运算 通过率: 60%+

📋 题目描述

给你一个仅由整数组成的有序数组，其中每个元素都会出现两次，唯有一个数只会出现一次。

请你找出并返回那个只出现了一次的元素。

你设计的解决方案必须满足 O(log n) 时间复杂度和 O(1) 空间复杂度。

示例 1：

输入: nums = [1,1,2,3,3,4,4,8,8]

输出: 2

示例 2：

输入: nums = [3,3,7,7,10,11,11]

输出: 10

🎯 核心考点

  • 二分查找的本质：不是找目标值，而是找「状态发生变化的边界」
  • 观察规律：单一元素出现之前，成对元素的第一个在偶数索引；出现之后，成对元素的第一个在奇数索引
  • 边界处理：数组长度为1的特殊情况

Python 实现 - 二分查找

📊 复杂度分析

方法 时间复杂度 空间复杂度 说明
线性扫描 O(n) O(1) 不满足题目要求，但可作为对拍验证
二分查找 ✅ O(log n) O(1) 最优解，利用有序数组特性
位运算（异或） O(n) O(1) 巧妙解法，但时间复杂度不满足要求

💡 进阶解法：位运算

利用异或运算的性质：a ^ a = 0, a ^ 0 = a，成对的数字异或后会抵消，剩下唯一的单一元素。

class Solution:
    def singleNonDuplicate(self, nums: List[int]) -> int:
        result = 0
        for num in nums:
            result ^= num
        return result

虽然时间复杂度为 O(n)，但这是通用的「找唯一出现奇数次的元素」的解法，值得掌握。

📝 解题心得

易错点提醒

  • 索引越界：检查 mid+1 是否越界，或改用 mid-1 比较
  • 边界移动：确定 mid 在哪一边后，正确收缩左右边界
  • 偶数奇数判断：index % 2 == 0 表示偶数索引，反之为奇数

模板记忆法

二分查找的核心框架：

while left < right:      # 注意是 < 不是 <=
    mid = (left + right) // 2
    if 条件满足:          # 单一元素在右半部分
        left = mid + 1
    else:                # 单一元素在左半部分（含mid）
        right = mid
return nums[left]       # 最终 left == right
