🎯 LeetCode #152 - 乘积最大子数组

Medium 动态规划 · 数组 · 状态设计

📋 题目描述

给你一个整数数组 nums，请你找出数组中乘积最大的非空连续子数组（该子数组中至少包含一个数字），并返回该子数组所对应的乘积。

测试用例的答案是一个 32-位 整数。

示例 1：

输入：nums = [2, 3, -2, 4]

输出：6

解释：子数组 [2, 3] 有最大乘积 6

示例 2：

输入：nums = [-2, 0, -1]

输出：0

解释：结果不能为 2，因为 [-2, -1] 不是子数组（中间隔着0）

🔑 核心难点

负数 × 负数 = 正数！这意味着当前最小乘积可能在下一个负数到来时变成最大乘积。
因此需要同时维护最大乘积和最小乘积两个状态。

1class Solution:


2    def maxProduct(self, nums: List[int]) -> int:


3        # 同时维护最大和最小乘积


4        max_dp = min_dp = result = nums[0]


5        


6        for i in range(1, len(nums)):


7            num = nums[i]


8            # 当前数为负时，max_dp和min_dp会交换


9            candidates = [num, max_dp * num, min_dp * num]


10            max_dp = max(candidates)


11            min_dp = min(candidates)


12            


13            result = max(result, max_dp)


14        


15        return result

📊 复杂度分析

度量值说明
时间复杂度 O(n) 只需遍历数组一次
空间复杂度 O(1) 只使用常数个变量

💡 关键点解析

  • 为什么需要维护最小值？ 因为负数 × 负数 = 正数，当前最小值在遇到下一个负数时可能变成最大值
  • 状态转移方程： 对于每个位置 i，max_dp[i] = max(nums[i], max_dp[i-1] * nums[i], min_dp[i-1] * nums[i])
  • 空间优化： 由于只依赖前一个状态，可以用滚动变量将空间复杂度优化到 O(1)
  • 包含0的情况： 遇到0时，从0重新开始计算（相当于子数组在此处断开）
📝 与第53题（最大子数组和）的对比
  • 最大和： 只用维护一个最大值，因为加法不会翻转符号
  • 最大积： 需要同时维护最大和最小值，因为乘法可能翻转符号
