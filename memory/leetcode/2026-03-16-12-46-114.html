LeetCode #114 - 二叉树展开为链表

Flatten Binary Tree to Linked List Medium

📋 题目描述

给你二叉树的根结点 root，请你将它展开为一个单链表：

  • 展开后的单链表应该与二叉树 先序遍历 顺序相同
  • 链表中的节点使用相同的 TreeNode 类，其中 right 子指针指向链表中下一个节点，左子指针始终为 null
  • 展开后的单链表应始终保持原二叉树的相对顺序
💡 核心考点：二叉树的前序遍历 + 链表指针操作 + 原地修改（空间优化）

🎬 算法执行过程（点击步骤或代码行查看对应关系）

当前处理的节点：1
1 → 2 → 3 → 4 → 5 → 6

🐍 Python 实现（O(1)空间）

📊 复杂度分析

解法 时间复杂度 空间复杂度 特点
前序遍历 + 重建 O(N) O(N) 需要额外数组存储节点
O(1)空间迭代 ⭐推荐 O(N) O(1) 原地修改，利用右子树前驱节点
递归（右-左-根） O(N) O(H) 递归栈空间，逆向前序遍历

🎯 解题思路详解

方法一：O(1)空间迭代法（Morris遍历思想）

核心思想：对于当前节点，如果它有左子树，找到左子树的最右节点（即左子树先序遍历的最后一个节点），将当前节点的右子树接到这个最右节点的右指针上，然后把左子树移到右边。

方法二：递归法（逆向思维）

按照 右-左-根 的顺序遍历（逆向前序遍历），使用一个全局变量 prev 记录上一个访问的节点，每次将当前节点的右指针指向 prev，左指针置空。

🔑 关键洞察：
对于节点1，其左子树先序遍历的最后一个节点，正好应该指向节点1原来的右子树的头节点。
即：左子树最右节点 → 右子树头节点
