🎯 LeetCode #234 - 回文链表

Palindrome Linked List · Medium · 链表经典问题

📋 题目描述

给你一个单链表的头节点 head，请你判断该链表是否为回文链表。如果是，返回 true；否则，返回 false。

回文定义：正读和反读都相同的序列。例如 [1,2,2,1] 和 [1,2,3,2,1] 都是回文。

示例 1：

输入: head = [1,2,2,1]
输出: true

示例 2：

输入: head = [1,2]
输出: false

进阶：能否用 O(n) 时间复杂度和 O(1) 空间复杂度解决？

🎬 算法执行过程（点击步骤或代码行查看对应关系）

Python 实现（O(1) 空间）

核心思路：① 快慢指针找到中点 ② 反转后半部分 ③ 前后两段逐一比较 ④ 恢复链表（可选）

📊 复杂度分析

方案时间复杂度空间复杂度说明
数组复制O(n)O(n)遍历存数组，双指针比较
递归栈O(n)O(n)利用递归隐式栈反向遍历
快慢指针+反转O(n)O(1)最优解，面试推荐

💡 关键技巧总结

技巧1 快慢指针找中点：快指针走两步，慢指针走一步，当快指针到达末尾时，慢指针正好在中点。

技巧2 链表反转：迭代反转链表是基本功，三指针法（prev, curr, next）要熟练。

技巧3 边界处理：奇数个节点时，慢指针正好在中间，比较时要跳过中间节点。

📝 参考实现（含注释）

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def isPalindrome(self, head: Optional[ListNode]) -> bool:
        # 边界情况：空链表或单个节点
        if not head or not head.next:
            return True
        
        # Step 1: 快慢指针找中点
        slow = fast = head
        while fast and fast.next:
            slow = slow.next      # 慢指针走一步
            fast = fast.next.next # 快指针走两步
        
        # Step 2: 反转后半部分链表
        second_half = self.reverse(slow)
        
        # Step 3: 比较前后两部分
        first_half = head
        result = True
        while result and second_half:
            if first_half.val != second_half.val:
                result = False
            first_half = first_half.next
            second_half = second_half.next
        
        # Step 4: 恢复链表（可选，面试时最好加上）
        # slow = self.reverse(slow)
        
        return result
    
    def reverse(self, head: ListNode) -> ListNode:
        # 迭代法反转链表
        prev = None
        curr = head
        while curr:
            next_temp = curr.next
            curr.next = prev
            prev = curr
            curr = next_temp
        return prev

🎯 面试要点

  • 先问空间限制：是否能用 O(n) 空间？如果必须 O(1)，再用快慢指针+反转。
  • 边界处理：注意奇偶节点数的不同情况，奇数时中间节点无需比较。
  • 恢复链表：面试中建议恢复链表，展示代码完整性意识。
  • 代码复用：反转链表部分可以单独写成函数，结构更清晰。
