🌲 LeetCode #103 - 二叉树的锯齿形层序遍历

📋 题目描述

给你二叉树的根节点 root，返回其节点值的锯齿形层序遍历。（即先从左往右，再从右往左进行下一层遍历，以此类推，层与层之间交替进行）。

示例：输入: root = [3,9,20,null,null,15,7]
输出: [[3],[20,9],[15,7]]

Python 实现


1class Solution:
2    def zigzagLevelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
3        # 锯齿形层序遍历：奇数层翻转
4        
5        result = []  # 存储最终结果
6        
7        if not root:  # 处理空树
8            return result
9        
10        queue = [root]  # BFS队列
11        level = 0  # 当前层数，从0开始
12        
13        while queue:  # 当队列不为空
14            level_size = len(queue)  # 当前层节点数
15            current_level = []  # 当前层节点值
16            
17            for _ in range(level_size):  # 处理本层所有节点
18                node = queue.pop(0)  # 取出队首
19                current_level.append(node.val)  # 记录节点值
20                
21                # 子节点入队（先左后右）
22                if node.left:
23                    queue.append(node.left)
24                if node.right:
25                    queue.append(node.right)
26            
27            # 奇数层翻转（锯齿形关键）
28            if level % 2 == 1:
29                current_level.reverse()
30            
31            result.append(current_level)  # 添加层结果
32            level += 1  # 进入下一层
33        
34        return result

📊 复杂度分析

指标 复杂度 说明
时间复杂度 O(N) 每个节点恰好被访问一次，N为节点总数
空间复杂度 O(N) 队列最多存储一层的节点数，最坏情况下为N/2（完全二叉树最底层）
翻转操作 O(L) 每层最多翻转L个元素，L为当前层节点数，总体仍是O(N)

💡 核心思路

  • 标准BFS：使用队列进行层序遍历，逐层处理节点
  • 层数标记：用level变量记录当前层数（从0开始）
  • 翻转策略：奇数层（level % 2 == 1）时翻转当前层结果
  • 关键点：子节点入队顺序始终先左后右，翻转只影响输出不影响遍历
